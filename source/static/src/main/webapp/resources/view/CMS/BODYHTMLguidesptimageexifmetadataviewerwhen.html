Image EXIF Metadata Viewer Quando Usar

Last reviewed: 2026-07-21

Abra o Image EXIF Metadata Viewer quando precisar ver tags de camera, data e GPS numa foto antes de partilhar - sem enviar o ficheiro nem instalar ExifTool.


Quando se aplica

Verificar uma foto do telemovel antes de publicar. Confirmar se um JPEG ainda tem data de captura. Detetar GPS numa imagem de viagem. Preferir um parse local quando o ficheiro deve ficar neste dispositivo.


O que esperar

Escolha uma imagem, clique Read Metadata e reveja Camera, Date / Time, GPS e Other. Se houver GPS, aparece aviso de privacidade e link OpenStreetMap; pode descarregar um JPEG sem metadados. Tudo no navegador - sem upload e sem conta.
