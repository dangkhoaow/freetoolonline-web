Como Explorar o Saros Cycle Eclipse Explorer 3D

Última revisão: 2026-07-23

Revisado por Equipe editorial da FreetoolOnline

Abra o Saros Cycle Eclipse Explorer 3D, avance pela série com o controle deslizante ou pressione Play stepping, e veja a trajetória de cada eclipse se deslocar cerca de 120 graus para oeste em relação à anterior.

  1. Abra /space-3d/saros-cycle.html e aguarde a Terra renderizar.
  2. Arraste o controle de índice de eclipse de 0 para 1 e veja a trajetória brilhante se deslocar cerca de 120 graus para oeste da trajetória inicial esmaecida.
  3. Pressione "Show all tracks so far" para ver todos os passos da série de uma vez, avançando juntos para oeste.
  4. Pressione "Jump 3 steps (exeligmos)" para ver a trajetória retornar perto da longitude inicial após três passos.
  5. Pressione "Play stepping" para percorrer automaticamente toda a série.
  6. Leia o painel de dados para o tempo decorrido, o deslocamento acumulado para oeste e quanto tempo dura uma série completa.
