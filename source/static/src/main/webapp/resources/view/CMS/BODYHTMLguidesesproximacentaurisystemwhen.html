Cuándo Usar el Explorador 3D del Sistema Próxima Centauri - Mejores Sesiones y Límites

Última revisión: 2026-07-24

Abre el Explorador 3D del Sistema Próxima Centauri cuando quieras ver exactamente qué tan cerca orbitan Próxima b, d y c de la estrella más cercana al Sol, y por qué una enana roja propensa a erupciones complica el titular de "la estrella más cercana tiene un planeta habitable" - no para modelar la retención de atmósfera ni la dosis real de radiación de una erupción.


Una sesión sobre la estrella más cercana

Una sesión empieza observando los dos planetas más internos, b y d, y alternando la banda de zona habitable para ver que b cae dentro de ella mientras que d, que orbita aún más cerca, no lo hace - luego dispara una erupción para ver la complicación de radiación alcanzar a todos los planetas a la vez, y lee el panel de datos para las distancias, períodos y masas exactos.


Sesiones para las que este explorador no es adecuado

Las sesiones para las que esta página no es adecuada incluyen estudiar un sistema distante de cadena resonante de 7 planetas - usa el Explorador 3D TRAPPIST-1 para eso - o modelar las probabilidades reales de retención de atmósfera bajo radiación estelar, algo que ninguna visualización de este sitio intenta hacer. Recorre los controles en la guía paso a paso.
