Speaker & Audio Channel Test Quando Usar

Last reviewed: 2026-07-21

Abra o Speaker & Audio Channel Test quando precisar de confirmar os canais esquerdo e direito - por exemplo apos ligar auscultadores, mudar de colunas Bluetooth ou verificar stereo antes de uma chamada.


Quando se aplica

Use para um teste rapido L / R / ambos no browser. Clique Play Left, Play Right ou Play Both; use auscultadores se precisar de um pan claro. Ajuste a frequencia, depois Stop.


Quando usar outra ferramenta

Esta pagina nao mede volume absoluto em dB nem calibra monitores de estudio. Para microfone, use o teste de microfone do site. Para varrimentos longos, uma utilidade de audio no ambiente de trabalho encaixa melhor.


Privacidade

Tudo corre no browser - sem microfone, sem upload e sem conta. O audio usa StereoPannerNode da Web Audio API para tocar so o canal escolhido.
