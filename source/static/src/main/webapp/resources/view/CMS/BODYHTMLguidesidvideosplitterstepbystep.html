Pembagi Video Langkah demi Langkah

Last reviewed: 2026-07-17

Pembagi Video berjalan dalam tiga langkah: buka alat, berikan input Anda, lalu unduh hasilnya. Halaman ini menjelaskan apa yang diharapkan di setiap langkah.


Langkah 1 - buka alat

Buka Pembagi Video. Tidak perlu memasang apa pun atau masuk akun sebelum memulai.


Langkah 2 - berikan input Anda

Unggah file video Anda, lalu ketik titik pembagian di kolom "Split at" - baik detik biasa ("90") maupun format mm:ss ("1:30") berfungsi. Titik itulah tempat potongan terjadi.


Langkah 3 - dapatkan hasilnya

Alat ini menghasilkan dua klip terpisah: semua sebelum titik pembagian, dan semua sesudahnya. Masing-masing muncul sebagai tautan unduhan sendiri - tidak ada langkah zip atau arsip, dan file asli tidak disimpan setelah proses selesai.
