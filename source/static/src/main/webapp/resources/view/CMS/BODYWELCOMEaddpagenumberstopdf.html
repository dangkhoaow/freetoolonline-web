Add Page Numbers to PDF

Last reviewed: 2026-07-22

Add page numbers to every page of a PDF, right in the browser.

Upload the PDF, pick the starting number, and click Add page numbers - the download prints "Page N" centered at the bottom of every page, counting up from the number you chose, so a multi-page report or contract gets a consistent running count without opening a desktop editor.

Applies one fixed bottom-center style and numbering sequence - does not support custom position, custom text/format, skipping a numbered cover page, or restarting the count partway through the document.

The file is processed in this browser tab and is never uploaded to a server.

To split, merge, rotate, delete pages from, or watermark the PDF instead, see the related-tools section below.
