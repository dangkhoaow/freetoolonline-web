Video to GIF Converter

Last reviewed: 2026-07-04

Convert a video into a looping GIF right in your browser - upload the file, pick the output options, and download the result.

How it works

Choose your video file, set the output options shown above the file picker, then start the conversion. When it finishes, download the GIF. The defaults match the most common downstream use, so you can convert first and adjust only if the result needs it.

When to use it

A GIF is the right output when your destination accepts an animated GIF but not a video file - chat threads, wikis, issue trackers, and many older embed contexts fall into this group.

Good to know

  • A GIF loops silently: any audio in the source video is not carried into the GIF output.
  • GIFs are larger than the equivalent short video, so keep the clip brief and the dimensions modest for a smaller file.
