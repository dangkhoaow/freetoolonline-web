When to Play Layer Flip Platformer

Layer Flip Platformer fits when you want a short, keyboard-first skill run: flip light and dark layers with Space, clear hand-crafted stages, and leave without saving. It loads about ~44 KB after Play and needs no account.

Last reviewed: 2026-07-17

When it fits

Pick it when you have a few minutes, a keyboard or gamepad, and you enjoy puzzle-platform timing more than endless score chasing. The distinctive mechanic is flipping the active layer mid-jump so platforms and spikes swap. Progress is session-only - closing the tab resets the run, which is ideal when you do not want saves on a shared machine.

When it is not the right pick

Skip it if you need touch-only mobile controls (this build has none), if you want an endless runner like Pixel Spike Run, or if you prefer one-tap dash platforming like One Tap Platformer. It also will not keep a high-score cloud save - there is no localStorage at all.

What to expect

Twenty-five hand-crafted stages, death and star counters, procedural Web Audio, and a title menu with controls help. Adapted from ONOFF (js13kGames 2018, MIT). Free, no ads in the iframe game, no install.
