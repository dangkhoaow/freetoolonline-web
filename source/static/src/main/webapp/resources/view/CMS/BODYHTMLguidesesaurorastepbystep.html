Como Usar Aurora 3D Explorer

Last reviewed: 2026-07-15

Sube el Kp de ensenanza, mira el ovalo expandirse hacia el ecuador y lee altitudes verde 557.7 nm / rojo 630.0 nm en el panel.

  1. Abre Aurora 3D Explorer y espera Tierra y el ovalo brillante.
  2. Arrastra para orbitar el lado nocturno; manten el panel de hechos visible.
  3. Pulsa Kp + / Kp - y observa el ovalo hacia latitudes mas bajas.
  4. Pausa el giro para comparar bandas verde (~90-150 km), roja (~200-300 km) y purpura.
  5. Reinicia a Kp 3 y repite para fijar las altitudes de emision.
