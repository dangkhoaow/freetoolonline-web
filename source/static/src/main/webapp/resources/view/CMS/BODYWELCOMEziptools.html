Three ZIP tools in one place: compress files into a new archive, remove a known password, or extract an archive you received - no software to install, no sign-in required.

When to land on this ZIP-tools hub

Land on this hub when the next step is not yet decided - whether to compress a folder into a new archive, drop the password off an existing archive, or extract the contents of one a sender forwarded. The one-screen choice between the three labels is faster than guessing from the navigation, and each label routes to a child tool that runs the work in your browser with no install and no signup. Skip the hub when the answer is already obvious and go straight to the matching child tool from a bookmark.

Targeting a specific archive size cap

When the destination has a hard size cap (the common 20 MB or 25 MB email-attachment ceiling, a 100 KB upload widget limit, a 5 MB chat-attachment cap), pick the Compress label and let the creator report the post-build archive byte count before downloading. The headroom comes from the compressibility of the input - text and source code typically shrink 60-80% inside ZIP, while JPGs, MP4s, and PDFs barely move because they were already compressed. If the post-build number still exceeds the cap, the realistic move is to compress the media first (with an image or video tool) and then re-archive the smaller files; ZIP alone cannot recover the size cut on already-compressed inputs.

Free, in-browser, no account needed

All three labels are free with no account to sign up for, no install to download, no subscription, no paid tier, and no watermark on the output - the same whether you process one file or two thousand. The Compress ZIP File tool offers an in-browser mode that keeps files on your device with no upload; the default path, along with Unzip and Remove Password, sends the file over HTTPS to the freetoolonline service, runs the operation server-side, and returns a download link, with a short retention window and an anonymous, sign-in-free session. The hub itself is informational - clicking through to a child tool is what runs the archive work.

Starting from a folder on your device

Picking the Compress label is the right move when the starting point is a folder and the goal is one shareable archive the recipient can open without installing anything. The Compress ZIP File child tool reads the folder structure, preserves nested directories, and writes a single .zip the recipient extracts on Windows, macOS, Linux, iOS, or Android without a third-party app. Folders holding tens of thousands of small files may exceed mobile-browser tab memory; in that case, split the input into batches of a few hundred so the build does not stall partway through.
