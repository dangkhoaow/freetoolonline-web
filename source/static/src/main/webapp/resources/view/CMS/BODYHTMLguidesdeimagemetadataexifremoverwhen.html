Wann EXIF vor dem Teilen entfernen

Nutzen Sie diesen Remover, wenn ein JPEG oder HEIC noch Kameramodell, Aufnahmezeit oder GPS traegt und Sie das Foto veroeffentlichen oder senden wollen.

Datei waehlen, optional Read Metadata, dann Download Without Metadata. Die Seite schreibt ein JPEG im Browser neu - das Original bleibt auf dem Geraet.

Nur Tags pruefen? Nutzen Sie den Image EXIF Metadata Viewer. Diesen Remover waehlen, wenn das saubere JPEG das Ziel ist.
