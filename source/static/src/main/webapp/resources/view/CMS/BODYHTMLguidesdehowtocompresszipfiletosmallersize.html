Wie man eine Zip kleiner komprimiert

Zuletzt geprueft: 2026-04-27. Oeffnen Sie das ZIP-Werkzeug, um in Ihrem Browser neu zu komprimieren, ohne das Archiv hochzuladen.

30-Sekunden-Antwort. Eine bestehende Zip neu zu zippen macht sie fast nie kleiner, weil der Inhalt bereits komprimiert ist. Zwei Dinge funktionieren: zuerst extrahieren, die schwersten Dateien darin (Bilder, PDFs, Audio) mit einem anderen Werkzeug komprimieren, dann neu zippen; oder ein staerkeres Format (7z, Zip-mit-LZMA) waehlen, wenn der Empfaenger es unterstuetzt.

Warum Neu-Zippen fast nichts bewirkt

Zip verwendet den Deflate-Algorithmus. Deflate findet wiederholte Byte-Muster und ersetzt sie durch kuerzere Codes. Nach einem Durchgang sind die Muster weg. Ein zweiter Durchgang ueber dieselben Daten findet nichts zum Ersetzen.

Was funktioniert: extrahieren, komprimieren, neu zippen

  1. Extrahieren Sie das Archiv lokal, damit Sie die einzelnen Dateien sehen koennen.
  2. Verwenden Sie den richtigen Kompressor fuer die schweren. Eine 5-MB-JPEG faellt unter 1 MB ueber Bildkomprimierung.
  3. Neu zippen Sie die kleineren Dateien. Verwenden Sie Best-Kompression bei Dokumenten und Store bei den (nun kleineren) Medien.
