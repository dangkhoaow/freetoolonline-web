Password Strength Checker - Comparado A Alternativas

Last reviewed: 2026-07-23

O Password Strength Checker é uma de três formas comuns de fazer isso: uma ferramenta baseada em navegador como esta, um aplicativo de computador, ou um serviço on-line que exige envio de dados. Cada uma tem um equilíbrio diferente entre para onde os dados vão e o que é preciso instalar.

A pontuação vem do zxcvbn, o mesmo modelo de comparação de padrões e ataque de dicionário usado pelo Dropbox, e não uma regra simples de tamanho ou tipo de caractere - ele identifica senhas comuns e padrões previsíveis que uma regra como "P@ssw0rd1" não pega.

Tudo roda direto no navegador - sem envio, sem conta, e a senha nunca é enviada para lugar nenhum, registrada ou verificada contra um banco de dados de violações.
