Cuando usar Pluto-Charon Double Planet 3D Explorer

Ultima revision: 2026-07-21

Abre el Pluto-Charon Double Planet 3D Explorer cuando necesites una leccion de bloqueo tidal mutuo con baricentro fuera del primario - sin un integrador gravitatorio.


Leccion de baricentro externo

Muestra que ~2,110 km desde el centro de Pluton supera el radio medio de 1188.3 km, asi que el marcador verde queda en el espacio entre ambos mundos.


Cifras New Horizons / JPL

Aqui encajan 1188.3 km / 606 km, 1.303e22 kg / 1.586e21 kg (~8.2:1), ~19,591 km, 6.387 days y bloqueo tidal mutuo.


Cuando este explorador no encaja

No lo uses para el baricentro interno Tierra-Luna o para una linea de tamanos de planetas enanos. Usa el Earth-Moon Barycenter 3D Explorer o el Dwarf Planet Comparison 3D Explorer. Controles en la guia paso a paso.
