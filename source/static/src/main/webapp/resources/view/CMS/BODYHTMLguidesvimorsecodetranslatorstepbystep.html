Morse Code Translator - Từng Bước

Last reviewed: 2026-07-21

Mở Morse Code Translator, dán chữ hoặc Morse bên trái, bấm chuyển đổi, rồi sao chép kết quả bên phải.


Các bước

  1. Dán chữ thường (mã hóa) hoặc chấm/gạch (giải mã) vào ô nhập.
  2. Bấm nút chuyển đổi màu đỏ.
  3. Đọc kết quả: chữ thành mã Morse cách nhau bằng khoảng trắng; từ dùng dấu gạch chéo; Morse thành chữ.
  4. Bấm Copy Output và dán nơi bạn cần.

Riêng tư

Việc dịch chạy trong trình duyệt. Văn bản không được tải lên.
