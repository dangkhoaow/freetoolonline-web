Kapan memakai Tides: Earth-Moon 3D Explorer

Pakai Tides: Earth-Moon 3D Explorer untuk melihat cepat di browser mengapa pantai punya dua pasang surut sehari dan bagaimana pasang purnama berbeda dari pasang perbani - lewati jika Anda perlu model prediksi pasang nyata untuk pelabuhan tertentu.

  • Demo kelas untuk model dua tonjolan pasang kesetimbangan
  • Membandingkan pasang purnama (sekitar 1.5x) vs pasang perbani (sekitar 0.5x)
  • Penyegaran cepat sebelum membaca tidal-locking atau moon-phases-3d

Halaman ini tidak menggantikan tabel pasang nyata untuk pelabuhan tertentu, tidak memodelkan resonansi teluk atau badai, dan tidak mensimulasikan arus laut.

Ikuti kontrol di panduan langkah demi langkah, atau bandingkan opsi di vs alternatif. Buka Tides: Earth-Moon 3D Explorer.
