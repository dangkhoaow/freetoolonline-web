Number Base Converter (binary/hex/octal/decimal) - Paso A Paso

Last reviewed: 2026-07-23

Number Base Converter (Binary/Hex/Octal/Decimal) funciona en tres pasos: abrir la herramienta, escribir tu valor y luego copiar el resultado convertido. Esta página explica qué espera cada paso.


Paso 1 - abrir la herramienta

Ve a Number Base Converter (Binary/Hex/Octal/Decimal). No necesitas instalar nada ni iniciar sesión antes de empezar.


Paso 2 - da tu entrada

Se admiten bases personalizadas de 2 a 36 para conversiones menos comunes más allá de los cuatro ajustes predefinidos.


Paso 3 - obtén el resultado

La conversión usa aritmética exacta dígito a dígito con BigInt, no el tipo Number nativo, así que los valores más allá de 2^53 - 1 se mantienen exactos en lugar de perder precisión en silencio.

No maneja números negativos en representación de complemento a dos para anchos de bits arbitrarios - solo la semántica de entero con signo de JavaScript para el signo menos.
