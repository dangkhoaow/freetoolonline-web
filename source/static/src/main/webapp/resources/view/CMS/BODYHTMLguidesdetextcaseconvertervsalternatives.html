Der Text Case Converter ist eine von drei gaengigen Wegen dafuer: ein browserbasiertes Tool wie dieses, eine Desktop-Anwendung, oder ein Upload-basierter Online-Dienst. Jede Variante hat einen anderen Trade-off, wohin Ihre Daten gehen und was Sie installieren muessen.

Dieses Tool formatiert nur die Gross-/Kleinschreibung des eingefuegten Textes um - es zaehlt keine Woerter und erstellt keinen Hash. Fuer Wort- und Zeichenzaehlung nutzen Sie den Word Counter; um Text in einen MD5-Hash umzuwandeln, nutzen Sie den MD5 Hash Generator.

Alles laeuft lokal mit reinem JavaScript; in keinem Schritt wird eine Datei erstellt oder hochgeladen.
