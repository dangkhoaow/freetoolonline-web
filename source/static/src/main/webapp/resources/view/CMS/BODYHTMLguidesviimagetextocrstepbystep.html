Image To Text (ocr) Tung Buoc

Last reviewed: 2026-07-16

Image to Text (OCR) chay qua ba buoc: mo cong cu, chon mot anh, roi copy hoac tai xuong chu da nhan dang. Trang nay di qua tung buoc can gi.


Buoc 1 - mo cong cu

Vao Image to Text (OCR). Khong can cai gi hay dang nhap truoc khi bat dau.


Buoc 2 - chon mot anh va bam Extract text

Chon mot file anh bang bo chon file - anh mo trong trinh duyet va khong bao gio duoc gui len server - roi bam Extract text. Lan chay dau tien tren thiet bi nay tai bo may OCR ngam, cac lan chay sau tren cung thiet bi bat dau nhanh hon.


Buoc 3 - copy hoac tai xuong chu da nhan dang

Chu da nhan dang hien trong mot o chi doc khi qua trinh quet xong. Copy chi voi mot lan bam hoac tai xuong dang file .txt. OCR khong hoan hao, nen hay kiem tra ten, so, va dau cau truoc khi tin vao ket qua.
