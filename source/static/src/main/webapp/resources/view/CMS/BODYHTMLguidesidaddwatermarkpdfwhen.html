Tambah Watermark ke PDF layak dibuka saat kamu perlu proses cepat sekali jalan tanpa memasang apa pun. Halaman ini membahas kapan itu cocok dan kapan tidak.


Kapan cocok

Hanya menerapkan satu gaya teks diagonal tetap (abu-abu, semi-transparan, di tengah) - tidak mendukung posisi, warna, ukuran khusus, watermark gambar, atau watermark berbeda per halaman.


Apa yang diharapkan

File diproses langsung di tab browser ini dan tidak pernah diunggah ke server.

Untuk memisahkan, menggabungkan, memutar, atau menghapus halaman PDF, lihat bagian alat terkait.
