Image To Text (ocr) Paso A Paso

Last reviewed: 2026-07-16

Image to Text (OCR) funciona en tres pasos: abrir la herramienta, elegir una imagen, luego copiar o descargar el texto reconocido. Esta pagina explica que espera cada paso.


Paso 1 - abrir la herramienta

Ve a Image to Text (OCR). No necesitas instalar nada ni iniciar sesion antes de empezar.


Paso 2 - elige una imagen y pulsa Extract text

Elige un archivo de imagen con el selector de archivos - se abre en el navegador y nunca se sube a un servidor - luego pulsa Extract text. La primera ejecucion en este dispositivo descarga el motor de OCR en segundo plano; las siguientes en el mismo dispositivo empiezan mas rapido.


Paso 3 - copia o descarga el texto reconocido

El texto reconocido aparece en un cuadro de solo lectura cuando termina el escaneo. Copialo con un clic o descargalo como archivo .txt. El OCR no es perfecto, asi que revisa nombres, numeros y puntuacion antes de confiar en el resultado.
