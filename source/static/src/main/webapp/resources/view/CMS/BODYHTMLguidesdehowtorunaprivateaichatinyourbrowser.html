Privaten KI-Chat im Browser ausfuehren - kostenlos, ohne Upload

Fuehre einen KI-Chatbot vollstaendig in deinem Browser aus - das Modell laedt einmal herunter und laeuft auf deinem eigenen Geraet, also werden deine Nachrichten nie hochgeladen und es ist kein Konto noetig. Waehle ein kleines Modell, lass es laden und chatte. Ein Browser mit WebGPU, etwa das neueste Chrome oder Edge auf dem Desktop, ist erforderlich.

Schritte

  1. Oeffne das Tool und waehle ein Modell aus der Liste - kleinere Modelle laden schneller.
  2. Klicke auf Load model. Beim ersten Mal laedt das Modell in deinen Browser (Hunderte MB) und wird fuer das naechste Mal zwischengespeichert.
  3. Warte, bis der Fortschrittsbalken fertig ist; der Status zeigt, wann das Modell bereit ist.
  4. Tippe eine Nachricht und klicke auf Send (oder druecke Ctrl+Enter). Antworten kommen als Stream, erzeugt auf deinem Geraet.
  5. Wechsle das Modell jederzeit; deine letzte Wahl wird in diesem Browser gemerkt.

Modelle und Downloadgroesse

Kleinere Modelle laden am schnellsten; groessere antworten besser, laden aber mehr. Jedes Modell laedt einmal herunter und wird dann von deinem Browser zwischengespeichert:

Was privat bleibt

Das Modell laeuft ueber WebGPU auf deinem eigenen Geraet. Deine Nachrichten werden nie hochgeladen, es gibt kein Konto und nichts wird an einen Server gesendet. Der einzige Netzwerk-Download ist das Modell selbst, einmal aus der Modell-Registry geholt und dann von deinem Browser zwischengespeichert.

Was dieses Tool nicht tut

Es braucht einen Browser mit WebGPU (das neueste Chrome oder Edge auf dem Desktop, oder Chrome auf Android); Browser ohne WebGPU zeigen einen Hinweis. Der Chat-Verlauf bleibt waehrend der Sitzung im Tab und wird nicht gespeichert oder ueber Geraete synchronisiert. Es ist ein lokales Chat-Modell, es durchsucht also nicht das Web und liest deine Dateien nicht.
