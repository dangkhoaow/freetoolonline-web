How the Tsintaosaurus size-vs-human toggle stays honest

The Tsintaosaurus model is drawn to a fixed on-screen length so it fits the canvas; the Size vs human control then places a 1.8 m person at the true length ratio used by the viewer (10 m). Tap Size vs human under the canvas to show or hide the scale figure - the comparison is proportional, not a decorative sticker. Published length still spans about 8.3-12 m, so the toggle picks one clear scale inside that disclosed range.

What the Tsintaosaurus size numbers are not

What these size numbers are not is a claim of one precise length or weight: the Tsintaosaurus size comparison discloses the about 8.3-12 m / 2.5-3 t ranges and uses 10 m only for the Size vs human scale; it does not invent soft-tissue color accuracy, and this is not a win/lose game.
