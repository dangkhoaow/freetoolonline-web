Gamepad / Controller Test Wann Nutzen

Last reviewed: 2026-07-21

Öffnen Sie Gamepad / Controller Test, wenn Sie prüfen wollen, ob ein USB- oder Bluetooth-Controller Tasten und Sticks meldet - ohne Desktop-Utility.


Wann es passt

Ein neuer Pad wirkt an einer Taste tot. Ein Stick driftet und Sie wollen Achsenwerte sehen. Sie bevorzugen einen Tab-Test ohne Konto.


Was zu erwarten ist

Polling aktivieren, eine Taste drücken, damit der Browser den Pad freigibt, Indikatoren und Achsen lesen. Reset leert die Sitzung. Nichts wird hochgeladen.

Genaue Schritte: Schritt-für-Schritt.
