Hash Generator (sha-256/512) Schritt Fuer Schritt

Last reviewed: 2026-07-16

Hash Generator (SHA-256/512) laeuft in drei Schritten: Tool oeffnen, Text eingeben oder einfuegen, dann einen Algorithmus waehlen und den Digest kopieren. Diese Seite geht durch, was jeder Schritt erwartet.


Schritt 1 - Tool oeffnen

Gehen Sie zu Hash Generator (SHA-256/512). Vor dem Start muss nichts installiert oder eingeloggt werden.


Schritt 2 - Text eingeben oder einfuegen

Tippen oder fuegen Sie den zu hashenden Text in das Feld ein; waehlen Sie dann SHA-1, SHA-256, SHA-384 oder SHA-512 im Dropdown. Der Digest aktualisiert sich live darunter, waehrend Sie tippen.


Schritt 3 - Digest kopieren

Klicken Sie Copy, um den resultierenden Hex-Digest zu holen. Das Hashen laeuft komplett clientseitig ueber die WebCrypto-API des Browsers - es wird nichts an einen Server gesendet.

MD5 wird hier nicht angeboten, da moderne Browser es als unsicher weglassen; nutzen Sie das verwandte md5-converter-Tool fuer diesen speziellen Algorithmus.
