PDF Page Trimmer: Cut or Split PDF Pages Easily

Last reviewed: 2026-06-29

When dealing with large PDF files, it's often necessary to isolate just the pages you need - a chapter from an e-book, the signature pages of a contract, or the appendix of a report. This free online tool lets you name the exact start and end pages and pulls that section into a new PDF.

Split PDF by range extracts a contiguous page range from a PDF into a new file while preserving text, bookmarks, and internal links.

  1. Upload the source PDF and wait for the page count to appear.
  2. Enter the start page and end page (1 ≤ start ≤ end ≤ total pages).
  3. Click Split and download the extracted page-range PDF.

Need more than one section? Enter several ranges at once, for example 1-3,7,10-12, and each range comes back as its own PDF.

Note: For splitting every page into its own file, use Split PDF to each page. To combine extracted ranges into a new document, chain with Join PDF from multiple files.

Making a PDF smaller after extracting a range

This tool extracts a contiguous page range into a new PDF, which often produces a smaller file because pages outside the range are dropped. If the file is still larger than the delivery target after the range extraction, open flatten PDF next: it bakes annotations and form fields into the page rendering, which typically removes the largest non-essential payload from a PDF. Both steps are free and run through the same secure upload pipeline, so a heavy source PDF can be shrunk in two passes without leaving the site.
