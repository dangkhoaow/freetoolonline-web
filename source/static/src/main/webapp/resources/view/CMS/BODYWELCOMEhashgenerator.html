Hash Generator (SHA-256/512)

Last reviewed: 2026-07-16

Hash the text you type or paste into a SHA digest.

Copy the resulting hex digest with one click.

Hashing runs entirely client-side via the browser's WebCrypto API - nothing is sent to a server. MD5 is not offered here since modern browsers omit it as insecure; use the related md5-converter tool for that specific algorithm.

Nothing you type here is stored beyond this page session, sent to any analytics or tracking service, or tied to an account - there is no signup required to use it.
