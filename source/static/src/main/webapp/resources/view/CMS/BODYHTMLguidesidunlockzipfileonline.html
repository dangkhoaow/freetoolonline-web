Buka Kunci File ZIP Online

"Buka kunci" sebuah ZIP bisa berarti dua hal yang berbeda, dan alat yang tepat tergantung pada apa yang dibutuhkan arsip Anda. ZIP yang dilindungi kata sandi membutuhkan kata sandi dihapus sebelum terbuka dengan bersih di sistem operasi Anda; ZIP biasa hanya membutuhkan ekstraktor yang dapat membaca header arsip dan mengunduh file di dalamnya.

Jika Anda klik dua kali ZIP di Windows atau macOS dan sistem operasi menampilkan prompt yang meminta kata sandi sebelum ada file yang muncul, arsip dilindungi kata sandi. Bawa ZIP ke /remove-zip-password.html dan berikan kata sandi yang Anda terima dari pengirim; alat menghasilkan salinan arsip dengan kata sandi dihapus. Jika ZIP terbuka ke pohon folder tanpa menanyakan apa pun, bawa ke /unzip-file.html.
