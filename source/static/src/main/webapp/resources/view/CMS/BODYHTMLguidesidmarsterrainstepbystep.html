Cara Pakai Mars Terrain Explorer

Last reviewed: 2026-07-15

Muat halaman, tunggu data elevasi asli terunduh, lalu tarik slider relief dan klik globe untuk membaca elevasi nyata di titik itu.

  1. Buka Mars Terrain Explorer dan tunggu bola placeholder berubah jadi permukaan Mars yang nyata.
  2. Seret untuk memutar globe; biarkan panel fakta tetap terlihat.
  3. Tekan Relief + beberapa kali dan lihat Olympus Mons naik dan Hellas Planitia turun.
  4. Tekan Real scale untuk melihat betapa datarnya Mars pada proporsi sebenarnya.
  5. Klik di mana saja pada globe untuk membaca elevasi nyata di titik itu.
