Wann der Moropus-Browser-Viewer passt

Waehlen Sie den Moropus-Browser-Viewer, wenn Sie ein miozaenes gekralles Chalicothere mit Size vs human im ehrlichen 3.3-m-Panel-Laengenverhaeltnis orbitieren und Fakten anklicken wollen - ohne etwas zu installieren. Nichts vom Besuch wird an einen Server gesendet.


Wann eine Telefon-AR-App oder ein Museum besser passt

Eine Telefon-AR-App gewinnt, wenn Sie die Silhouette auf dem Wohnzimmerboden brauchen; dieser Viewer behauptet nie Raumplatzierung. Ein Museum gewinnt fuer das Gehen neben echten Chalicothere- oder Saeugetier-Montagen. Ein Standbild gewinnt nur, wenn Sie einen einzelnen teilbaren Frame ohne Interaktivitaet brauchen.


Was dieser Moropus-Vergleich nicht ist

Dieser Moropus-Vergleich ist eine Trade-off-Tabelle fuer den gelieferten Viewer - kein Ranking kommerzieller AR-Marken, erfindet keine AR-Funktionen, die die Seite nicht hat, behauptet nicht, Moropus sei ein Dinosaurier, behauptet nicht, das optionale glTF sei eine wissenschaftlich exakte Rekonstruktion, und erfindet keinen Spielstand. Zum Steuerungs-Walkthrough siehe Moropus in 3D ansehen. Zu Laenge und Gewicht neben einer 1.8-m-Person siehe Moropus Groessenvergleich.
