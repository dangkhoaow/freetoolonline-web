SQL Formatter & Beautifier - Paso a Paso

Last reviewed: 2026-07-21

Abre SQL Formatter & Beautifier, pega el SQL a la izquierda, pulsa Format y copia el resultado indentado a la derecha.


Pasos

  1. Pega o escribe el SQL en el cuadro de entrada.
  2. Pulsa el botón rojo Format.
  3. Revisa la salida: cláusulas principales en líneas nuevas, palabras clave en mayúsculas.
  4. Pulsa Copy Output y pégalo en tu editor.

Privacidad

El formateo se ejecuta en el navegador. El texto de la consulta no se sube.
