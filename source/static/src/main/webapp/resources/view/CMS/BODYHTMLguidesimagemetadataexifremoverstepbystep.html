How to remove photo metadata in your browser

  1. Open Image Metadata (EXIF) Remover and choose a local image.
  2. Optional: click Read Metadata and scan Camera, Date / Time, GPS, and Other.
  3. If a GPS warning appears, open the OpenStreetMap link to see the leak, then continue.
  4. Click Download Without Metadata to save a JPEG re-encoded at 0.95 quality without EXIF/XMP.
  5. Clear when you are done so the preview and tables leave the tab.

Nothing uploads. Canvas strip does not write tags back into the original file losslessly - you get a new JPEG download.
