Cuando Usar Mars Terrain Explorer - Mejores Sesiones y Limites

Last reviewed: 2026-07-16

Abre el Mars Terrain Explorer cuando necesites alfabetizacion real de elevacion NASA MOLA - Olympus Mons cerca de ~21.2 km, Hellas cerca de ~-8.2 km y Valles Marineris unos 3.870 km de largo - sin instalar software GIS planetario.


Una demo de clase de cinco minutos

Una demo de clase de cinco minutos empieza cuando abres Mars Terrain Explorer, tiras Relief + para que Olympus Mons suba, y luego tocas el globo para una lectura real de elevacion - mas claro que listar cuatro numeros MOLA en una diapositiva sola.


Relieve frente a escala planetaria

El relieve frente a la escala planetaria sigue legible cuando activas Real scale y ves lo sutil que es el relieve real frente al diametro medio de Marte 6.779 km. El panel de hechos mantiene el rango publicado -8.201 m a +21.241 m relativo al areoide.


Sesiones para las que Mars Terrain Explorer no encaja

Sesiones para las que Mars Terrain Explorer no encaja son navegadores de imagenes de Marte, planificadores de rutas de rover, editores shapefile o telemetria en vivo. Usa Earth 3D Globe o Planet Size Comparison. Recorre los controles en la guia paso a paso de Mars Terrain Explorer.
