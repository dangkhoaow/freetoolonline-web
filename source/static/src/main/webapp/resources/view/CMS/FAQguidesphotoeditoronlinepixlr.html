Frequently asked questions

Does the browser photo editor upload my image to a server?

No. The photo editor reads your file into the page and processes it on your own machine. Nothing is sent to a server, and there is no account needed. When you close or reload the tab, the working copy is released - save the exported file before navigating away.

Which image formats does the editor open and export?

The editor opens JPG, PNG, and WebP files up to 8K resolution. Export supports the same three formats. Choose PNG when you need transparency or the sharpest text edges; choose JPG when a smaller file size matters more; choose WebP for web use where you can guarantee browser support.

Can I add text or a watermark without losing the original photo?

Yes. Text is added on its own layer above the photo, so the original pixel data underneath is not changed. Move, resize, or delete the text layer before exporting, or merge all layers when the result looks right. Keep the original file if you need to redo the edit from scratch - the tool does not auto-save.
