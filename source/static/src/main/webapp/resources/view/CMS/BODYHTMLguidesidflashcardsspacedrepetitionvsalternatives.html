Kartu Hafalan (Pengulangan Berjarak) Vs Alternatif

Last reviewed: 2026-07-13

Kartu Hafalan (Pengulangan Berjarak) adalah satu dari tiga cara umum membuat dan belajar dengan set kartu: alat berbasis browser seperti ini, aplikasi desktop, atau aplikasi/layanan repetisi berjarak khusus. Masing-masing punya kelebihan berbeda soal tempat kartu disimpan dan apa yang perlu dipasang.

Alat ini cocok untuk set kartu yang cepat dan mandiri yang ingin kamu buat dan latih di browser. Saat sebuah set butuh disinkronkan lintas perangkat atau menjadwalkan pengulangan otomatis selama berminggu-minggu, aplikasi repetisi berjarak khusus adalah pilihan yang lebih baik.

Tandai kartu sebagai "I knew it" untuk menyisihkannya pada putaran ini, atau "Still learning" untuk tetap menyimpannya di antrean; Shuffle mengacak urutan belajar.

Kartu disimpan di browser ini lewat penyimpanan lokal, jadi tetap ada setelah memuat ulang halaman; ekspor cadangan (JSON) sebelum menghapus data browser atau ganti perangkat, karena keduanya tidak disinkronkan otomatis.
