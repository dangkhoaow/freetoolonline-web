Bloc de Notas / Notas vale la pena abrirlo cuando necesitas algo rapido y puntual, sin instalar nada. Esta pagina explica cuando eso encaja y cuando no.


Cuando encaja

Usalo como un bloc de borrador rapido - un numero de telefono, una linea de texto, o un recordatorio que no quieres escribir en una app permanente.


Que esperar

Cada nota tiene un boton Copiar para copiar su texto, y un boton Eliminar para quitarla; las notas no se pueden editar en el lugar, asi que elimina y agrega de nuevo para cambiar una.

Las notas se guardan en este navegador mediante almacenamiento local, asi que sobreviven a una recarga de la pagina; exporta una copia de seguridad (JSON) antes de borrar los datos del navegador o cambiar de dispositivo, ya que ninguno se sincroniza automaticamente.
