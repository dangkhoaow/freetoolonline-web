Windows 11 Insider format FAT32 hingga 2 TB lewat command line

Build Windows 11 Insider Preview April 2026 menaikkan batas format FAT32 lewat command line dari 32 GB ke 2 TB. Ini penting jika Anda menyiapkan USB untuk kamera, mobil, atau mesin campur OS yang masih mengharapkan FAT32 - tetapi tidak menghapus batas terpisah 4 GiB per file. Untuk memahami mengapa video besar tetap gagal di FAT32 meski stick sudah besar, lihat batas file 4 GiB FAT32.

Last reviewed: 2026-07-19

Published July 19, 2026

What happened

Blog Windows Insider untuk build Dev 26300.8170 (diumumkan 10 April 2026) menyatakan batas ukuran saat memformat volume FAT32 lewat command line naik dari 32 GB ke 2 TB. Liputan independen PCWorld dan TechSpot mengonfirmasi angka yang sama dan menyebut build Beta yang cocok (26220.8165). Perubahan memakai utilitas bawaan format; dialog format grafis masih menerapkan batas lama 32 GB. FAT32 sendiri selalu bisa mendeskripsikan volume lebih besar - Windows menolak membuatnya. Membaca volume FAT32 lebih besar yang dibuat di tempat lain sudah berfungsi di banyak build Windows.

Why it matters for everyday files

Orang masih memformat USB sebagai FAT32 agar kamera, head unit mobil, TV, atau Mac dan PC teman bisa melihat stick yang sama. Batas lama 32 GB di UI/CLI Windows mendorong banyak pengguna ke exFAT atau formatter pihak ketiga saat stick lebih besar. Jalur CLI 2 TB memungkinkan tetap kompatibel FAT32 di media lebih besar tanpa meninggalkan Windows - selama tidak ada satu file yang perlu melebihi 4 GiB. Batas ukuran file itu tidak berubah dan masih merusak dump video besar, image disk, dan ZIP berat. Saat transfer gagal di stick yang "cukup besar", periksa ukuran file dulu, lalu format volume.


What to do with your files right now

Sebelum memformat ulang drive yang sudah berjalan:

  • Pastikan Anda di build Insider yang memuat perubahan (Dev 26300.8170 / Beta 26220.8165 atau lebih baru). Windows 11 stabil mungkin masih memaksa 32 GB sampai fitur ini meluas.
  • Gunakan Command Prompt atau Terminal dengan perintah bawaan format untuk volume FAT32 di atas 32 GB; jangan berharap dialog File Explorer menawarkan batas baru sekarang.
  • Ingat aturan 4 GiB per file - pecah atau kompres ulang video dan arsip besar, atau pindah ke exFAT/NTFS ketika semua perangkat dalam rantai mendukungnya (lihat catatan batas 4 GiB FAT32).
  • Lebih baik ZIP saat berbagi banyak file kecil antar mesin - alat seperti ZIP file dan Unzip file tidak bergantung pada format volume setelah arsip muat di bawah batas per file tujuan.
