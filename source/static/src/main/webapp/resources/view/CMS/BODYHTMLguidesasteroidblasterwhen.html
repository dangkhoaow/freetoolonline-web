When to Play Asteroid Blaster

Asteroid Blaster fits when you want a keyboard Canvas space shooter in the browser: ~245 KB, rock splitting, screen wrap, and an occasional alien - no install and no save file.

Last reviewed: 2026-07-18

When you have a keyboard

Left/right rotate, up thrusts, space fires. Pause with P, mute with M. Touch-only phones are not the target.

When you want a short arcade run

Waves grow from 2 rocks up to 12. Two spare ships. Closing the tab ends the score - there is no localStorage save.

When a tiny download is enough

About 245 KB total, cached after the first load. No network calls after load, no accounts inside the game frame.

When to pick something else

Want a WebGL Underrun run at ~120 KB? Try Retro Arcade Shooter. Want a 3D marble tilt at ~1.7 MB? Use Marble Maze.
