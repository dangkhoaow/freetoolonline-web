Image EXIF Metadata Viewer Cuando Usarlo

Last reviewed: 2026-07-21

Abre Image EXIF Metadata Viewer cuando necesites ver etiquetas de camara, fecha y GPS en una foto antes de compartirla - sin subir el archivo ni instalar ExifTool.


Cuando aplica

Revisar una foto del telefono antes de publicarla. Confirmar si un JPEG aun tiene fecha de captura. Detectar GPS en una imagen de viaje. Preferir un parse local cuando el archivo debe quedarse en este dispositivo.


Que esperar

Elige una imagen, pulsa Read Metadata y revisa Camera, Date / Time, GPS y Other. Si hay GPS, aparece aviso de privacidad y enlace OpenStreetMap; puedes descargar un JPEG sin metadatos. Todo en el navegador - sin subida y sin cuenta.

Para los pasos exactos, ve la guia paso a paso.
