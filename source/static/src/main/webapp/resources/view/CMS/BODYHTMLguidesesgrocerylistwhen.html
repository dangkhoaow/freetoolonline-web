Grocery List Cuando Usarlo

Last reviewed: 2026-07-09

Vale la pena abrir Grocery List cuando necesitas una ejecucion rapida y puntual sin instalar nada. Esta pagina cubre cuando eso tiene sentido y cuando no.


Cuando tiene sentido

Grocery List funciona bien cuando necesitas una lista de compras rapida y puntual, sin instalar una app ni crear una cuenta.


Que esperar

Grocery List es una lista de verificacion minima basada en el navegador: cada elemento tiene una casilla para marcarlo como hecho y un boton para quitarlo.

La lista se guarda en este navegador mediante local storage, asi que sobrevive a una recarga de la pagina; borrar los datos del navegador de este sitio la elimina.
