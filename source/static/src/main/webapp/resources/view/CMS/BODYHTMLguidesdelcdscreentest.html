LCD-Bildschirmtest

Fuehren Sie einen kostenlosen LCD-Bildschirmtest im Browser in unter einer Minute durch. Waehlen Sie eine Farbkachel - rot, gruen, blau, schwarz, weiss oder gelb - und die Seite malt diese Volltonfuellung ueber das gesamte Viewport, damit ein festsitzender Pixel, ein toter Pixel oder Hintergrundbeleuchtungs-Lichthof vor einem einheitlichen Feld auffaellt. Keine Installation, kein Konto, kein Upload.

Unter den Farbkacheln meldet die Seite drei Live-Zahlen ueber Ihr Display: das aktive Viewport, die vom Betriebssystem angegebene Bildschirmaufloesung und die native Aufloesung, die diese Aufloesung mit dem Geraete-Pixelverhaeltnis multipliziert. Bestaetigen Sie, dass die native Zahl mit dem Datenblatt Ihres Laptops oder Monitors uebereinstimmt, bevor Sie einen verdaechtigen Punkt markieren, denn auf einem Hi-DPI-Panel sitzt ein CSS-Pixel vor mehreren physischen Sub-Pixeln, und ein festsitzender Sub-Pixel kann sich dazwischen verbergen. Die Anzeige aktualisiert sich bei jedem Laden, sodass eine frische Pruefung nur einen Reload entfernt ist.

Wenn Sie zuerst bestimmen muessen, ob Ihr Display LED oder LCD ist (der Unterschied beeinflusst, welches Farbmuster Defekte am besten sichtbar macht), lesen Sie LED-Test vs. LCD-Test - was auf Ihren Bildschirm zutrifft.
