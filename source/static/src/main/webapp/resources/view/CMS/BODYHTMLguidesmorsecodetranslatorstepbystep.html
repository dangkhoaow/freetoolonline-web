Morse Code Translator Step By Step

Last reviewed: 2026-07-21

Open Morse Code Translator, paste plain text or Morse into the left pane, click convert, then copy the result from the right pane.


Steps

  1. Paste or type plain text (encode) or dots and dashes (decode) into the input box.
  2. Click the red convert button.
  3. Read the output: letters become spaced Morse codes; words use a slash separator; Morse input becomes text.
  4. Click Copy Output and paste the result where you need it.

What stays private

Translation runs in the browser. Your text is not uploaded.
