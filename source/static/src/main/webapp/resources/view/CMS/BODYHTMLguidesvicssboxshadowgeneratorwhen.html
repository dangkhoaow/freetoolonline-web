CSS Box-Shadow Generator - Khi Nào Dùng

Last reviewed: 2026-07-24

Dùng CSS Box-Shadow Generator khi bạn muốn bóng đổ ra ngoài hoặc bóng bên trong (inset) cho một thẻ, nút, hoặc bảng, và muốn kéo thanh trượt thay vì đoán giá trị pixel. Trang này nói khi nào công cụ này phù hợp và khi nào công cụ khác phù hợp hơn.


Khi phù hợp

Bạn đang tạo kiểu cho một phần tử - thẻ, nút, hộp thoại, hoặc khung ảnh - và muốn thấy bóng thay đổi theo thời gian thực khi chỉnh độ lệch, độ mờ, độ lan, màu sắc, và độ trong suốt, rồi sao chép một dòng box-shadow vào tệp CSS của bạn.


Khi không phù hợp

Nếu bạn cần nền gradient nhiều màu kèm tiền tố trình duyệt thay vì đổ bóng, hãy dùng CSS Gradient Generator - đó là một thuộc tính CSS khác, dành cho một hiệu ứng hình ảnh khác.
