One thing to check first: this pulls out text that is already embedded in the PDF, which most files exported from a word processor contain. If your PDF is a scan or a photo of a page - where you cannot select the text with your cursor - extraction returns nothing, because there is no text layer to read. Those files need OCR (optical character recognition) first, a separate step this tool does not perform. A quick test: open the source PDF and try to highlight a sentence. If the cursor selects words, extraction will work. Formatting is dropped too - bold, headings, and tables come out as plain lines - so you get text to paste and reformat, not a styled copy.
