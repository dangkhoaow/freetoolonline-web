Sun Structure 3D Explorer paso a paso

Abre el Sun Structure 3D Explorer, resalta cada capa y lee nucleo ~15,7 millones K, fotosfera ~5.772 K y corona ~1-3 millones K.

  1. Abre /space-3d/sun-structure.html y espera los hemisferios en corte y el halo coronal.
  2. Lee el panel: nucleo ~15,7 millones K, zona radiativa, convectiva, fotosfera ~5.772 K, corona.
  3. Pulsa Core para la region de fusion en el centro.
  4. Pulsa Radiative o Convective para capas de transporte de energia.
  5. Pulsa Photosphere o Corona para ~5.772 K y ~1-3 millones K, o All para el corte completo.
