Add Page Numbers To Pdf When To Use

Last reviewed: 2026-07-22

Add Page Numbers to PDF is worth opening when you need a quick, one-off run without installing anything. This page covers when that fits and when it does not.


When it fits

Applies one fixed bottom-center style and numbering sequence - does not support custom position, custom text/format, skipping a numbered cover page, or restarting the count partway through the document.


What to expect

The file is processed in this browser tab and is never uploaded to a server.

To split, merge, rotate, delete pages from, or watermark the PDF instead, see the related-tools section.
