ZIP-Groesse online verkleinern - Kostenloser Leitfaden

Brauchen Sie ein kleineres ZIP vor dem Versand per E-Mail oder dem Upload in die Cloud? Waehlen Sie Ihre Dateien aus, waehlen Sie eine Komprimierungsstufe und laden Sie ein einziges Archiv herunter - ohne Installation, ohne Konto. Funktioniert im Browser, wobei die Dateien bis zum Download auf Ihrem Geraet bleiben.

Verwendetes Werkzeug/zip-file.html

Beginnen Sie mit einem Ordner oder einer Handvoll Dateien statt mit einem ganzen Laufwerk. Bei der Komprimierung Normal haelt das Werkzeug Groesse und Geschwindigkeit im Gleichgewicht; Maximum presst staerker zusammen, dauert aber bei grossen Ordnern laenger. Wenn das Archiv auf aelteren Windows-Versionen geoeffnet werden muss, bleiben Sie beim Standard-ZIP-Format. Pruefen Sie nach dem Download eine Datei im Inneren, bevor Sie die Originale loeschen. Fuer Passwortschutz oder Archive mit mehreren Formaten nutzen Sie das spezielle ZIP-Tools-Center statt dieses schnellen Verkleinerungsschritts.
