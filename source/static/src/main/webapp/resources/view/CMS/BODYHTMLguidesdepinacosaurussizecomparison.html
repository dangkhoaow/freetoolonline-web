What the Pinacosaurus size numbers are not

What these size numbers are not is a single settled length or mass for every Pinacosaurus. The fossil record is juvenile-heavy, so the page cites about 4-5 m and roughly 1,000-2,100 kg alongside Paul adult values. Soft-tissue color is still an artistic reconstruction.
