Gerador de Layout CSS Grid - Passo a Passo

Last reviewed: 2026-07-23

O Gerador de Layout CSS Grid monta um bloco CSS display: grid pronto para colar em tres passos: definir o numero de trilhas, definir os espacamentos e copiar o resultado. Nada e enviado e nada precisa ser instalado.


Passo 1 - defina colunas e linhas

Abra o Gerador de Layout CSS Grid. Arraste o controle deslizante Columns (1 a 12) e o controle Rows (1 a 6) para corresponder ao layout desejado - a grade de previsualizacao a direita redesenha as celulas imediatamente.


Passo 2 - defina os espacamentos

Arraste os controles Column gap e Row gap (0px a 50px cada) para controlar o espaco entre as celulas. A grade de previsualizacao e o texto CSS gerado abaixo atualizam ao vivo com os valores exatos em pixels.


Passo 3 - copie o resultado

Clique em "Copy CSS" para copiar o bloco gerado para a area de transferencia. Se o navegador bloquear o acesso a area de transferencia, uma mensagem de status pede que voce selecione e copie o texto manualmente. Toda a pagina roda no navegador, entao nenhum valor dos controles deslizantes e enviado a um servidor.
