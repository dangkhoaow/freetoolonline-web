Cómo Usar Europa Subsurface Ocean 3D Explorer

Last reviewed: 2026-07-23

Desliza el control Peel shell de 0 a 100 por ciento y lee las cifras reales publicadas en el panel de datos a medida que el hielo, el océano, el manto y el núcleo se exponen en orden.

  1. Abre Europa Subsurface Ocean 3D Explorer y espera a que se dibuje la luna helada intacta.
  2. Arrastra para orbitar la vista; mantén visible el panel de datos a la derecha.
  3. Desliza Peel shell más allá del 25 por ciento y observa cómo la cuña de la corteza de hielo se abre, revelando la capa oceánica completa debajo.
  4. Sigue deslizando más allá del 50 y el 75 por ciento para exponer el manto rocoso y luego el núcleo metálico, una capa a la vez.
  5. Marca la casilla Clipper flyby tracks para ver seis arcos ilustrativos de trayectoria sobre la corteza de hielo.
  6. Reinicia Peel shell a 0 y usa los cuatro botones rápidos - Ice shell, Ocean, Mantle, Core - para ir directo a cualquier capa.
