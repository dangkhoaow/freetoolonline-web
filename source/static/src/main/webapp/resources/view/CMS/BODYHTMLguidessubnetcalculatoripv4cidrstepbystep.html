Subnet Calculator (ipv4 Cidr) Step By Step

Last reviewed: 2026-07-24

Subnet Calculator (IPv4 CIDR) runs in two steps: open the tool, then type in your address and prefix. This page walks through what each step expects.


Step 1 - open the tool

Go to Subnet Calculator (IPv4 CIDR). Nothing needs to be installed or signed in before you start.


Step 2 - enter the address and prefix

Type an IPv4 address into the first field (for example 192.168.1.0) and a CIDR prefix length from 0 to 32 into the second (for example 24). Both fields have a default value pre-filled, so the page shows a real result before you type anything.


Step 3 - read the result

The network address, broadcast address, subnet mask, wildcard mask, usable host range and total/usable host counts appear below the two fields and update on every keystroke - there is no separate "calculate" button and nothing to copy or download.

The calculation runs locally in the browser - no upload and no account.
