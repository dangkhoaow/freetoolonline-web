Hash Generator (sha-256/512) Passo A Passo

Last reviewed: 2026-07-16

Hash Generator (SHA-256/512) funciona em tres passos: abrir a ferramenta, digitar ou colar seu texto, depois escolher um algoritmo e copiar o digest. Esta pagina detalha o que cada passo espera.


Passo 1 - abrir a ferramenta

Va ate Hash Generator (SHA-256/512). Nao e preciso instalar nada nem fazer login antes de comecar.


Passo 2 - digitar ou colar seu texto

Digite ou cole o texto que quer hashear na caixa; depois escolha SHA-1, SHA-256, SHA-384 ou SHA-512 no menu suspenso. O digest atualiza em tempo real abaixo enquanto voce digita.


Passo 3 - copiar o digest

Clique em Copy para pegar o digest hexadecimal resultante. O calculo do hash roda totalmente no navegador via a WebCrypto API - nada e enviado a um servidor.

MD5 nao esta disponivel aqui porque navegadores modernos o omitem por ser inseguro; use a ferramenta relacionada md5-converter para esse algoritmo especifico.
