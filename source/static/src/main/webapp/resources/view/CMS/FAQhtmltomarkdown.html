Frequently Asked Questions

What does HTML to Markdown Converter do?

It converts pasted HTML into Markdown text in your browser using Turndown, then lets you copy the result.

When should I use it?

When a README, wiki, or CMS field wants Markdown but you only have HTML from email, a docs export, or a rich-text editor.

Does my HTML leave this device?

No. After the Turndown script loads from this site, conversion stays in the tab. Closing the tab discards the text.

What about the reverse direction?

Use the related Markdown to HTML previewer from the related-tools section when you need sanitized HTML from Markdown instead.
