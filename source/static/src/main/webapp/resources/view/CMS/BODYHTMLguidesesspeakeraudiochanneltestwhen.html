Speaker & Audio Channel Test Cuando Usar

Last reviewed: 2026-07-21

Abra Speaker & Audio Channel Test cuando necesite confirmar los canales izquierdo y derecho - por ejemplo tras conectar auriculares, cambiar altavoces Bluetooth o revisar stereo antes de una llamada.


Cuando encaja

Uselo para una comprobacion rapida L / R / ambos en el navegador. Pulse Play Left, Play Right o Play Both; use auriculares si necesita un pan claro. Ajuste la frecuencia y luego Stop.


Cuando usar otra cosa

Esta pagina no mide volumen absoluto en dB ni calibra monitores de estudio. Para el microfono, use la prueba de microfono del sitio. Para barridos largos, una utilidad de escritorio encaja mejor.


Privacidad

Todo corre en el navegador - sin microfono, sin subida y sin cuenta. El audio usa StereoPannerNode de Web Audio API para tocar solo el canal elegido.
