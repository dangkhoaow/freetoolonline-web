Der CSS Box-Shadow Generator ist eine von drei gängigen Wegen, einen Schatten zu formen: ein Browser-Tool wie dieses, das Schattenpanel einer Design-App (Figma, Sketch), oder die festen Preset-Klassen eines CSS-Frameworks. Jeder Weg wägt Geschwindigkeit, Präzision und Aufwand unterschiedlich ab.
