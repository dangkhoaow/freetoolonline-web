Cuándo Usar el Explorador 3D de Chorros Relativistas y Efecto Doppler - Mejores Sesiones y Límites

Última revisión: 2026-07-24

Abre este explorador cuando quieras ver por qué el chorro de un agujero negro parece unilateral desde la Tierra, o para enseñar el efecto Doppler y el movimiento superluminal aparente con los números reales de M87 - no para ejecutar una simulación magnetohidrodinámica de lanzamiento de chorro.


Una lección sobre el efecto Doppler

Una lección comienza en la base de reinicio (sin efecto Doppler) para ver dos chorros igualmente brillantes, luego carga el ajuste "M87" para observar cómo un chorro se ilumina y el otro casi desaparece, leyendo el factor Doppler calculado y la velocidad aparente en el panel de datos en cada paso.


Sesiones para las que este explorador no sirve

Las sesiones para las que esta página no sirve son explorar la geometría exacta de órbita de fotones e ISCO de Kerr - usa el Explorador 3D de Esfera de Fotones e ISCO para eso - o ver un resplandor artístico de disco de acreción sin física de efecto Doppler - usa el Explorador 3D de Agujero Negro.
