Tra loi nhanh. Mo Nen Video, chon muc nen (Can bang la muc mac dinh), chon file video cua ban, roi doi lien ket tai xuong - file khong bao gio roi khoi trinh duyet cua ban.

Buoc 1 - mo cong cu

Mo Nen Video. Khong can cai dat hay dang nhap gi truoc khi bat dau.

Buoc 2 - chon muc va file

Chon mot trong ba muc nen - Nho hon, Can bang, hoac Chat luong cao hon - roi chon file video cua ban. Chi cac dinh dang video mp4/mov/avi/mkv/webm duoc chap nhan; file chi co am thanh se bi tu choi voi mot loi ro rang.

Buoc 3 - doi, roi tai xuong

FFmpeg.wasm ma hoa lai toan bo file ngay trong tab trinh duyet cua ban; cac file 4K lon hoac toc do bit cao co the mat vai phut vi chay mot luong tren thiet bi cua ban, khong phai tren server. Khi xong, mot lien ket tai file da nen xuat hien, voi hau to "-compressed" va giu nguyen phan mo rong file.

Frequently Asked Questions

Toi co can tao tai khoan de dung khong?

Khong. Cong cu chay truc tiep trong trinh duyet, khong can cai dat hay dang nhap.

File cua toi co duoc gui di dau khong?

Khong. File luu tren thiet bi cua ban suot thoi gian va khong bao gio duoc tai len mot server.

Toi co the chon dung luong dau ra chinh xac khong?

Khong. Chi co ba muc chat luong co dinh - khong co dung luong hay toc do bit chinh xac.
