Video Splitter

Last reviewed: 2026-07-17

Enter one split point (in seconds, or mm:ss) and this tool cuts your video into two separate clips at that exact spot - everything before, and everything after. Both clips keep the original video and audio codecs, since the cut runs as a lossless copy rather than a re-encode, so there is no quality loss and no long wait.

Everything happens inside this browser tab via FFmpeg.wasm - the file you upload is never sent anywhere. Each resulting clip downloads as its own separate file; the source file is not retained after processing.
