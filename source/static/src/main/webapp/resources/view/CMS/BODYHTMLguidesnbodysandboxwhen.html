When to use N-Body Sandbox 3D Explorer

Use the N-Body Sandbox 3D Explorer when you need a free-form gravity playground - and skip it when you only need a fixed choreography or a static potential-well diagram.

  • Classroom demos of emergent orbits, scatters, and captures
  • Showing that chaos increases with more bodies
  • Introducing the real value of Newtonian G

This page uses a scaled gravitational constant for visible motion and is not a general-purpose ephemeris.
