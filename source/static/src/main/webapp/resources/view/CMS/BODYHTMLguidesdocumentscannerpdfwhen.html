Document Scanner (to Pdf) When To Use

Last reviewed: 2026-07-13

Document Scanner (to PDF) is worth opening when you need a quick, one-off run without installing anything. This page covers when that fits and when it does not.


When it fits

Reach for it when you have a few photos of a paper document - a receipt, a form, a signed page - and want them combined into one PDF to send or file, without installing a scanner app or signing up for a service.


What to expect

Choose the photos (the file picker also offers your camera on mobile), optionally check "Enhance for text" for a grayscale and contrast boost, then click Scan to PDF. Everything runs locally in this browser tab; photos are never uploaded to a server.

Each page is saved as a JPEG image at high quality, not vector text, so a stack of several full-resolution phone photos can add up to a multi-megabyte PDF - worth checking the file size before you email a long scan.

This captures and combines photos into a PDF - it does not run OCR (the pages are images, not searchable text) and it does not auto-detect or crop to the document's edges, so frame the paper before you take the photo.
