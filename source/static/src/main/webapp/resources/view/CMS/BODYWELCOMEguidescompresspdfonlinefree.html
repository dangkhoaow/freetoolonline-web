Before reaching for a converter, work out why the file is heavy. A form or contract that grew from stacked layers usually slims down once you flatten it, with the page looking unchanged. A scan that ballooned is really a stack of photos, so the honest fix is to pull the pages out as images, shrink each one, and build a fresh PDF - knowing that step trades selectable text for a flat picture.

If you are not sure which problem you have, inspect the file first with preflight PDF - it shows whether stacked layers or page images are carrying the weight, so you choose flatten or the rebuild route without guessing. Whichever route you take, keep the untouched original on hand: it is the only copy that still holds selectable, searchable text once a rebuild has run.

When the heavy file is a filled-in form or a marked-up contract, flattening is the route that pays off: it folds the form fields, the comment annotations, and any stacked layers down into a single printed page, so the saved copy carries one flat surface instead of an editable stack. The text you can already read stays put and the page looks the same, which is why a form-bloated PDF can shed weight without anyone noticing a visual change.

Every step here is free and needs no account, which is the honest version of the offer: these are free online tools, not a private in-browser converter. The work happens on a server, so you upload the file, the tool compresses or rebuilds it, and the smaller copy comes back for you to download once it is ready.
