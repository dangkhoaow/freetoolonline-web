Pomodoro Timer Online Langkah Demi Langkah

Last reviewed: 2026-07-11

Pomodoro Timer Online punya tiga langkah: atur menit Work/Break, klik Start, dan biarkan fase bergantian. Halaman ini menjelaskan setiap langkah.


Langkah 1 - atur menit Work dan Break

Buka Pomodoro Timer Online. Dua kolom angka sudah siap sebelum kamu mulai: Work (1-120 menit, bawaan 25) dan Break (1-60 menit, bawaan 5). Ubah salah satunya kalau pembagian standar 25/5 tidak cocok untuk sesimu.


Langkah 2 - klik Start

Klik Start untuk memulai interval Work. Label fase menunjukkan 'Work' dan hitungan mundur berjalan turun dari menit Work yang dipilih. Pause menahan hitungan tanpa mengubah fase; klik Start lagi untuk melanjutkan.


Langkah 3 - biarkan bergantian antara Work dan Break

Saat hitungan Work mencapai nol, label fase berubah menjadi 'Break' dan hitungan dimuat ulang dari menit Break; saat Break berakhir kembali ke Work, dan ini berulang selama kamu membiarkan tab tetap terbuka. Reset kapan pun akan mengembalikan ke awal interval Work saat ini.

Semuanya berjalan lokal di tab browser ini - tanpa unggah dan tanpa akun. Menutup atau memuat ulang tab akan menghentikan timer; timer tidak terus berjalan atau memberi notifikasi di latar belakang.
