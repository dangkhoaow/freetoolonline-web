Datei zu ZIP

Eine einzelne Datei in ein .zip-Archiv umzuwandeln ist ein kurzer Schritt im Ersteller zip-file: Datei in das Upload-Feld ziehen, auf Zip klicken, und das Tool gibt eine .zip zurueck, die Sie per E-Mail senden, anhaengen oder hochladen koennen. Ob das Ergebnis kleiner als die Eingabe ausfaellt, haengt vom Format ab - Text und Code schrumpfen 60-80%, waehrend Fotos und PDFs kaum kleiner werden, weil sie bereits vom eigenen Codec komprimiert wurden.

Die Wirksamkeit der Komprimierung haengt vom Inhalt der Datei ab. Text, Code, JSON und CSV schrumpfen typischerweise 60-80%; Office-Dokumente 10-25%; PDFs 5-15%; bereits komprimierte Fotos (JPG, PNG, HEIC) nur 0-5% - fuer diese zuerst mit einem Bild-Tool komprimieren und dann zippen. Die Tabelle "Reduce ZIP file size - what to expect" auf der Tool-Seite zeigt die Rechnung fuer gaengige Ziele wie die 25-MB-Grenze fuer E-Mail-Anhaenge. Fuer sensible Inhalte oeffnen Sie das Settings-Feld im Ersteller und vergeben ein Passwort vor dem Zippen - drei Verschluesselungsstufen stehen zur Verfuegung (Standard, AES-128, AES-256). Senden Sie das Passwort ueber einen anderen Kanal als das Archiv.
