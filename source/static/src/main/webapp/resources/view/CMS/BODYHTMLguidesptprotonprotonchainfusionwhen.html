Quando Usar o Proton-Proton Chain Fusion 3D Explorer - Melhores Sessões e Limites

Última revisão: 2026-07-23

Abra o Proton-Proton Chain Fusion 3D Explorer quando precisar da sequência real de 3 etapas que funde hidrogênio em hélio no núcleo do Sol, com a energia real liberada em cada etapa - não uma visão geral do Sol.


Uma lição sobre a química da reação

Uma lição começa quando você percorre cada uma das 3 reações em ordem e lê quanta energia cada uma libera, depois observa o total líquido de uma cadeia completa - uma forma clara de mostrar por que a fusão acontece em etapas, e não de uma só vez.


Sessões para as quais este explorador não serve

Sessões para as quais esta página não serve são lições sobre a geografia interna em camadas do Sol - use o Sun Structure 3D Explorer para isso - ou sobre quanto tempo a taxa de fusão de uma estrela permite que ela queime - use o Main Sequence Lifetime 3D Explorer. Percorra os controles no guia passo a passo.
