Quando Usar o Explorador 3D de Esfera de Fótons e ISCO - Melhores Sessões e Limites

Última revisão: 2026-07-24

Abra o Explorador 3D de Esfera de Fótons e ISCO quando quiser ver exatamente como a rotação de um buraco negro remodela a geometria da órbita de fóton e do ISCO, ou para ensinar a regra prógrado/retrógrado com fórmulas reais - não para executar uma simulação de relatividade numérica ou traçar a trajetória real de um fóton.


Uma lição de rotação crescente

Uma lição começa em a*=0 (Schwarzschild) para ler os valores clássicos de 1,5 Rs / 3 Rs, depois arrasta o controle de rotação em direção ao limite de Thorne (a*=0,998) para ver os anéis prógrados se aproximarem do horizonte e os anéis retrógrados se afastarem, lendo os números exatos no painel de dados a cada passo.


Sessões para as quais este explorador não serve

Sessões para as quais esta página não serve são visualizar um brilho artístico de disco de acreção sem controle de rotação - use o Explorador 3D de Buraco Negro para isso - ou explorar uma grade didática geral de curvatura do espaço-tempo - use o Explorador 3D de Curvatura do Espaço-Tempo. Percorra os controles no guia passo a passo.
