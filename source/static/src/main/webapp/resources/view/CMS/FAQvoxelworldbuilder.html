Frequently Asked Questions

What does Voxel World Builder do?

Click the ground or any block face to place the selected block; switch to Remove mode to take blocks away - a starter cottage is on the lot so you are never staring at an empty field.

How is my world saved?

The world autosaves to this browser as you build, and Export downloads it as a JSON file so a build you care about survives a cleared cache.

How big can I build?

The build area is a finite 48x48 lot with a height cap of 32 - room for real builds while staying fast on phones.
