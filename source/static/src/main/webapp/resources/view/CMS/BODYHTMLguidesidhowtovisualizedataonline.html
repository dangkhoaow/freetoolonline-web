Cara memvisualisasikan data online - buat grafik dari angka Anda

Alat visualisasi data mengubah baris angka menjadi grafik di browser Anda. Tempel CSV, JSON, atau baris sederhana Label, Nilai, pilih jenis grafik, dan lihat langsung tergambar. Batang, garis, lingkaran, dan lainnya hanya satu klik. Semua berjalan di perangkat Anda, tanpa unggah dan tanpa akun.

Langkah

  1. Tempel data Anda sebagai CSV dengan baris header, JSON, atau satu pasangan "Label, Nilai" per baris - atau muat contoh bawaan dengan satu klik.
  2. Pilih jenis grafik: batang, garis, lingkaran, donat, area polar, radar, sebar, gelembung, atau batang bertumpuk.
  3. Beri judul pada grafik lalu klik Visualisasikan; grafik tergambar ulang saat Anda mengubah jenis, judul, atau data.
  4. Buka kembali grafik terbaru dari daftar - grafik terbaru diingat di browser ini.

Format masukan

Tiga format masukan dideteksi otomatis. CSV memakai baris header dengan pemisah koma atau baris baru. JSON bisa berupa array objek (kunci nama plus kunci nilai numerik) atau objek kunci/nilai sederhana. Teks sederhana adalah satu pasangan "Label, Nilai" per baris. Contoh di bawah menggambar lima nilai sebagai grafik batang.

Wawasan AI di perangkat

Pada browser dengan WebGPU Anda dapat menjalankan wawasan AI privat opsional. Model bahasa kecil diunduh sekali dan berjalan di perangkat Anda, membaca data Anda, dan membangun ulang grafik - tidak ada yang diunggah. Tanpa WebGPU mesin grafik bekerja sendiri, jadi alat selalu menggambar data Anda.

Privasi

Semua berjalan di browser Anda. Tidak ada widget unggah dan tidak ada akun; grafik terbaru hanya ada di perangkat Anda dan data Anda tidak pernah meninggalkan halaman.
