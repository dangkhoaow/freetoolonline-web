Como remover metadados de foto no browser

  1. Abra Image Metadata (EXIF) Remover e escolha uma imagem local.
  2. Opcional: clique Read Metadata e veja Camera, Date / Time, GPS e Other.
  3. Se aparecer aviso de GPS, abra o link OpenStreetMap e depois continue.
  4. Clique Download Without Metadata para guardar um JPEG a 0.95 sem EXIF/XMP.
  5. Clear limpa preview e tabelas. Nada e enviado para o servidor.
