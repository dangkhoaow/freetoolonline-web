How the Ornithomimus size-vs-human toggle stays honest

The Ornithomimus model is drawn to a fixed on-screen length so it fits the canvas; the Size vs human control then places a 1.8 m person at the true length ratio used by the viewer (3.8 m). Tap Size vs human under the canvas to show or hide the scale figure - the comparison is proportional, not a decorative sticker. Published length still spans about 3.3-3.8 m, so the toggle picks one clear scale inside that disclosed range.

What the Ornithomimus size numbers are not

What these size numbers are not is a claim of one precise length or weight: the Ornithomimus size comparison discloses the about 3.3-3.8 m / 105-170 kg ranges and uses 3.8 m only for the Size vs human scale; it does not invent soft-tissue color accuracy, and this is not a win/lose game.
