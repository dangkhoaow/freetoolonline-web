CSS Box-Shadow Generator - Quando Usar

Last reviewed: 2026-07-24

Use o CSS Box-Shadow Generator quando quiser uma sombra projetada ou uma sombra interna (inset) em um cartão, botão ou painel e preferir arrastar controles deslizantes em vez de adivinhar valores em pixels. Esta página mostra quando ele se encaixa e quando outra ferramenta se encaixa melhor.


Quando se encaixa

Você está estilizando um único elemento - um cartão, botão, modal ou moldura de imagem - e quer ver a sombra mudar em tempo real ao ajustar deslocamento, desfoque, espalhamento, cor e opacidade, depois copiar uma linha box-shadow para sua folha de estilos.


Quando não se encaixa

Se você precisa de um fundo em gradiente multicolorido com prefixos de navegador em vez de uma sombra, use o CSS Gradient Generator - é uma propriedade CSS diferente, feita para um efeito visual diferente.
