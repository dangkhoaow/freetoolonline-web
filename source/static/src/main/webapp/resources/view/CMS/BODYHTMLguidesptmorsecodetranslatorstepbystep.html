Morse Code Translator - Passo a Passo

Last reviewed: 2026-07-21

Abra o Morse Code Translator, cole texto ou Morse à esquerda, clique em converter e copie o resultado à direita.


Passos

  1. Cole texto simples (codificar) ou pontos e traços (decodificar) na caixa.
  2. Clique no botão vermelho de converter.
  3. Leia a saída: letras viram códigos Morse; palavras usam barra; Morse vira texto.
  4. Clique em Copy Output e cole onde precisar.

Privacidade

A tradução roda no navegador. Seu texto não é enviado.
