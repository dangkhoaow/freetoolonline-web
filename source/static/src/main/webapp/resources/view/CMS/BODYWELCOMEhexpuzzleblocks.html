Hex Puzzle Blocks

Last reviewed: 2026-07-09

Press Play and colored blocks start falling toward a hexagon sitting in the middle of the screen. Rotate the hexagon so a falling block lands next to two or more blocks of the same color - once 3 or more matching blocks touch, they clear and add to your score. Blocks keep arriving from all six sides in shifting patterns, and the pace speeds up gradually the longer you last, so the puzzle is part quick color-matching and part staying ahead of the incoming waves.

Controls at a glance:

ActionInput
Rotate hexagonLeft / Right arrow, A / D, or click / tap either half of the screen
Speed up temporarilyDown arrow or S (hold)
Pause / resumeP, Space, or the pause icon
RestartEnter, or the restart icon

Clearing a match within a short window of a previous one builds a combo multiplier, shown as colored lines running around the outer hexagon that drain away as the window closes. Chaining matches before the lines empty keeps the multiplier - and the points per clear - climbing. A run ends when blocks pile up past the hexagon's edge on any one of its six sides without clearing.

Progress mid-game and your top 3 scores are kept in this browser's local storage, so closing the tab and coming back later can resume where you left off; nothing is sent to a server or tied to an account, and progress does not follow you to a different browser or device.

The game is about 1.1 MB - a 2D canvas puzzle engine plus a small UI icon set and font, downloaded once from this site and then cached; it runs in any modern browser, including touchscreens, with no WebGL requirement. It runs on Hextris, an open-source puzzle game originally released by Logan Engstrom and Garrett Finucane under the GNU General Public License v3.0 - the license file ships next to the game files.
