Konverter GIF ke Frame

Konverter GIF ke frame mendekode setiap siklus animasi dari GIF animasi dan menyimpan setiap frame sebagai gambar terpisah yang bisa diunduh, diedit, atau digunakan dalam alat desain.

Upload GIF, pilih PNG atau JPG sebagai format output dan konverter mengembalikan setiap frame animasi sebagai file gambar tersendiri. PNG mempertahankan transparansi palet; JPG menghasilkan file lebih kecil tetapi meratakan area transparan. Tidak perlu instalasi.

Memilih antara PNG dan JPG

PNG adalah pilihan yang tepat ketika GIF menggunakan transparansi palet - frame PNG yang diekstrak mempertahankan saluran transparansi tersebut. JPG adalah pilihan yang tepat ketika ukuran file lebih penting daripada transparansi piksel sempurna, karena menghasilkan file yang jauh lebih kecil per frame.
