How to convert an audio file

  1. Pick the target format from the Format dropdown (MP3, WAV, AAC, OGG, FLAC, or M4A).
  2. Drop or select your audio file - conversion starts in the tab using FFmpeg.wasm and nothing is uploaded.
  3. When the status clears, click the download link to save the converted file.
