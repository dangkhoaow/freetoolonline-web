Morse Code Translator - Schritt für Schritt

Last reviewed: 2026-07-21

Öffne Morse Code Translator, füge links Text oder Morse ein, klicke auf Umwandeln und kopiere rechts das Ergebnis.


Schritte

  1. Klartext (kodieren) oder Punkte und Striche (dekodieren) in das Eingabefeld einfügen.
  2. Den roten Umwandeln-Button klicken.
  3. Ausgabe prüfen: Buchstaben werden zu Morse-Codes; Wörter nutzen Schrägstrich; Morse wird zu Text.
  4. Copy Output klicken und einfügen, wo du es brauchst.

Privatsphäre

Die Übersetzung läuft im Browser. Dein Text wird nicht hochgeladen.
