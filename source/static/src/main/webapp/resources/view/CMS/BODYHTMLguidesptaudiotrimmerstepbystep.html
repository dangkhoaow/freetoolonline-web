Cortador de Áudio Passo a Passo

Last reviewed: 2026-07-05

O Cortador de Áudio funciona em três passos: abra a ferramenta, forneça sua entrada e depois baixe o resultado. Esta página explica o que cada passo espera.


Passo 1 - abra a ferramenta

Vá para o Cortador de Áudio. Nada precisa ser instalado ou conectado antes de começar.


Passo 2 - forneça sua entrada

O corte não recodifica o arquivo - a qualidade e o formato originais são mantidos.


Passo 3 - obtenha o resultado

O arquivo é processado nesta aba do navegador e nunca é enviado para um servidor.

Para converter para um formato diferente, veja a seção de ferramentas relacionadas.
