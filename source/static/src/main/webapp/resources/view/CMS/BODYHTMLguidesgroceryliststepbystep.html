Grocery List Step By Step

Last reviewed: 2026-07-03

Grocery List runs in three steps: open the tool, provide your input, then download the result. This page walks through what each step expects.


Step 1 - open the tool

Go to Grocery List. Nothing needs to be installed or signed in before you start.


Step 2 - provide your input

Use it for a quick checklist when you do not want to install an app or sign in.


Step 3 - get the result

Each item gets a checkbox to mark it done and a button to remove it.

The list is saved in this browser via local storage, so it survives a page reload; clearing this site's browser data removes it.
