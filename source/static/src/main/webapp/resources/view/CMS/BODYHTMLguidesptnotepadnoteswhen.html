Bloco de Notas / Notas Quando Usar

Last reviewed: 2026-07-13

Bloco de Notas / Notas vale a pena abrir quando voce precisa de algo rapido e pontual, sem instalar nada. Esta pagina explica quando isso se encaixa e quando nao.


Quando se encaixa

Use como um bloco de rascunho rapido - um numero de telefone, uma linha de texto, ou um lembrete que voce nao quer digitar em um aplicativo permanente.


O que esperar

Cada nota tem um botao Copiar para copiar o texto, e um botao Excluir para remove-la; as notas nao podem ser editadas no local, entao exclua e adicione novamente para alterar uma.

As notas sao salvas neste navegador via armazenamento local, entao elas sobrevivem a um recarregamento da pagina; exporte um backup (JSON) antes de limpar os dados do navegador ou trocar de dispositivo, ja que nenhum dos dois e sincronizado automaticamente.

Cada nota pode ter ate 20.000 caracteres em varias linhas, e pressionar Ctrl/Cmd+Enter salva sem soltar o teclado - so o Enter apenas comeca uma nova linha. Para restaurar um backup, use Importar backup e escolha esse arquivo JSON; isso mescla as notas salvas com sua lista atual em vez de substitui-la, entao qualquer nota que ja esteja no navegador permanece, a menos que o arquivo importado atualize essa mesma nota.
