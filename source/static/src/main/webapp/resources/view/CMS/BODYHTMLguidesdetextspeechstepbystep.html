Text zu Sprache Schritt fuer Schritt

Text zu Sprache wandelt Text in gesprochenes Audio in drei Schritten um: Werkzeug oeffnen, Text eingeben oder einfuegen und auf Play klicken. Diese Anleitung zeigt, was jeder Schritt erfordert.

Schnellantwort. Oeffne Text zu Sprache, fuege den Text in das Eingabefeld ein, waehle eine Stimme und klicke auf Play. Alles laeuft im Browser - keine Datei wird hochgeladen.

Schritt 1 - Werkzeug oeffnen

Oeffne Text zu Sprache. Vor dem Start ist keine Installation oder Anmeldung erforderlich.

Schritt 2 - Text eingeben

Tippe oder fuege den Text in das Eingabefeld ein. Waehle eine Stimme im Selektor - die verfuegbaren Stimmen haengen vom Betriebssystem des Geraets ab. Passe Geschwindigkeit (0.5-2.0), Tonoehe und Lautstaerke nach Bedarf an.

Schritt 3 - Ergebnis anhoeren

Klicke auf Play, um die Wiedergabe zu starten. Der Text wird Wort fuer Wort hervorgehoben, waehrend er gelesen wird. Klicke auf Pause zum Anhalten oder auf Stop zum Abbrechen. Das Audio wird direkt im Browser wiedergegeben - es wird nicht als MP3-Datei gespeichert.

Wenn du die umgekehrte Richtung benoeligst, sieh dir die verwandten Werkzeuge fuer den SPRACHE-zu-TEXT-Konverter an.

Haeufige Fragen

Kann ich das Audio als MP3 herunterladen?

Nein. Das Werkzeug gibt Audio direkt im Browser wieder. Zum Speichern als MP3 waere eine Server-API erforderlich - was einen Upload bedeuten wuerde.

Warum klingen manche Stimmen auf verschiedenen Geraeten unterschiedlich?

Die verfuegbaren Stimmen haengen vom Betriebssystem ab. Derselbe Text kann auf macOS, Windows oder Android unterschiedlich klingen, da jedes System seine eigenen Stimmen hat.
