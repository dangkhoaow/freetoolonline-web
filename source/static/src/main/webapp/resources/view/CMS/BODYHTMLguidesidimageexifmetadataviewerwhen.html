Image EXIF Metadata Viewer Kapan Digunakan

Last reviewed: 2026-07-21

Buka Image EXIF Metadata Viewer saat Anda perlu melihat tag kamera, tanggal, dan GPS pada foto sebelum berbagi - tanpa upload file atau memasang ExifTool.


Kapan cocok

Cek foto ponsel sebelum diposting. Pastikan JPEG masih punya tanggal pengambilan. Deteksi GPS pada foto perjalanan. Pilih parse lokal saat file harus tetap di perangkat ini.


Apa yang diharapkan

Pilih gambar, klik Read Metadata, lalu tinjau Camera, Date / Time, GPS, dan Other. Jika ada GPS, peringatan privasi dan tautan OpenStreetMap muncul; Anda bisa unduh JPEG tanpa metadata. Semua di browser - tanpa upload dan tanpa akun.

Langkah tepat: panduan langkah demi langkah.
