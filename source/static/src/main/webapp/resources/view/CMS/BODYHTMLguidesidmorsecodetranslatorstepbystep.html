Morse Code Translator - Langkah demi Langkah

Last reviewed: 2026-07-21

Buka Morse Code Translator, tempel teks atau Morse di kiri, klik konversi, lalu salin hasil di kanan.


Langkah

  1. Tempel teks biasa (encode) atau titik dan garis (decode) di kotak input.
  2. Klik tombol konversi merah.
  3. Baca keluaran: huruf jadi kode Morse; kata memakai garis miring; Morse jadi teks.
  4. Klik Copy Output dan tempel di tempat yang dibutuhkan.

Privasi

Terjemahan berjalan di peramban. Teks Anda tidak diunggah.
