Doppelte Zeilen Schritt fuer Schritt entfernen: eine Liste im Browser bereinigen

Das kostenlose Tool Remove Duplicate Lines auf dieser Seite entfernt wiederholte Zeilen aus einer eingefuegten Liste, behaelt die erste Kopie jeder Zeile und laesst den Rest der Reihenfolge unveraendert. Diese Anleitung geht die genauen Schritte durch und erklaert, was als Duplikat zaehlt.

Antwort in 30 Sekunden. Oeffnen Sie Remove Duplicate Lines, fuegen Sie Ihre Liste mit einem Eintrag pro Zeile in das obere Feld ein und klicken Sie auf Run. Das Ausgabefeld darunter zeigt dieselbe Liste mit jeder Wiederholung entfernt, behaelt nur das erste Auftreten jeder Zeile und bewahrt die urspruengliche Reihenfolge der verbleibenden Zeilen.

So entfernen Sie die Duplikate

  1. Oeffnen Sie das Tool Remove Duplicate Lines.
  2. Fuegen Sie Ihren Text in das Eingabefeld ein oder tippen Sie ihn, und halten Sie einen Eintrag pro Zeile ein - das Tool behandelt jeden Zeilenumbruch als Grenze zwischen zwei Eintraegen.
  3. Klicken Sie auf Run. Das Tool liest die Liste von oben nach unten, behaelt das erste Auftreten jeder exakten Zeile und verwirft jede spaetere Wiederholung.
  4. Kopieren Sie die bereinigte Liste aus dem Ausgabefeld. Bearbeiten Sie die Eingabe und klicken Sie jederzeit erneut auf Run - das Feld wird zwischen Bearbeitungen nicht gespeichert.

Was als Duplikat zaehlt

Das Tool vergleicht Zeilen als exakten Text, nicht als aehnliche Idee, daher muessen zwei Zeilen Zeichen fuer Zeichen uebereinstimmen, um als derselbe Eintrag zu gelten:

Da die Pruefung Gross-/Kleinschreibung und Leerzeichen beruecksichtigt, verwenden Sie zuerst einen Gross-/Kleinschreibungs-Konverter oder entfernen Sie ueberfluessige Leerzeichen, wenn Ihre Liste unterschiedliche Schreibweisen mischt oder unregelmaessige Abstaende hat und Sie diese Varianten zusammenfassen moechten.

Haeufige Anwendungen

Eine eingefuegte Liste von E-Mail-Adressen bereinigen, bevor Sie sie in ein Versandtool importieren; eine Liste von Tags, Hashtags oder Produkt-IDs auf die eindeutige Menge reduzieren; Duplikate aus Zeilen entfernen, die aus einer Protokolldatei oder einer Tabellenspalte kopiert wurden; oder eine Schluesselwortliste aufraeumen, bevor Sie sie irgendwo einfuegen, das Wiederholungen ablehnt.

Der Weg ueber die Tabellenkalkulation funktioniert gut, wenn die Liste bereits in einer Tabelle steht, bedeutet aber, die Daten an einen Cloud-Dienst hochzuladen, sofern Sie keine Desktop-Anwendung verwenden. Datenschutz

Das Entfernen von Duplikaten laeuft vollstaendig im Tab Ihres Browsers mit einfachem JavaScript. Nichts, was Sie tippen oder einfuegen, wird hochgeladen, an einen Analysedienst gesendet oder auf einem Server gespeichert - ein Neuladen oder Verlassen der Seite leert das Feld, ohne Verlauf und ohne Konto.

Haeufig gestellte Fragen

Welche Kopie einer wiederholten Zeile behaelt das Tool?

Die erste. Das Tool liest Ihre Liste von oben nach unten und behaelt das fruehste Auftreten jeder exakten Zeile, wobei jede spaetere Wiederholung verworfen wird.

Warum wurden "Apple" und "apple" nicht zu einer Zeile zusammengefuehrt?

Der Vergleich unterscheidet Gross- und Kleinschreibung, daher zaehlt eine andere Schreibweise als andere Zeile. Wandeln Sie die Liste zuerst in eine einheitliche Schreibweise um (Grossbuchstaben oder Kleinbuchstaben), wenn diese Varianten als Duplikate behandelt werden sollen.

Aendert das Tool die Reihenfolge der verbleibenden Zeilen?

Nein. Nur die wiederholten Zeilen werden entfernt; die verbleibenden Zeilen behalten dieselbe relative Reihenfolge, die sie in Ihrer urspruenglichen Liste hatten.

Wird mein Text irgendwo hochgeladen oder gespeichert?

Nein. Das Entfernen von Duplikaten laeuft vollstaendig in Ihrem Browser; nichts, was Sie tippen oder einfuegen, wird an einen Server gesendet, und nichts wird gespeichert, sobald Sie die Seite neu laden oder verlassen.
