Bagaimana cara menggabungkan file PDF online gratis?

Buka alat gabung PDF, tambahkan PDF yang ingin Anda satukan, dan halaman akan menyambungnya halaman demi halaman menjadi satu file keluaran yang Anda unduh. Penggabungan mengikuti urutan file yang muncul di panel, jadi seret baris ke urutan yang Anda inginkan sebelum menggabungkan - misalnya sampul dulu, isi berikutnya, lampiran terakhir. Tidak ada langkah pendaftaran dan tidak ada tanda air pada hasilnya. Opsi di browser tidak memiliki batas penggabungan selain memori perangkat Anda, dan itulah arti praktis dari "tanpa batas": Anda tidak dibatasi kuota harian, Anda hanya dibatasi oleh apa yang dapat ditampung mesin Anda sendiri saat membangun file.

Gabung di browser versus jalur unggah - mana yang harus dipakai?

Halaman ini menawarkan dua cara untuk menyatukan file, dan keduanya cocok untuk situasi yang berbeda. Gabung di browser berjalan sepenuhnya di perangkat Anda melalui pustaka PDF yang dimuat ke dalam tab; tidak ada yang meninggalkan komputer Anda, sehingga ini pilihan tepat untuk kontrak, rekam medis, atau apa pun yang lebih baik tidak Anda kirim ke server. Jalur unggah mengirim file melalui HTTPS, membangun PDF gabungan di server, dan menghapus unggahan sumber setelah masa simpan singkat; ini berguna saat Anda memakai perangkat dengan memori rendah dan file gabungannya besar. Keduanya menghasilkan keluaran yang sama - satu PDF dalam urutan yang Anda tetapkan - jadi keputusannya soal sensitivitas dokumen dan mesin yang Anda pakai, bukan soal hasilnya.

Apa yang tetap utuh saat Anda menggabungkan PDF?

Penggabungan menyalin halaman tiap file sumber alih-alih merendernya ulang, sehingga teksnya tetap dapat dipilih dan dicari, bukan berubah menjadi gambar datar. Font tertanam, kolom formulir, dan bookmark ikut terbawa ke file gabungan ketika file asli menyimpannya sebagai struktur yang valid - perlu diperiksa di pembaca PDF Anda setelah mengunduh jika sumber Anda memiliki formulir interaktif. Karena halaman disalin alih-alih dikompresi ulang, file gabungan berukuran kira-kira sebesar gabungan bagian-bagiannya; menggabungkan tidak memperkecil totalnya. Jika hasilnya perlu lebih kecil, kompres sebagai langkah terpisah setelah menggabungkan, bukan mengharapkan penyambungan melakukan kedua tugas sekaligus.

Apa yang tidak dilakukan alat ini?

Alat gabung menyatukan PDF yang sudah bisa Anda buka - alat ini tidak membuka file terenkripsi. Jika salah satu masukan Anda dilindungi kata sandi, hapus kata sandinya dulu dengan Remove PDF Password (Anda perlu kata sandinya untuk melakukannya), lalu gabungkan salinan yang sudah terbuka. Ini juga bukan pengonversi format: beri ia PDF, bukan file Word atau gambar, dan ubah file tersebut menjadi PDF dulu jika Anda membutuhkannya dalam dokumen gabungan. Terakhir, alat ini menyatukan seluruh file dalam urutan yang Anda susun; ia tidak mengatur ulang atau menghapus halaman individual di dalam satu PDF sumber, jadi pangkas halaman yang tidak diinginkan sebelum Anda menambahkan file ke penggabungan.
