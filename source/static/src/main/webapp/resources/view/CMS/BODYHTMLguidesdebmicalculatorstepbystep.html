BMI-Rechner Schritt fuer Schritt

Der BMI Calculator laeuft in drei Schritten ab: Werkzeug oeffnen, Zahlen eingeben und Ergebnis ablesen. Dieser Leitfaden zeigt, was jeder Schritt erfordert.

Last reviewed: 2026-07-12

freetoolonline.com Editorial Team

Kurze Antwort. Oeffnen Sie den BMI Calculator, waehlen Sie metrisch oder imperial, geben Sie Gewicht und Groesse ein, und lesen Sie das Ergebnis - es aktualisiert sich, waehrend Sie tippen.

Schritt 1 - Werkzeug oeffnen

Oeffnen Sie den BMI Calculator. Vor dem Start ist weder eine Installation noch eine Anmeldung erforderlich.

Schritt 2 - Zahlen eingeben

Waehlen Sie metrisch (kg/cm) oder imperial (lb/in) und geben Sie Ihr Gewicht und Ihre Groesse ein.

Schritt 3 - Ergebnis erhalten

Der BMI-Wert und die Kategorie aktualisieren sich, waehrend Sie tippen - kein Button zum Klicken. Die Berechnung laeuft lokal im Browser; es wird nichts hochgeladen.

Haeufige Fragen

Muss ich ein Konto erstellen, um es zu nutzen?

Nein. Das Werkzeug laeuft direkt im Browser, ohne Installation oder Anmeldung.

Werden meine Daten irgendwohin gesendet?

Nein. Die eingegebenen Zahlen bleiben auf Ihrem Geraet und werden nie an einen Server oder Analysedienst gesendet.

Ist das Ergebnis eine medizinische Beratung?

Nein. Das Ergebnis ist nur eine allgemeine Screening-Schaetzung; sie beruecksichtigt keine Muskelmasse, Alter oder Koerperzusammensetzung.
