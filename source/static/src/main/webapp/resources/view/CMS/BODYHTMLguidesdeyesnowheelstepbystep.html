Ja-oder-Nein-Rad Online: Schritt-fuer-Schritt-Anleitung

Das Tool Yes or No Wheel startet bereits in zwei Segmente geteilt, Ja und Nein, und entscheidet mit einer einzigen Drehung. Diese Anleitung zeigt, was standardmaessig geladen wird, wie Sie das Rad in eine Auswahl fuer jede andere Entscheidung mit 2 oder mehr Optionen verwandeln, und wie die Drehung fair bleibt.

Last reviewed: 2026-07-02

Antwort in 30 Sekunden. Oeffnen Sie das Tool Yes or No Wheel und klicken Sie auf Spin - das Rad ist bereits in Ja und Nein geteilt, sodass Sie mit einem Klick eine Antwort erhalten. Um zwischen anderen Optionen zu entscheiden, ersetzen Sie die beiden Zeilen im Textfeld durch Ihre eigene Liste (eine pro Zeile), bevor Sie drehen.

Was standardmaessig geladen wird

Das Textfeld ueber dem Rad startet mit genau zwei Zeilen, "Yes" und "No", sodass das Rad bereits in zwei gleich grosse Segmente geteilt ist, sobald die Seite geoeffnet wird. Klicken Sie sofort auf Spin fuer eine einfache Ja-oder-Nein-Antwort - Sie muessen vorher nichts eingeben, es sei denn, Sie moechten eine andere Frage.

Es in ein Rad fuer alles andere verwandeln

Loeschen Sie das Textfeld und geben Sie Ihre eigene Liste ein, einen Eintrag pro Zeile - zwei Optionen, fuenf Optionen, oder so viele Sie brauchen. Das Rad teilt sich immer in ein gleich grosses Segment pro Zeile, sodass eine laengere Liste nur duennere Segmente bedeutet, keine schlechteren Chancen fuer einen Eintrag. Klicken Sie erneut auf Spin, sobald die Liste wie gewuenscht ist.

Wie die Drehung fair bleibt

Das gewinnende Segment wird mit dem crypto.getRandomValues Ihres Browsers ausgewaehlt, derselben Fairness-Quelle, die auch die anderen Zufallsauswahl-Tools der Website verwenden, sodass jedes Segment die gleiche Chance hat, unabhaengig davon, wo es auf dem Rad sitzt oder in welcher Zeile es eingegeben wurde. Die Drehung laeuft vollstaendig auf Ihrem Geraet - waehrend der Entscheidung findet keine Serverkommunikation statt.

Yes or no wheel im Vergleich zu den anderen Zufallsauswahl-Tools

Dieses Tool ist fuer eine schnelle Zwei-Optionen-Entscheidung gebaut, ist aber nicht auf Ja/Nein beschraenkt - es ist derselbe Rad-Mechanismus wie der wheel spinner, nur mit einer kuerzeren Startliste. Nutzen Sie den coin flip, wenn Sie ein Kopf/Zahl-Ergebnis ohne Drehanimation moechten, oder den dice roller, wenn Sie ein numerisches Ergebnis von eins bis sechs brauchen.

Haeufige Anwendungen

Menschen nutzen das Ja-oder-Nein-Rad, um eine schnelle Entscheidung ohne langes Nachdenken zu treffen, ein Unentschieden zwischen zwei Optionen aufzuloesen, zu entscheiden, wer anfaengt, oder - nach dem Bearbeiten der Liste - eine kleine Verlosung durchzufuehren oder einen zufaelligen Gewinner aus einer kurzen Namensliste zu waehlen.

Was dieses Tool nicht macht

Es merkt sich Ihre letzte Liste nicht zwischen Besuchen - die Seite kehrt beim Neuladen immer zum Ja/Nein-Standard zurueck. Es bevorzugt keinen Eintrag; jeder Eintrag auf dem Rad hat genau die gleiche Chance, und nichts, was Sie eingeben, wird hochgeladen oder auf einem Server gespeichert.

Datenschutz

Die Drehung laeuft vollstaendig in Ihrem Browser-Tab. Es wird nichts irgendwohin hochgeladen, kein Konto ist erforderlich, und die Seite fuehrt waehrend der Drehung keine Serverkommunikation durch.

Haeufig gestellte Fragen

Startet das Rad immer bei Ja und Nein?

Ja - das Textfeld ist bei jedem Laden der Seite mit "Yes" und "No" vorbelegt. Ersetzen Sie diese beiden Zeilen durch Ihre eigene Liste, wenn Sie zwischen anderen Optionen entscheiden moechten.

Ist ein 2er-Rad wirklich 50/50?

Ja. Jeder Eintrag erhaelt ein gleich grosses Segment, und das gewinnende Segment wird mit crypto.getRandomValues ausgewaehlt, sodass ein Rad mit zwei Eintraegen ueber viele Drehungen etwa zur Haelfte auf jeder Seite landet.

Kann ich mehr als zwei Optionen hinzufuegen?

Ja - geben Sie beliebig viele Eintraege ein, einen pro Zeile, bevor Sie auf Spin klicken. Das Rad teilt sich in ein gleich grosses Segment pro Eintrag, egal wie viele Sie hinzufuegen.

Wird meine Liste fuer das naechste Mal gespeichert?

Nein. Nur die aktuelle Liste wird angezeigt - ein Neuladen oder Verlassen der Seite setzt sie auf den Ja/Nein-Standard zurueck, und nichts wird hochgeladen oder auf einem Server gespeichert.
