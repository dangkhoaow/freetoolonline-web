So erkunden Sie den Venus Runaway Greenhouse Explorer 3D

Zuletzt geprüft: 2026-07-23

Geprüft von FreetoolOnline-Redaktionsteam

Öffnen Sie den Venus Runaway Greenhouse Explorer 3D, ziehen Sie den CO2-Regler oder drücken Sie Reset, und sehen Sie, warum eine dicke CO2-Decke die Venus heißer hält als Merkur, obwohl die Venus fast doppelt so weit von der Sonne entfernt kreist.

  1. Öffnen Sie /space-3d/venus-runaway-greenhouse.html und warten Sie, bis die Venus gerendert wird.
  2. Ziehen Sie den CO2-Regler von 96% weg und beobachten Sie, wie die Dunstschicht um die Venus dünner wird und die Farbe wechselt.
  3. Drücken Sie "Reset", um den Regler auf die real gemessene Zusammensetzung der Venus (96,5% CO2) zurückzusetzen, und lesen Sie die reale Oberflächentemperatur von 464 C / 737 K im Datenfeld.
  4. Aktivieren Sie den Merkur-Vergleich, um eine maßstabsgetreue Merkurkugel neben der Venus zu sehen, und lesen Sie, warum Merkur - fast doppelt so nah an der Sonne - im Schnitt trotzdem kälter ist.
  5. Aktivieren Sie die Druckvergleichsbalken, um den Oberflächendruck der Venus (92 bar) neben dem der Erde (1 bar) und dem von Merkur (nahezu null) zu sehen.
  6. Lesen Sie das Datenfeld für die vollständige Referenztabelle aller auf dieser Seite verwendeten veröffentlichten Werte.
