Cara menggunakan JWST L2 Halo Orbit 3D Explorer

  1. Buka JWST L2 Halo Orbit 3D Explorer dan tunggu kanvas selesai digambar.
  2. Seret untuk memutar tampilan dan gulir untuk memperbesar lingkaran halo di sekitar penanda L2.
  3. Geser penggeser hari halo dari 0 ke 168 untuk memindahkan Webb ke titik mana pun pada jalurnya.
  4. Tekan Putar orbit untuk menganimasikan satu putaran halo penuh secara terus-menerus.
  5. Aktifkan atau matikan penanda manuver untuk melihat kira-kira di mana manuver penjagaan orbit terjadi setiap 21 hari.
