Passport Photo Maker Schritt Fuer Schritt

Last reviewed: 2026-07-16

Passport Photo Maker laeuft in drei Schritten ab: Tool oeffnen, Eingabe bereitstellen, dann Ergebnis herunterladen. Diese Seite erklaert, was jeder Schritt erwartet.


Schritt 1 - Tool oeffnen

Gehen Sie zu Passport Photo Maker. Vor dem Start muss nichts installiert oder angemeldet werden.


Schritt 2 - Eingabe bereitstellen

Die Ausgabe wird auf einem einfarbig weissen Hintergrund in der exakten Pixelgroesse des Presets zusammengesetzt, berechnet mit 300 DPI fuer den Druck; aktivieren Sie "Also generate a 4x6in print sheet", um zusaetzlich einen Druckbogen mit so vielen Kopien herunterzuladen, wie auf eine Seite passen.


Schritt 3 - Ergebnis abholen

Alles laeuft lokal ueber die Canvas API - das Foto wird nie auf einen Server hochgeladen.

Dieses Tool schneidet nur zu und passt die Groesse auf einem weissen Hintergrund an - es entfernt oder ersetzt keinen vorhandenen Hintergrund, erkennt oder zentriert das Gesicht nicht automatisch und prueft nicht die Einhaltung der vollstaendigen Anforderungen einer bestimmten Passbehoerde (Beleuchtung, Ausdruck, Kopfbedeckung); pruefen Sie die offiziellen Regeln Ihres Ziellandes vor dem Einreichen.
