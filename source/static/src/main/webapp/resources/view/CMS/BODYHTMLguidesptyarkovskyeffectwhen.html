Quando Usar o Explorador 3D do Efeito Yarkovsky - Melhores Sessoes e Limites

Ultima revisao: 2026-07-23

Abra o Explorador 3D do Efeito Yarkovsky quando quiser entender por que a orbita medida do asteroide Bennu esta lentamente encolhendo, ou para ensinar lado a lado a regra geral de que a rotacao prograda faz crescer e a retrograda faz encolher - nao para executar uma propagacao orbital em escala real ou acompanhar um rastreamento em tempo real.


Uma licao prograda versus retrograda

Uma licao comeca em Retrograda (Bennu, medida) e avanca o controle de anos decorridos até 100 para ver o anel orbital encolher e a leitura de deriva acumulada tornar-se mais negativa, depois muda para Prograda (espelho didatico) e repete o avanco para ver a mesma magnitude empurrar a orbita para fora.


Sessoes para as quais este explorador nao serve

Sessoes para as quais esta pagina nao serve incluem executar uma manobra impulsiva de assistencia gravitacional - use o Explorador 3D de Assistencia Gravitacional para isso - ou verificar um limite de rompimento por mare - use o Explorador 3D do Limite de Roche. Percorra os controles no guia passo a passo.
