When to Use the Chariklo Rings 3D Explorer - Best Sessions and Limits

Last reviewed: 2026-07-24

Open Chariklo Rings 3D Explorer when you want to see the smallest known ring system and grasp just how small it is next to Saturn's - not to check a live orbital-tracking feed or study Chariklo's precise 3D shape, which is still only approximately known.


A discovery-story session

A session starts by orbiting the two real rings around Chariklo to read the 2013 stellar-occultation discovery figures in the facts panel, then toggles the scale comparison to watch Chariklo shrink to a speck next to Saturn's much larger ring span - the same one-consistent-scale trick makes the size gap concrete instead of abstract.


Sessions this explorer does not fit

Sessions this page does not fit are checking a live orbital-tracking feed for Chariklo - use the Saturn Rings 3D Explorer for the giant-planet case this page compares against, or read about the Centaur's exact shape, which is still only approximately modeled from limited occultation data.
