QR Code Scanner

Last reviewed: 2026-07-03

Find out what a QR code says from a screenshot or saved photo - choose the image and the decoded text appears in seconds, no app install.

When the code contains a link, the tool shows the exact URL as a clickable link BEFORE you visit it - an easy way to check a suspicious QR code for phishing.

Copy the decoded value with one click - handy for wifi passwords, serial numbers, coupon strings or long payment references.
