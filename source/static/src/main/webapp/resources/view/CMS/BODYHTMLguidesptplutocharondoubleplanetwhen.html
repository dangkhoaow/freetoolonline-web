Quando usar o Pluto-Charon Double Planet 3D Explorer

Ultima revisao: 2026-07-21

Abra o Pluto-Charon Double Planet 3D Explorer quando precisar de uma licao de trava tidal mutua com baricentro fora do primario - sem um integrador gravitacional.


Licao de baricentro externo

Mostre que ~2,110 km a partir do centro de Plutao ultrapassa o raio medio de 1188.3 km, entao o marcador verde fica no espaco entre os mundos.


Cifras New Horizons / JPL

Aqui cabem 1188.3 km / 606 km, 1.303e22 kg / 1.586e21 kg (~8.2:1), ~19,591 km, 6.387 days e trava tidal mutua.


Quando este explorador nao serve

Nao use para o baricentro interno Terra-Lua ou para uma linha de tamanhos de planetas anoes. Veja o Earth-Moon Barycenter 3D Explorer ou o Dwarf Planet Comparison 3D Explorer. Controles no guia passo a passo.
