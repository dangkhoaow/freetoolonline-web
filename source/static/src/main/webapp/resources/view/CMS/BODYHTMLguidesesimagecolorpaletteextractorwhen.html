Cuando extraer una paleta de colores de una foto

Use este extractor cuando tiene un JPEG, PNG o WebP local y necesita los HEX dominantes para un design system, mood board o tema CSS.

Elija el archivo, defina cuantos colores quiere y pulse Extract Palette. El muestreo permanece en el navegador - la imagen nunca se sube.

Si solo necesita elegir un tono a mano, use Color Picker. Para legibilidad texto-sobre-fondo, use WCAG Contrast Checker.
