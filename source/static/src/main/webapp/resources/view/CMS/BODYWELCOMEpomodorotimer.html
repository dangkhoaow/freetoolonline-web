Pomodoro Timer Online

Last reviewed: 2026-07-11

Run a Pomodoro work/break timer to structure a focus session - not a plain single countdown, since the timer keeps alternating between a Work interval and a Break interval on its own once you start it.

Set the Work length (1-120 minutes, default 25) and the Break length (1-60 minutes, default 5) before you start, so the cycle matches your own routine instead of a fixed 25/5 split. A phase label always shows which interval is active ("Work" or "Break") so you never lose track of where you are.

Click Start to begin the Work interval; the timer switches to Break automatically when it reaches zero, then back to Work when the Break ends, and keeps repeating for as long as you leave it running. Pause holds the countdown in place; Reset returns to the start of the current Work interval using whatever Work/Break minutes are currently set.

Everything runs locally in this browser tab - no upload and no account. Closing or reloading the tab stops the timer; it does not run or notify in the background, and it does not count how many Work/Break cycles you have completed this session.

Want a single one-off countdown instead of a repeating cycle? Use the plain countdown timer. Want the classic 25-minute/5-minute split with no adjustment needed? Leave the defaults as they are and just click Start.
