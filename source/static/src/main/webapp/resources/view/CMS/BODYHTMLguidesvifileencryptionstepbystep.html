Cong Cu Ma Hoa File Tung Buoc

Last reviewed: 2026-07-13
freetoolonline.com Editorial Team

Cong Cu Ma Hoa File chay theo ba buoc: mo cong cu, nhap du lieu, roi tai xuong ket qua. Trang nay noi chi tiet dieu moi buoc can.


Buoc 1 - mo cong cu

Vao Cong Cu Ma Hoa File. Khong can cai dat hay dang nhap truoc khi bat dau.


Buoc 2 - nhap du lieu

File duoc xao tron bang AES-256 dung mot khoa duoc suy ra tu mat khau cua ban (PBKDF2, 250.000 vong lap) cong voi salt va IV ngau nhien moi lan, nen ma hoa cung mot file hai lan voi cung mat khau se tao ra hai file .enc khac nhau - dieu nay la binh thuong, khong phai loi.


Buoc 3 - lay ket qua

Mat khau sai, hoac file .enc bi sua doi hoac hong, se bi tu choi voi mot thong bao loi ro rang thay vi tao ra mot file hong hoac day du lieu vo nghia.

Khong co cach khoi phuc mat khau: cong cu nay khong luu mat khau cua ban o bat ky dau, nen mat mat khau khien file da ma hoa vinh vien khong the khoi phuc - hay ghi lai o mot noi an toan truoc khi dong tab nay.
