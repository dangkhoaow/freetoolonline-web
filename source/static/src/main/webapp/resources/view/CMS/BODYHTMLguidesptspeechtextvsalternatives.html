Perguntas frequentes

Por que e limitado ao Chrome e Safari?

A API Web Speech Recognition nao esta disponivel no Firefox. O Chrome e Edge (Chromium) e o Safari suportam a API nativamente.

Posso usar para transcrever arquivos de audio pre-gravados?

Nao. Esta ferramenta captura audio ao vivo do microfone. Para arquivos pre-gravados, um servico dedicado de transcricao seria mais adequado.
