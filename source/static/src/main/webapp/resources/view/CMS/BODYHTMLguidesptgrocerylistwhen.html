Grocery List Quando Usar

Last reviewed: 2026-07-09

O Grocery List vale a pena abrir quando voce precisa de uma execucao rapida e unica sem instalar nada. Esta pagina explica quando isso se aplica e quando nao.


Quando faz sentido

O Grocery List funciona bem quando voce precisa de uma lista de compras rapida e unica, sem instalar um aplicativo ou criar uma conta.


O que esperar

O Grocery List e uma lista de verificacao minima, baseada no navegador: cada item recebe uma caixa de selecao para marcar como concluido e um botao para remove-lo.

A lista e salva neste navegador via local storage, entao ela sobrevive a um recarregamento da pagina; limpar os dados do navegador deste site a remove.
