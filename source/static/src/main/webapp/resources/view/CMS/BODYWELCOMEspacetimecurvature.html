This browser explorer shows spacetime curvature as a teaching grid - mass bends nearby geodesics, with literacy r_s = 2GM/c^2 (Sun ~2.95 km, Earth ~8.87 mm).

Gravity Well 3D owns escape-velocity embedding wells. Light Cone 3D owns flat Minkowski causal regions. This page owns curved-grid geodesic intuition.
