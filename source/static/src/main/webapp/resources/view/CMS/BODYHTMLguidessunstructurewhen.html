When to use Sun Structure 3D Explorer

Use the Sun Structure 3D Explorer when you need a quick browser cutaway of core, radiative, convective, photosphere, and corona layers - and skip it when you need a hydrodynamics model.

  • Classroom demos linking core fusion, radiative transport, and convective boiling
  • Showing photosphere ~5,772 K versus corona ~1-3 million K and the temperature puzzle
  • One-screen refresher before HR diagram or star-lifecycle siblings in Space 3D

This page does not run solar hydrodynamics, does not fetch live SDO data, and uses schematic shells for teaching layers only.
