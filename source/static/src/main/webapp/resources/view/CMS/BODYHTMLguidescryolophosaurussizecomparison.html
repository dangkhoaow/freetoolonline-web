What the Cryolophosaurus size numbers are not

What these size numbers are not is a claim of one precise length or weight: the Cryolophosaurus size comparison discloses the about 6-7 m / 350-465 kg ranges (with larger estimates noted) and uses 6.5 m only for the Size vs human scale; it does not claim the crest looked exactly like Elvis hair, does not invent soft-tissue color accuracy, and this is not a win/lose game.
