Pdf Compressor Online es una de tres formas comunes de hacer esto: una herramienta de navegador como esta, una aplicación de escritorio, o un servicio en línea basado en subida de archivos. Cada una compensa de forma distinta a dónde van tus datos y qué necesitas instalar.

Esta herramienta aplica un único guardado estructural sin control de nivel de compresión; una aplicación de escritorio o un servicio con subida puede ofrecer una recompresión de imagen más fuerte y con pérdida, a costa de instalar software o enviar el archivo a un servidor.

El archivo original no se conserva más allá de la sesión de la página, y el archivo nunca se sube para esta herramienta.
