When to Play Mono Grid Duel

Mono Grid Duel fits quick brain breaks where you want a classic three-in-a-row duel vs CPU or a friend on one keyboard - about ~64 KB after Play.

Last reviewed: 2026-07-18

Short grid rounds

Each board resolves in a few moves. Restart with R and chase win counts on Hard minimax without leaving the tab.

Two-player on one device

Hotseat 2P mode alternates first player each game. Good for a quick pass-and-play duel at a desk.

When to pick another game

Want paddle reflex? Use Mono Paddle Duel. Want a minefield grid? Use Mono Minefield Grid.
