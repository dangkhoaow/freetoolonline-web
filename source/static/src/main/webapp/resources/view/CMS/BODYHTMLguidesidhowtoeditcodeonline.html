Cara mengedit kode online - gratis, di browser Anda

Edit kode langsung di browser Anda di editor lengkap dan gratis - mesin di balik VS Code. Pilih dari 18 bahasa, ganti tema, buka berkas untuk mengeditnya, lalu simpan pekerjaan Anda ke berkas atau unduh. Semua berjalan di perangkat Anda; tidak ada yang diunggah dan tidak ada yang perlu diinstal.

Langkah

  1. Buka alat dan klik Launch editor; editor dimuat di browser Anda pada kali pertama.
  2. Pilih bahasa dari daftar agar mendapat penyorotan sintaks yang cocok dengan kode Anda.
  3. Pilih tema terang, gelap, atau kontras tinggi untuk layar Anda.
  4. Ketik atau tempel kode Anda, atau klik Open a file untuk memuat berkas dari perangkat Anda.
  5. Klik Save / Download untuk menulis pekerjaan Anda ke berkas (atau mengunduhnya).
  6. Opsional, beri nama konten dan klik Save snippet untuk menyimpannya di browser ini untuk nanti.

Bahasa dan tipe berkas

Editor menyorot 18 bahasa dan memilih ekstensi berkas yang sesuai saat menyimpan. Beberapa yang umum:

BahasaEkstensi berkas tersimpan
JavaScript.js
TypeScript.ts
Python.py
HTML.html
CSS.css
JSON.json

Menyimpan dan snippet

Di browser berbasis Chromium tombol Save menulis langsung ke berkas yang Anda pilih; di Firefox dan Safari ia mengunduh berkas. Snippet bernama disimpan di penyimpanan lokal browser ini, jadi tetap ada saat Anda kembali di perangkat dan browser yang sama - tidak tersinkron antar perangkat, dan menghapus data situs ini di browser akan menghapusnya.

Yang tidak dilakukan alat ini

Ini editor, bukan pelari. Ia menulis, menyorot, dan menyimpan kode, tetapi tidak menjalankan atau mengeksekusi kode Anda - jalankan itu di lingkungan Anda sendiri. Tidak ada yang Anda ketik diunggah; editor bekerja sepenuhnya di perangkat Anda.
