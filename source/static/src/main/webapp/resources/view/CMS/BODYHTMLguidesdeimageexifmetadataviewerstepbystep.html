Image EXIF Metadata Viewer Schritt Fuer Schritt

Last reviewed: 2026-07-21

Image EXIF Metadata Viewer laeuft in drei Schritten: oeffnen, lokales Bild waehlen, Tags lesen (optional strip).


Schritt 1 - Werkzeug oeffnen

Gehen Sie zu Image EXIF Metadata Viewer. Keine Installation und kein Login noetig.


Schritt 2 - Bild waehlen

Dateiauswahl fuer JPEG, PNG, WebP, HEIC oder TIFF auf diesem Geraet. Vorschau erscheint. Read Metadata bleibt deaktiviert, bis eine Datei gewaehlt ist.


Schritt 3 - Tags lesen (optional strip)

Read Metadata klicken. Tabellen Camera, Date / Time, GPS und Other fuellen die Seite. Bei GPS zeigt die rote Warnung Koordinaten und OpenStreetMap-Link. Download Without Metadata speichert JPEG mit Qualitaet 0.95 ohne EXIF. Clear setzt die Sitzung zurueck - nichts wurde hochgeladen.
