To-Do List

Last reviewed: 2026-06-15

Keep a todo list in the browser - add an item, check it off when it is done, and delete it when you no longer need it. Nothing is uploaded and there is no sign-in.


What you can do with it

  • Each item gets a checkbox to mark it done and a button to remove it.
  • A running count shows how many items you have and how many are still left to do.

When a quick checklist beats an app

Use it for a quick checklist when you do not want to install an app or sign in - a short errands list, packing items, or the steps for a small task you want to tick off as you go.


Where your list is saved

The list is saved in this browser via local storage, so it survives a page reload; clearing this site's browser data removes it. There is no account and nothing is sent to a server, so your items stay on this device.
