Cuando usar Orbital Velocity 3D Explorer

Usa el Orbital Velocity 3D Explorer cuando necesites una capa cuantitativa de velocidad a lo largo de una orbita excentrica - no para propagacion n-body.

  • Demos de clase: Vis-viva con periapsis mas rapido y apoapsis mas lento
  • Comparar Mercury ~47,36 km/s, Earth ~29,78 km/s, Neptune ~5,43 km/s
  • Repaso rapido despues de kepler-orbits o antes de escape-velocity

Esta pagina no ejecuta un integrador gravitacional y escala la orbita solo para lectura didactica.
