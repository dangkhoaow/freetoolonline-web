Milky Way Map 3D Explorer Langkah demi Langkah

Last reviewed: 2026-07-14

Milky Way Map 3D Explorer langsung berjalan begitu halaman termuat: seret untuk memutar cakram, gulir untuk zoom, dan tekan Play spin. Semua diproses di perangkat Anda sendiri - tanpa instal, tanpa akun.


Langkah 1 - temukan Matahari

Klik Sun agar titik kuning dan angka ~26.000 ly muncul di panel di samping cakram ~100.000 ly.


Langkah 2 - lengan bernama

Lompat ke Sagittarius, Orion, dan Perseus untuk membaca setiap label pengajaran saat cakram berputar perlahan.


Langkah 3 - pusat galaksi

Tekan Center, lalu layar penuh agar kontrol tetap terlihat saat Anda mengorbit inti yang hangat.
