Document Scanner (to PDF)

Last reviewed: 2026-07-13

Turn one or more photos of a paper document into a single PDF, right in this browser - no upload, no app to install.

Each photo becomes one page in the output PDF, in the order you selected them - a multi-page document in one file.

Everything runs locally in this browser tab; photos are never uploaded to a server.

Each page is saved as a JPEG image at high quality inside the PDF, not as vector text - a stack of several full-resolution phone photos can add up to a multi-megabyte file, so check the size before emailing a long scan.
