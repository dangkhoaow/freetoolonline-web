Como editar codigo online - gratis, en tu navegador

Edita codigo en tu navegador en un editor completo y gratuito - el motor detras de VS Code. Elige entre 18 lenguajes, cambia el tema, abre un archivo para editarlo y guarda tu trabajo en un archivo o descargalo. Todo funciona en tu dispositivo; no se sube nada y no hay nada que instalar.

Pasos

  1. Abre la herramienta y haz clic en Launch editor; el editor se carga en tu navegador la primera vez.
  2. Elige un lenguaje en la lista para tener resaltado de sintaxis acorde a tu codigo.
  3. Elige un tema claro, oscuro o de alto contraste para tu pantalla.
  4. Escribe o pega tu codigo, o haz clic en Open a file para cargar un archivo de tu dispositivo.
  5. Haz clic en Save / Download para escribir tu trabajo en un archivo (o descargarlo).
  6. Opcionalmente, nombra el contenido y haz clic en Save snippet para guardarlo en este navegador para mas tarde.

Lenguajes y tipos de archivo

El editor resalta 18 lenguajes y elige una extension de archivo adecuada al guardar. Algunas comunes:

LenguajeExtension de archivo guardada
JavaScript.js
TypeScript.ts
Python.py
HTML.html
CSS.css
JSON.json

Guardar y snippets

En navegadores basados en Chromium el boton Save escribe directo en un archivo que elijas; en Firefox y Safari descarga el archivo. Los snippets con nombre se guardan en el almacenamiento local de este navegador, asi que siguen ahi cuando vuelves en el mismo dispositivo y navegador - no se sincronizan entre dispositivos, y borrar los datos de este sitio en el navegador los elimina.

Lo que esta herramienta no hace

Es un editor, no un ejecutor. Escribe, resalta y guarda codigo, pero no ejecuta tu codigo - ejecutalo en tu propio entorno. Nada de lo que escribes se sube; el editor funciona totalmente en tu dispositivo.
