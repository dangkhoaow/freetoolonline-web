Password Strength Checker - Cuándo Usar

Last reviewed: 2026-07-23

Abre Password Strength Checker cuando necesites una comprobación rápida y puntual, sin instalar nada.


Cuándo encaja

La puntuación viene de zxcvbn, el mismo modelo de coincidencia de patrones y ataque de diccionario que usa Dropbox, no una regla simple de longitud o tipo de carácter - detecta contraseñas comunes y patrones predecibles que una regla como "P@ssw0rd1" no detecta.


Qué esperar

Todo se ejecuta en el navegador - sin subir nada, sin cuenta, y la contraseña nunca se envía a ningún lado, se registra ni se comprueba contra una base de datos de filtraciones.

Esta herramienta no guarda ni recupera contraseñas; solo puntúa lo que esté escrito en el campo en ese momento.
