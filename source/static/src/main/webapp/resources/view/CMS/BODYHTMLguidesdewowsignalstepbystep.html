How to explore Wow! Signal 3D Explorer

Open the explorer and watch the real recorded code build up character by character.

  1. Open /space-3d/wow-signal.html and wait for the dish and source to appear.
  2. Press Play and watch the source drift through the telescope's fixed beam.
  3. Watch the intensity rise from 6 up to the real peak of 30, then fall to 5.
  4. Read the live "6EQUJ5" code build up one real character at a time.
  5. Drag to orbit and use the speed slider to slow down or speed up the 72-second window.
