Password Strength Checker - Im Vergleich Zu Alternativen

Last reviewed: 2026-07-23

Der Password Strength Checker ist einer von drei gängigen Wegen dafür: ein browserbasiertes Werkzeug wie dieses, eine Desktop-Anwendung oder ein Upload-basierter Online-Dienst. Jede Variante hat andere Kompromisse bei Datenverbleib und Installationsaufwand.

Die Bewertung stammt von zxcvbn, demselben Muster- und Wörterbuchangriff-Modell, das auch Dropbox nutzt, und nicht aus einer einfachen Längen- oder Zeichenklassenregel - es erkennt gängige Passwörter und vorhersehbare Muster, die eine Regel wie "P@ssw0rd1" übersieht.

Alles läuft direkt im Browser - ohne Upload, ohne Konto, und das Passwort wird nirgendwohin gesendet, protokolliert oder gegen eine Datenleck-Datenbank geprüft.
