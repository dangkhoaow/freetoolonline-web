Favicon Generator - Paso a Paso

Last reviewed: 2026-07-21

Abre Favicon Generator, elige una imagen local y descarga favicon.ico o PNG individuales.


Pasos

  1. Haz clic en Choose File y elige PNG, JPEG, WebP, GIF o BMP.
  2. Espera a que se rellenen las vistas previas 16, 32, 48, 180, 192 y 512.
  3. Descarga favicon.ico para navegadores, o PNG 32x32 / 180x180 para usos comunes.
  4. Usa los enlaces PNG por tamaño cuando solo necesites una dimensión.

Privacidad

El redimensionado se ejecuta en el navegador. Tu imagen no se sube.
