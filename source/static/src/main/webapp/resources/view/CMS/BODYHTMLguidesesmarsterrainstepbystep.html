Como Usar Mars Terrain Explorer

Last reviewed: 2026-07-15

Carga la pagina, espera a que los datos reales de elevacion se descarguen, luego estira el control de relieve y haz clic en el globo para una lectura real de elevacion.

  1. Abre Mars Terrain Explorer y espera a que la esfera provisional se convierta en el terreno real de Marte.
  2. Arrastra para orbitar el globo; mantén el panel de datos visible.
  3. Presiona Relief + varias veces y observa como Olympus Mons sube y Hellas Planitia baja.
  4. Presiona Real scale para ver que tan plano es Marte realmente en proporcion verdadera.
  5. Haz clic en cualquier punto del globo para leer la elevacion real en ese punto.
