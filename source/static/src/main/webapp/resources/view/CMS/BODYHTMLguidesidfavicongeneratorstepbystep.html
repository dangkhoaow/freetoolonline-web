Favicon Generator - Langkah demi Langkah

Last reviewed: 2026-07-21

Buka Favicon Generator, pilih gambar lokal, lalu unduh favicon.ico atau PNG per ukuran.


Langkah

  1. Klik Choose File dan pilih PNG, JPEG, WebP, GIF, atau BMP.
  2. Tunggu pratinjau 16, 32, 48, 180, 192, dan 512 terisi.
  3. Unduh favicon.ico untuk peramban, atau PNG 32x32 / 180x180 untuk penggunaan umum.
  4. Pakai tautan PNG per ukuran bila hanya butuh satu dimensi.

Privasi

Pengubahan ukuran berjalan di peramban. Gambar Anda tidak diunggah.
