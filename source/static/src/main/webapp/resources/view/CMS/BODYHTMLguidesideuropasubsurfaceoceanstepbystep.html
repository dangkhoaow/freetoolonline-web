Cara Menggunakan Europa Subsurface Ocean 3D Explorer

Last reviewed: 2026-07-23

Geser Peel shell dari 0 sampai 100 persen dan baca angka nyata yang dipublikasikan di panel fakta saat lapisan es, lautan, mantel, dan inti terbuka secara berurutan.

  1. Buka Europa Subsurface Ocean 3D Explorer dan tunggu bulan es yang masih utuh tergambar.
  2. Seret untuk memutar tampilan; jaga panel fakta tetap terlihat di sisi kanan.
  3. Geser Peel shell melewati 25 persen dan lihat potongan lapisan es terbuka, memperlihatkan seluruh lapisan lautan di bawahnya.
  4. Lanjutkan menggeser melewati 50 dan 75 persen untuk memperlihatkan mantel berbatu, lalu inti logam, satu lapisan pada satu waktu.
  5. Centang kotak Clipper flyby tracks untuk melihat enam lintasan ilustratif di lapisan es.
  6. Atur ulang Peel shell ke 0 dan gunakan empat tombol cepat - Ice shell, Ocean, Mantle, Core - untuk langsung menuju lapisan mana pun.
