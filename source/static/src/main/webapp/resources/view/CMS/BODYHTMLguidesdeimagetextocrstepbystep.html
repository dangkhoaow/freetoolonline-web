Image To Text (ocr) Schritt Fuer Schritt

Last reviewed: 2026-07-16

Image to Text (OCR) laeuft in drei Schritten: Tool oeffnen, ein Bild auswaehlen, dann den erkannten Text kopieren oder herunterladen. Diese Seite geht durch, was jeder Schritt erwartet.


Schritt 1 - Tool oeffnen

Gehen Sie zu Image to Text (OCR). Vor dem Start muss nichts installiert oder eingeloggt werden.


Schritt 2 - Bild auswaehlen und Extract text klicken

Waehlen Sie eine Bilddatei ueber die Dateiauswahl - sie oeffnet sich im Browser und wird nie an einen Server hochgeladen - und klicken Sie dann Extract text. Der erste Durchlauf auf diesem Geraet laedt die OCR-Engine im Hintergrund herunter; spaetere Durchlaeufe auf demselben Geraet starten schneller.


Schritt 3 - erkannten Text kopieren oder herunterladen

Der erkannte Text erscheint in einem nur lesbaren Feld, sobald der Scan fertig ist. Kopieren Sie ihn mit einem Klick oder laden Sie ihn als .txt-Datei herunter. OCR ist nicht perfekt, pruefen Sie also Namen, Zahlen und Zeichensetzung, bevor Sie sich auf das Ergebnis verlassen.
