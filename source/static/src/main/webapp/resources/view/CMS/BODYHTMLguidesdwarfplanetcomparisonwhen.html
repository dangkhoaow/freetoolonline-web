When to Use Dwarf Planet Size Comparison 3D Explorer

Last reviewed: 2026-07-19

Open the Dwarf Planet Size Comparison 3D Explorer when you need to compare Ceres, Pluto, Haumea, Makemake, and Eris side by side - five real diameter, distance, and period figures in one WebGL scene - without hunting through five separate reference pages.


A five-minute classroom demo

A five-minute classroom demo starts when you open the explorer, let the class see all five spheres in the lineup, then click Ceres and Eris in turn so a roughly 940 km world and a 2,326 km world sit on the same screen at the same time.


Settling a "how big is Pluto really" argument

Settling a "how big is Pluto really" argument is exactly the question this page answers: Pluto and Eris are close in size to each other, both far larger than Ceres, and both a fraction of any of the eight IAU planets.


Where this page does not fit

Where this page does not fit: it does not include the Sun or the eight IAU planets - for those see Planet Size Comparison 3D Explorer - and it does not model the Kuiper Belt as a structure of countless smaller objects, only these five named bodies; for the region itself see Kuiper Belt and Oort Cloud 3D Explorer.
