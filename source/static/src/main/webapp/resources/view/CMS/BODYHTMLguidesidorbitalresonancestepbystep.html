Orbital Resonance 3D Explorer Langkah demi Langkah

Last reviewed: 2026-07-14

Orbital Resonance 3D Explorer mulai begitu halaman termuat: seret untuk mengorbit, gulir untuk memperbesar, dan tekan Play orbits. Semua dirender di perangkat Anda sendiri - tanpa instalasi, tanpa akun.


Langkah 1 - mainkan iramanya

Tekan Play orbits agar Io, Europa, dan Ganymede menjaga rantai periode 1:2:4; perhatikan kilatan putih tanda kesejajaran.


Langkah 2 - fokus ke bulan

Tekan Io, Europa, dan Ganymede untuk membaca peran periode masing-masing di panel.


Langkah 3 - pasangan klasik kedua

Tekan Neptune:Pluto 3:2, lalu layar penuh agar kontrol tetap terlihat saat rantai Galileo terus berjalan.
