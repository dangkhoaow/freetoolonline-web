Pertanyaan umum

Mengapa alat ini tidak menyimpan audio sebagai MP3?

Browser tidak mengekspos audio yang disintesis sebagai file audio yang dapat direkam. Untuk mendapatkan MP3, diperlukan layanan API sisi server - yang akan membuat unggahan.

Mengapa memilih alat browser?

Tidak ada instalasi dan tidak ada teks yang dikirim ke server - ini adalah pilihan tercepat untuk penggunaan sesekali ketika privasi penting.
