Cara Main Retro FPS Online - Langkah demi Langkah

Halaman Retro FPS Online menjalankan Freedoom lengkap di WebAssembly. Pilih kampanye, tunggu unduhan sekali, main dan simpan di browser ini. Ekspor .ftolfps untuk memindah progres ke PC lain.

Last reviewed: 2026-07-18

Keyboard dulu (panah gerak, Ctrl tembak, Space buka pintu). Mouse setelah klik layar. Tidak ada unduhan sebelum klik kampanye.

Langkah 1 - lihat kartu

Kampanye 1 (36 level), Kampanye 2 (32 level), dan Import muncul segera. Nol jaringan sampai klik.

Langkah 2 - klik satu kampanye

Unduhan sekali masuk IndexedDB. Tidak ada fetch sebelum klik ini.

Langkah 3 - layar judul

Setelah unduh, menu game. Kunjungan berikutnya dari IndexedDB dan bisa offline.

Langkah 4 - simpan dan ekspor

Esc membuka Save/Load. Export membuat .ftolfps; Import di PC lain memvalidasi versi engine.

Langkah 5 - batas

Ganti kampanye memuat ulang halaman. Tanpa multiplayer atau akun. Ponsel hanya sentuh bukan target. Hapus data situs menghapus cache dan save.
