Cómo usar el Explorador 3D de la Órbita Halo L2 del JWST

  1. Abre el Explorador 3D de la Órbita Halo L2 del JWST y espera a que se pinte el lienzo.
  2. Arrastra para orbitar la vista y desplázate para acercar el lazo halo alrededor del marcador L2.
  3. Desliza el control de día halo de 0 a 168 para mover el Webb a cualquier punto del trayecto.
  4. Pulsa Reproducir órbita para animar un ciclo halo completo de forma continua.
  5. Activa o desactiva los marcadores de maniobra para ver aproximadamente dónde ocurren las maniobras cada 21 días.
