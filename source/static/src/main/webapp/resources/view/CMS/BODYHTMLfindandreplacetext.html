How to find and replace text

  1. Paste or type the text into the input box - nothing is uploaded, the text stays in this browser tab.
  2. Type the word or phrase to find, then type what to replace it with; check "Match case" if the search should be case-sensitive.
  3. Click Replace All; the updated text appears in the output box along with a count of how many replacements were made.
