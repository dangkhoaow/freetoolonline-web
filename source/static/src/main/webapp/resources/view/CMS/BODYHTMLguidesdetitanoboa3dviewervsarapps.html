Wann der Titanoboa-Browser-Viewer passt

Waehlen Sie den Titanoboa-Browser-Viewer, wenn Sie eine palaeozaene Riesenschlange mit Size vs human im ehrlichen 12.8-m-Mittel-Laengenverhaeltnis orbitieren und Fakten anklicken wollen - ohne etwas zu installieren. Nichts vom Besuch wird an einen Server gesendet.


Wann eine Telefon-AR-App oder ein Museum besser passt

Eine Telefon-AR-App gewinnt, wenn Sie die Silhouette auf dem Wohnzimmerboden brauchen; dieser Viewer behauptet nie Raumplatzierung. Ein Museum gewinnt fuer das Gehen neben echten Schlangen- oder Meeresreptil-Montagen. Ein Standbild gewinnt nur, wenn Sie einen einzelnen teilbaren Frame ohne Interaktivitaet brauchen.


Was dieser Titanoboa-Vergleich nicht ist

Dieser Titanoboa-Vergleich ist eine Trade-off-Tabelle fuer den gelieferten Viewer - kein Ranking kommerzieller AR-Marken, erfindet keine AR-Funktionen, die die Seite nicht hat, behauptet nicht, Titanoboa sei ein Dinosaurier, behauptet nicht, das optionale glTF sei eine wissenschaftlich exakte Rekonstruktion, und erfindet keinen Spielstand. Zum Steuerungs-Walkthrough siehe Titanoboa in 3D ansehen. Zu Laenge und Gewicht neben einer 1.8-m-Person siehe Titanoboa Groessenvergleich.
