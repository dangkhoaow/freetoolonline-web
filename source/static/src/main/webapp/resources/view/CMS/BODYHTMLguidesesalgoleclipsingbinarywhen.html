Cuándo usar el Algol Eclipsing Binary 3D Explorer - Mejores sesiones y límites

Última revisión: 2026-07-22

Abre el Algol Eclipsing Binary 3D Explorer cuando necesites una lección sobre binarias eclipsantes y curva de luz con cifras publicadas de magnitud, período, masa y radio - sin un laboratorio de fotometría.


Una lección sobre eclipses y curva de luz

La lección comienza cuando mueves el control de fase por 0% (eclipse primario profundo) y 50% (eclipse secundario leve), o pulsas Play orbit para ver ambos automáticamente mientras la curva de luz grafica la caída de brillo y el panel conserva las cifras publicadas.


Sesiones para las que este explorador no es adecuado

Este explorador no es adecuado para lecciones generales de dinámica orbital o baricentro sin eclipse. Usa el Binary Star System 3D Explorer para eso. Sigue los controles en la guía paso a paso.
