Quando Usar o Colisao Via Lactea - Andromeda 3D Explorer

Last reviewed: 2026-07-20

Abra o Colisao Via Lactea - Andromeda 3D Explorer quando precisar mostrar como duas galaxias reais realmente se fundem ao longo de bilhoes de anos - nao uma animacao generica de ensino, mas a linha do tempo publicada para o futuro da nossa propria galaxia.


Uma demonstracao de sala de aula de cinco minutos

Uma demonstracao de sala de aula de cinco minutos comeca quando voce abre o explorador, aperta Play e pausa bem quando as caudas de mare aparecem - depois explica que a mesma distorcao e visivel hoje em pares reais em fusao, como as Galaxias Antenas.


Entendendo como as fusoes de galaxias realmente acontecem

Entendendo como as fusoes de galaxias realmente acontecem e exatamente a pergunta que esta pagina responde: dois discos se aproximam, a gravidade arranca estrelas em caudas de mare durante aproximacoes proximas, e todo o sistema eventualmente se acomoda em uma unica galaxia fundida sem estrutura espiral restante.


Sessoes que este explorador nao encaixa

Sessoes que este explorador nao encaixa sao uma aula sobre a estrutura interna de uma unica galaxia - para isso, use o Spiral Galaxy 3D Explorer - ou um tour pelo layout da propria Via Lactea, para o qual use o Milky Way Map 3D Explorer.
