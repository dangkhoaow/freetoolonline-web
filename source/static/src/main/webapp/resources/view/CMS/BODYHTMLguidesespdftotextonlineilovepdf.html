Una cosa para verificar primero: esto extrae el texto que ya esta incrustado en el PDF, que la mayoria de los archivos exportados desde un procesador de texto contienen. Si tu PDF es un escaneo o una foto de una pagina - donde no puedes seleccionar el texto con el cursor - la extraccion no devuelve nada, porque no hay una capa de texto que leer. Esos archivos necesitan OCR (reconocimiento optico de caracteres) primero, un paso aparte que esta herramienta no realiza. Una prueba rapida: abre el PDF de origen e intenta resaltar una frase. Si el cursor selecciona palabras, la extraccion funcionara. El formato tambien se pierde - negrita, titulos y tablas salen como lineas planas - asi que obtienes texto para pegar y volver a formatear, no una copia con estilo.
