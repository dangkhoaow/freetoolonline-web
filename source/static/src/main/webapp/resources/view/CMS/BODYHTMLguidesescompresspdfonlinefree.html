Dos rutas gratuitas cubren la mayoria de los PDF demasiado grandes. Primero, aplanar: muchos PDF llevan campos de formulario, anotaciones y capas apiladas que inflan el archivo, y aplanar las fusiona con el contenido de la pagina - a menudo recortando el tamano sin cambio visible. Usa la herramienta flatten PDF para eso. Segundo, para escaneos y documentos llenos de fotos (la razon habitual de que un PDF crezca), extrae cada pagina como imagen, comprime esas imagenes y luego reconstruye el PDF. Esa reconstruccion convierte el texto seleccionable en una imagen plana, asi que conserva el original si todavia necesitas copiar texto. No sabes cual es tu caso? Inspecciona el archivo primero.
