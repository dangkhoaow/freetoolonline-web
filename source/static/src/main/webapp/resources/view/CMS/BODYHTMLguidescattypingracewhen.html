When to Play Cat Typing Race

Cat Typing Race fits short breaks where you want a keyboard skill duel against an AI cat - about ~28 KB after Play, no install.

Last reviewed: 2026-07-18

Typing warm-ups

Each round is a quick burst of random A-Z keys. Good for warming up finger placement before writing or coding.

When you like escalating AI

The cat predicts letters faster each level. Round bonuses reward finishing first with at least half the letters correct.

When to pick another game

Want mouse clicks instead of keys? Use Potion Brew Shop. Want pipe valves? Use Wash The Cat.
