How the Dimetrodon size-vs-human toggle stays honest

The Dimetrodon model is drawn to a fixed on-screen length so it fits the canvas; the Size vs human control then places a 1.8 m person at the true length ratio used by the viewer (3.2 m). Tap Size vs human under the canvas to show or hide the scale figure - the comparison is proportional, not a decorative sticker. Published length still spans about 1.7-4.6 m across the genus, so the toggle picks one clear scale (D. grandis about 3.2 m) inside that disclosed range.

What the Dimetrodon size numbers are not

What these size numbers are not is a claim of one precise length or weight for every Dimetrodon: the size comparison discloses the about 3.2 m / 250 kg D. grandis figures plus genus ranges and uses 3.2 m only for the Size vs human scale; it does not invent soft-tissue or sail-membrane color accuracy, and this is not a win/lose game.
