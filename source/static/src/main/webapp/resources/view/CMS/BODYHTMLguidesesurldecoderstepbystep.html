Url Decoder Paso A Paso

Last reviewed: 2026-07-03

URL Decoder funciona en tres pasos: abrir la herramienta, proporcionar la entrada y obtener el resultado. Esta pagina muestra lo que requiere cada paso.


Paso 1 - abrir la herramienta

Ve a URL Decoder. No hay que instalar ni iniciar sesion antes de empezar.


Paso 2 - proporcionar la entrada

Usalo para crear un valor de cadena de consulta, decodificar un enlace compartido o inspeccionar un parametro con codificacion porcentual.


Paso 3 - obtener el resultado

La codificacion usa encodeURIComponent/decodeURIComponent del propio navegador y se ejecuta localmente - sin subida y sin cuenta.

Nada de lo que escribes se guarda; recargar la pagina borra el campo.
