CSS Box-Shadow Generator - Cuándo Usar

Last reviewed: 2026-07-24

Usa el CSS Box-Shadow Generator cuando quieras una sombra proyectada o una sombra interna (inset) en una tarjeta, botón o panel y prefieras arrastrar controles deslizantes en vez de adivinar valores en píxeles. Esta página explica cuándo encaja y cuándo encaja mejor otra herramienta.


Cuándo encaja

Estás dando estilo a un solo elemento - una tarjeta, botón, modal o marco de imagen - y quieres ver la sombra cambiar en tiempo real al ajustar desplazamiento, desenfoque, propagación, color y opacidad, y luego copiar una línea box-shadow a tu hoja de estilos.


Cuándo no encaja

Si necesitas un fondo en gradiente multicolor con prefijos de navegador en lugar de una sombra, usa el CSS Gradient Generator - es una propiedad CSS distinta, hecha para un efecto visual distinto.
