Steps

  1. Pick a category - length, weight, temperature, and seven more.
  2. Choose the unit to convert from and the unit to convert to.
  3. Type a number in the value box; the result updates as you type.
  4. Use the Swap button to reverse the from and to units.

How the conversions work

Most categories use standard conversion factors: the value is converted to the category's base unit and then to the unit you picked. Temperature is different because it is not a simple ratio, so Celsius, Fahrenheit, and Kelvin use offset formulas, which the tool shows on screen. Your recent conversions are kept in this browser so you can glance back at them, and you can clear them any time.

Privacy
