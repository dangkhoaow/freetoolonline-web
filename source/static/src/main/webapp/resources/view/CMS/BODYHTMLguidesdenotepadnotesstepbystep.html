Notizblock / Notizen Schritt Fuer Schritt

Last reviewed: 2026-07-13
freetoolonline.com Editorial Team

Notizblock / Notizen laeuft in drei Schritten ab: das Tool oeffnen, eine Notiz eintippen, dann in der Liste speichern. Diese Seite erklaert, was jeder Schritt erwartet.


Schritt 1 - das Tool oeffnen

Gehe zum Notizblock / Notizen. Vor dem Start muss nichts installiert oder eingeloggt werden.


Schritt 2 - eine Notiz eintippen

Tippe in das Feld - nutze es als schnellen Notizzettel, eine Telefonnummer, eine Entwurfszeile, oder eine Erinnerung, die du nicht in eine dauerhafte App eintippen willst - dann klicke auf Notiz speichern (oder druecke Strg/Cmd+Enter).


Schritt 3 - mit deinen gespeicherten Notizen arbeiten

Jede Notiz hat eine Kopieren-Schaltflaeche, um ihren Text zu kopieren, und eine Loeschen-Schaltflaeche, um sie zu entfernen; Notizen koennen nicht direkt bearbeitet werden - loesche also eine und fuege eine neue hinzu, um den Text zu aendern.

Notizen werden in diesem Browser ueber lokalen Speicher gesichert, sodass sie ein Neuladen der Seite ueberstehen; exportiere ein Backup (JSON), bevor du Browserdaten loeschst oder das Geraet wechselst, da keines der beiden automatisch synchronisiert wird.
