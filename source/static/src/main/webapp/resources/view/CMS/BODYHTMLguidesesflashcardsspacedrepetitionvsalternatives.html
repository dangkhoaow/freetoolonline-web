Tarjetas de Memoria (Repeticion Espaciada) Vs Alternativas

Last reviewed: 2026-07-13

Tarjetas de Memoria (Repeticion Espaciada) es una de tres formas comunes de crear y estudiar un conjunto de tarjetas: una herramienta basada en el navegador como esta, una aplicacion de escritorio, o una aplicacion o servicio dedicado de repeticion espaciada. Cada una tiene ventajas distintas sobre donde se guardan tus tarjetas y que necesitas instalar.

Esta herramienta es la opcion correcta para un conjunto rapido y autosuficiente que quieres crear y practicar en el navegador. Cuando un conjunto necesita sincronizarse entre dispositivos o programar repasos automaticamente durante semanas, una aplicacion dedicada de repeticion espaciada es la mejor opcion.

Marca una tarjeta como "I knew it" para dejarla de lado en esta pasada, o "Still learning" para mantenerla en la cola; Shuffle aleatoriza el orden de estudio.

Las tarjetas se guardan en este navegador mediante almacenamiento local, asi que sobreviven a una recarga de pagina; exporta un backup (JSON) antes de borrar los datos del navegador o cambiar de dispositivo, ya que ninguno de los dos se sincroniza automaticamente.
