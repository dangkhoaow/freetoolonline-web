Gamepad / Controller Test When To Use

Last reviewed: 2026-07-21

Open Gamepad / Controller Test when you need a quick check that a USB or Bluetooth controller reports buttons and sticks - without installing a desktop utility.


When it fits

A new pad feels dead on one face button. A stick drifts and you want to see axis numbers. You prefer an in-tab check so no driver pack or account is required.


What to expect

Enable polling, press a button so the browser exposes the pad, then watch button indicators and axes. Reset clears the session. Nothing is uploaded.
