The implementing tool for this guide is Milliseconds to Date Converter. Open it in a second tab and the input field on that tool pre-fills with the current millis at page load - refresh the tab and the prefill updates to the new "now". The integer never uploads anywhere because the converter runs the date-format library locally in the browser; the only network call after page load is the CloudFront fetch for the format library itself. That is the privacy posture every guide on this hub shares: bytes you work with stay on-device.

If you would rather not open a tool at all, the browser devtools console reads the value in one line. Press F12 (Cmd+Option+J on macOS), switch to the Console tab, type Date.now(), and press Enter - the live 13-digit integer prints straight away and you can copy it from there. Some corporate-locked browsers hide the console behind a policy; when F12 does nothing, fall back to the converter tab above, whose input field already holds the current millis the moment the page loads.
