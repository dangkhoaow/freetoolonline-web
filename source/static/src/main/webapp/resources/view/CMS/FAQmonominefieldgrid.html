Frequently Asked Questions

What does Mono Minefield Grid do?

It is a minefield clear puzzle: open safe cells, flag mines, and chord on numbered cells across Easy, Medium, and Hard grids. Win score rewards faster clears.

Does it work on a phone?

The page loads on a phone, but flagging and chords need a pointer (right-click / middle-click or Shift-click). A desktop or trackpad is the practical way to play.

Is anything saved in my browser?

Yes. High scores and settings use localStorage key ftol:monominefieldgrid:storage. Reset all high scores in Settings clears the score map. Nothing is uploaded.

How big is the download?

About ~52 KB of HTML, CSS, and JS after you press Play. The browser caches it for later visits on the same device.

How do I play?

Left-click opens a cell. Right-click or Shift/Cmd-click flags. Middle-click chords when neighboring flags match the number. Press R to restart and Esc for the menu. Settings sets Easy / Medium / Hard.

Is this open source?

Yes. It is adapted from the sweeper module in Cursor Arcade by flancast90 (flancast90/cursor-arcade-games) under the MIT license. This site build stubs the VS Code webview API and ships LICENSE and CREDITS.
