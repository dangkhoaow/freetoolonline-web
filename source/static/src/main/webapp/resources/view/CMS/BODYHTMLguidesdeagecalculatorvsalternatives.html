Altersrechner ist eine von drei gaengigen Moeglichkeiten dafuer: ein Browser-Werkzeug wie dieses, eine Desktop-Anwendung oder ein Upload-basierter Online-Dienst. Jede Variante wiegt anders ab, wohin deine Daten gehen und was du installieren musst.
