Quando Usar o Explorador 3D dos Anéis de Chariklo - Melhores Sessões e Limites

Última revisão: 2026-07-24

Abra o Explorador 3D dos Anéis de Chariklo quando quiser ver o menor sistema de anéis conhecido e entender o quanto ele é pequeno perto do de Saturno - não para verificar um feed de rastreamento orbital em tempo real ou estudar a forma 3D exata de Chariklo, que ainda é apenas aproximadamente conhecida.


Uma sessão de história de descoberta

Uma sessão começa orbitando os dois anéis reais em torno de Chariklo para ler os números da descoberta por ocultação estelar de 2013 no painel de fatos, depois ativa a comparação de escala para ver Chariklo encolher a um ponto perto do alcance muito maior dos anéis de Saturno - o mesmo truque de escala consistente torna a diferença de tamanho concreta em vez de abstrata.


Sessões para as quais este explorador não serve

Sessões para as quais esta página não serve incluem verificar um feed de rastreamento orbital em tempo real de Chariklo - use o Explorador 3D dos Anéis de Saturno para o caso do planeta gigante com o qual esta página faz a comparação, ou ler sobre a forma exata do Centauro, que ainda é apenas aproximadamente modelada a partir de dados limitados de ocultação. Percorra os controles no guia passo a passo.
