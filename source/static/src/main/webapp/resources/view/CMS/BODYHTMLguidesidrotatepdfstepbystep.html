Memutar PDF Langkah demi Langkah

Last reviewed: 2026-07-12

Memutar PDF berjalan dalam tiga langkah: buka alat, berikan input Anda, lalu unduh hasilnya. Halaman ini menjelaskan apa yang diharapkan di setiap langkah.


Langkah 1 - buka alat

Buka Memutar PDF. Tidak perlu memasang apa pun atau masuk akun sebelum memulai.


Langkah 2 - berikan input Anda

Sudut rotasi diterapkan secara merata ke seluruh dokumen; tidak mendukung memutar halaman tertentu ke sudut yang berbeda.


Langkah 3 - dapatkan hasilnya

File diproses langsung di tab browser ini dan tidak pernah diunggah ke server.

Untuk memisahkan, menggabungkan, atau mengompres PDF, lihat bagian alat terkait.
