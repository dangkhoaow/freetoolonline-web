Image EXIF Metadata Viewer Langkah Demi Langkah

Last reviewed: 2026-07-21

Image EXIF Metadata Viewer berjalan dalam tiga langkah: buka, pilih gambar lokal, baca tag (strip opsional).


Langkah 1 - buka alat

Buka Image EXIF Metadata Viewer. Tidak perlu instal atau masuk akun.


Langkah 2 - pilih gambar

Gunakan pemilih file untuk JPEG, PNG, WebP, HEIC, atau TIFF dari perangkat ini. Pratinjau muncul. Read Metadata nonaktif sampai file dipilih.


Langkah 3 - baca tag (strip opsional)

Klik Read Metadata. Tabel Camera, Date / Time, GPS, dan Other terisi. Dengan GPS, peringatan merah menampilkan koordinat dan tautan OpenStreetMap. Download Without Metadata menyimpan JPEG 0.95 tanpa EXIF. Clear mereset sesi - tidak ada yang diunggah.

Bandingkan dengan ExifTool desktop: vs alternatif.
