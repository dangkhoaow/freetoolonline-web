Cuándo Usar el Explorador 3D de Estallidos Rápidos de Radio - Mejores Sesiones y Límites

Última revisión: 2026-07-23

Abre el Explorador 3D de Estallidos Rápidos de Radio cuando quieras entender por qué la curva de dispersión de un estallido rápido de radio demuestra que viajó por el espacio intergaláctico, y para comparar Lorimer Burst, FRB 121102 y FRB 180916.J0158+65 lado a lado con sus datos publicados - no para ejecutar un pipeline de dedispersión ni ver un flujo de datos en vivo del CHIME.


Una lección sobre la curva de dispersión

Una lección empieza reproduciendo primero el Lorimer Burst (el descubrimiento original de 2007), luego cambiando al FRB 121102 para ver la misma forma de curva repetirse en la primera fuente repetidora confirmada, y por último el FRB 180916.J0158+65 para leer su ciclo de actividad de 16,35 días en el panel de datos.


Sesiones para las que este explorador no sirve

Sesiones para las que esta página no sirve son estudiar el sincronismo estable de haz-faro de una estrella de neutrones en rotación - usa el Explorador 3D de Púlsar de Estrella de Neutrones para eso - o la intensidad estable del campo magnético de un magnetar - usa el Explorador 3D de Magnetar.
