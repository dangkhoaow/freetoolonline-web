Quando Usar o Explorador 3D de Explosões Rápidas de Rádio - Melhores Sessões e Limites

Última revisão: 2026-07-23

Abra o Explorador 3D de Explosões Rápidas de Rádio quando quiser entender por que a curva de dispersão de uma explosão rápida de rádio prova que ela viajou pelo espaço intergaláctico, e para comparar Lorimer Burst, FRB 121102 e FRB 180916.J0158+65 lado a lado pelos seus dados publicados - não para executar um pipeline de dedispersão nem assistir a um fluxo de dados do CHIME em tempo real.


Uma lição sobre a curva de dispersão

Uma lição começa reproduzindo primeiro o Lorimer Burst (a descoberta original de 2007), depois trocando para o FRB 121102 para ver a mesma forma de curva se repetir na primeira fonte repetidora confirmada, e por fim o FRB 180916.J0158+65 para ler seu ciclo de atividade de 16,35 dias no painel de fatos.


Sessões para as quais este explorador não serve

Sessões para as quais esta página não serve são o estudo do sincronismo estável de feixe-farol de uma estrela de neutrons em rotação - use o Explorador 3D de Pulsar de Estrela de Neutrons para isso - ou a intensidade estável do campo magnético de um magnetar - use o Explorador 3D de Magnetar. Percorra os controles no guia passo a passo.
