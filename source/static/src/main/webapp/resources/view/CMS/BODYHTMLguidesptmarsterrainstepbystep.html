Como Usar Mars Terrain Explorer

Last reviewed: 2026-07-15

Carregue a pagina, espere os dados reais de elevacao baixarem, depois estique o controle de relevo e clique no globo para uma leitura real de elevacao.

  1. Abra Mars Terrain Explorer e espere a esfera provisoria se tornar o terreno real de Marte.
  2. Arraste para orbitar o globo; mantenha o painel de fatos visivel.
  3. Pressione Relief + algumas vezes e veja Olympus Mons subir e Hellas Planitia descer.
  4. Pressione Real scale para ver como Marte e realmente plano na proporcao verdadeira.
  5. Clique em qualquer ponto do globo para ler a elevacao real naquele ponto.
