Quando usar Orbital Velocity 3D Explorer

Use o Orbital Velocity 3D Explorer quando precisar de uma camada quantitativa de velocidade ao longo de uma orbita excentrica - nao para propagacao n-body.

  • Demos de aula: Vis-viva com periapsis mais rapido e apoapsis mais lento
  • Comparar Mercury ~47,36 km/s, Earth ~29,78 km/s, Neptune ~5,43 km/s
  • Repasse rapido apos kepler-orbits ou antes de escape-velocity

Esta pagina nao executa um integrador gravitacional e escala a orbita apenas para leitura didatica.
