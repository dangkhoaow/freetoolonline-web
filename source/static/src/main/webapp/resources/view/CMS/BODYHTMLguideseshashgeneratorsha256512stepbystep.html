Hash Generator (sha-256/512) Paso A Paso

Last reviewed: 2026-07-16

Hash Generator (SHA-256/512) funciona en tres pasos: abrir la herramienta, escribir o pegar tu texto, luego elegir un algoritmo y copiar el digest. Esta pagina explica que espera cada paso.


Paso 1 - abrir la herramienta

Ve a Hash Generator (SHA-256/512). No necesitas instalar nada ni iniciar sesion antes de empezar.


Paso 2 - escribe o pega tu texto

Escribe o pega el texto que quieres hashear en el cuadro; luego elige SHA-1, SHA-256, SHA-384 o SHA-512 en el menu desplegable. El digest se actualiza en vivo abajo mientras escribes.


Paso 3 - copia el digest

Pulsa Copy para obtener el digest hexadecimal resultante. El hash se calcula totalmente en el navegador mediante la API WebCrypto - no se envia nada a un servidor.

MD5 no esta disponible aqui porque los navegadores modernos lo omiten por ser inseguro; usa la herramienta relacionada md5-converter para ese algoritmo especifico.
