What the Quetzalcoatlus size numbers are not

What these size numbers are not is a single tight mass census: mass stays in the published about 70-250 kg range with only the panel baseline 200 kg disclosed - the viewer does not claim one mass outside that range. Soft-tissue color in the viewer is still an artistic reconstruction. This model is not a fossil-accurate skeleton.
