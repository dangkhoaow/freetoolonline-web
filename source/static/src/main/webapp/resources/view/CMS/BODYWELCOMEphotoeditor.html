When to switch to a desktop editor instead

Switch to a desktop editor (Affinity, Photoshop, GIMP) for batch RAW processing, 32-bit HDR pipelines, or 8K canvases. Browser editors hit memory limits past ~25 megapixels on most laptops.

Saving and resuming your work

The home panel keeps a Working list of recent projects. Click any entry to reopen the canvas with its full layer stack intact, pin a project to the top, rename it, or delete it when the edit is done. The list is local to the browser - opening a different device or a private window starts the panel empty.

Non-destructive layers

Each adjustment - filter, exposure, overlay, text, or crop - lives on its own layer. Re-order or remove any one later and the rest of the edit stays put, so the crop can change without redoing the filter.

Exporting a logo or an icon from your edit? Read PNG vs SVG - when to use which for the rule that picks vector output (SVG, scales without re-export) over pixel output (PNG, locks the resolution at edit time). Photographic edits stay in PNG or JPG; SVG only fits flat-colour artwork like logos, icons, and line art.

PNG or JPG at export

PNG keeps every pixel without re-encoding loss - use it for compositions that include text, overlays, or flat-colour artwork where compression artefacts would show. JPG re-encodes the canvas to a smaller file - use it for photographic results where a 200 KB limit (email attachment, social post) matters more than a perfect pixel copy. The layer stack does not export; only the visible composited canvas does.
