Quando Usar o Explorador 3D do Sistema Proxima Centauri - Melhores Sessões e Limites

Última revisão: 2026-07-24

Abra o Explorador 3D do Sistema Proxima Centauri quando você quiser ver exatamente como Proxima b, d e c orbitam a estrela mais próxima do Sol e por que uma nana vermelha propensa a erupções complica a manchete "a estrela mais próxima tem um planeta habitável" - não para modelar a retenção de atmosfera ou a dose real de radiação de uma erupção.


Uma sessão sobre a estrela mais próxima

Uma sessão começa observando os dois planetas mais internos, b e d, e alternando a faixa de zona habitável para ver que b está dentro dela enquanto d, orbitando ainda mais perto, não está - depois dispara uma erupção para ver a complicação de radiação alcançar todos os planetas de uma vez, e lê o painel de dados para as distâncias, períodos e massas exatos.


Sessões que este explorador não atende

Sessões que esta página não atende incluem estudar um sistema distante de cadeia ressonante de 7 planetas - use o Explorador 3D TRAPPIST-1 para isso - ou modelar as chances reais de retenção de atmosfera sob radiação estelar, algo que nenhuma visualização deste site tenta fazer. Percorra os controles no guia passo a passo.
