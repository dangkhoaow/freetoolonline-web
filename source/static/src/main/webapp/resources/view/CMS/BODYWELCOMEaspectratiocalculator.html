Aspect Ratio Calculator

Last reviewed: 2026-07-24

Enter an original width and height and the tool simplifies the ratio down to its lowest terms (for example 1920x1080 reduces to 16:9), showing the decimal value too.

Type a new width or height and the matching side scales automatically along the same ratio - useful for resizing an image, video frame, or design canvas without distorting it. Tap a common preset (16:9, 4:3, 1:1, 21:9, and more) to jump straight to that ratio.

The calculation runs locally in the browser - no upload and no account.
