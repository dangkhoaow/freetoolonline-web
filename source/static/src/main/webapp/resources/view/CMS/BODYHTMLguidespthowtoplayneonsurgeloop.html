Como Jogar Neon Surge Loop - Passo a Passo

A pagina Neon Surge Loop carrega shooter roguelite ~40 KB no iframe.

Passo 2

Mouse move o navio; tiros automaticos.

Passo 3

Level up abre tres cartas de upgrade.
