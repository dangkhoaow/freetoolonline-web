Star Trails 3D Explorer

Last reviewed: 2026-07-17

This browser explorer builds real star trails around the celestial pole, the same long-exposure diurnal-motion effect photographers capture at night. Real literacy: sidereal day ~23h 56m 4s, diurnal rotation rate 15 deg per hour, and the circumpolar-circle radius equals the observer latitude in degrees.

Constellation Sphere 3D owns static asterism shapes and star catalog figures. Sidereal vs Solar Day 3D owns the day-length comparison. This page owns the diurnal-motion trail-building mechanism itself, with real pole-tilt vector geometry.

  • Stars that trace growing circular or partial-arc trails around a golden pole marker
  • Equator, mid-latitude, and near-pole presets that change which stars stay circumpolar
  • Trails that pause when a star dips below the horizon, and resume when it rises again
  • Facts panel lists the sidereal day, rotation rate, and live circumpolar-star count
  • Distinct from constellation-sphere and sidereal-vs-solar-day
  • Runs fully in the browser with the vendored three.js engine - no account, no upload

Students see why circumpolar stars never set while others rise and set nightly; teachers connect the pole-altitude-equals-latitude rule to real geography; curious readers learn the exact sidereal day length behind every star-trail photo.
