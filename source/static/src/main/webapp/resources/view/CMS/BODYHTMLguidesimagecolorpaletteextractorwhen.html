When to extract a color palette from a photo

Use this extractor when you have a local JPEG, PNG, or WebP and need the dominant HEX colors for a design system, mood board, or CSS theme.

Choose the file, set how many colors you want, and click Extract Palette. Sampling stays in the browser - the image is never uploaded.

If you only need to pick a shade by hand, use the Color Picker. For text-on-background readability, use the WCAG Contrast Checker instead.
