Linux im Browser ausfuehren - Schritt fuer Schritt

Linux Online benoetigt vier Schritte von einem leeren Tab bis zu einer laufenden Linux-Maschine, die Sie speichern, exportieren und ueberall fortsetzen koennen. Diese Seite geht jeden Schritt durch, mit den genauen Schaltflaechen-Bezeichnungen. Gesamtzeit bei einer typischen Verbindung: zwei bis fuenf Minuten, fast alles der einmalige Download.

Schritt 1 - Maschine auswaehlen

Oeffnen Sie das Werkzeug und waehlen Sie eine Start-Karte. Terminal gibt Ihnen nach einem kleinen Download eine Befehlszeile; Desktop fuegt eine Taskleiste, Datei-Manager, Texteditor und Terminal fuer einen viel groesseren Download hinzu und benoetigt einen Desktop-Browser. Die Karte zeigt die Download-Groesse an, bevor etwas abgerufen wird; das Image wird in diesem Browser zwischengespeichert, Sie zahlen die Wartezeit also nur einmal.

Schritt 2 - Maschine benutzen

Klicken Sie auf den Bildschirm, um in Linux einzugeben; klicken Sie irgendwo ausserhalb, um die Tastatur an die Seite zurueckzugeben. Es ist echtes Alpine Linux: versuchen Sie uname -a, ls /, vi hello.txt oder htop. Im Desktop-Modus erfasst ein Klick auch die Maus (Esc gibt sie frei), und die Schaltflaeche Strg+Alt+Entf ist vorhanden, wenn Sie sie benoetigen. Die Maschine hat keinen Internetzugang - nichts, was Sie innen ausfuehren, kann nach aussen verbinden.

Schritt 3 - Sitzung speichern

Sitzung speichern speichert die gesamte laufende Maschine - einschliesslich offener Programme - in diesem Browser, und die Auto-Speicher-Option wiederholt das alle fuenf Minuten. Wenn Sie zur Seite zurueckkehren, setzt eine Weiter-Karte genau dort fort, wo Sie aufgehoert haben. Ein frischer Start ist immer moeglich, da Aenderungen nie in das heruntergeladene Image geschrieben werden.

Schritt 4 - Auf einen anderen Computer mitnehmen

Sitzung exportieren laedt eine .ftolvm-Datei herunter - die eingefrorene Maschine. Auf jedem anderen Computer oeffnen Sie dieselbe Seite, waehlen Sitzungsdatei importieren, und die Maschine wird mit Ihrer Shell-Historie und offenen Programmen fortgesetzt. Nicht sicher, ob dieses Werkzeug fuer den Job geeignet ist? Lesen Sie wann Browser-Linux passt und wie es im Vergleich zu den Alternativen abschneidet.
