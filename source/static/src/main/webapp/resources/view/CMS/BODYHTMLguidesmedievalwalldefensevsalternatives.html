Pick Medieval Wall Defense for the deepest browser incremental on this site - wall defense, hero hiring, skill slots, research trees, offline RP, and Crowns prestige without a download. Pick Garden Defense when you want a self-contained five-wave lane puzzle in under fifteen minutes. Pick Pixel Pipeline Reflex for the smallest payload and fastest reflex loop. Pick an installed incremental when you need cross-device cloud saves, live events, and store economies - at the cost of tens to hundreds of megabytes and sign-in.

All three browser games here run client-side with zero server calls during play. Medieval Wall Defense is the largest single file (~631 KB) because it bundles the full Crownfall-derived experience; Garden Defense and Pixel Pipeline Reflex stay lighter for shorter moods.

Numeric takeaway: if install size is your top constraint, Pixel Pipeline Reflex wins at ~48 KB. If session depth is your top constraint, Medieval Wall Defense wins at 30-120+ minutes per run with prestige meta. If you want zero learning curve and a fixed endpoint, Garden Defense wins at exactly 5 waves. Installed incremental TD apps only win when you value cloud sync and live ops over those browser trade-offs.

Control overlap is worth noting: all three browser games accept click or tap, but only Medieval Wall Defense adds a full keyboard layer (1-4 skills, Space, R, Q, M, Esc). Garden Defense is grid-card placement only; Pixel Pipeline Reflex maps four pipeline stages to keys 1-4. Pick the control scheme that matches your device for that session.
