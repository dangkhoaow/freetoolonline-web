Una lección de corrección activada frente a desactivada

Una lección empieza con el interruptor de corrección ACTIVADO para ver la deriva mantenida en cero sin importar cuánto se mueva el control de días, luego cambia el interruptor a DESACTIVADO para ver cómo el mismo número de días produce una deriva de microsegundos creciente y una estimación de error de posición a escala de kilómetros a la par.
