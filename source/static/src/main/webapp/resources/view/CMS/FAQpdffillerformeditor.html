Frequently Asked Questions

Does this tool add fields to a PDF that does not have any?

No. It fills fields that already exist in the PDF (an AcroForm). It does not add new fields to a flat or scanned page and does not run OCR to detect blanks.

Is my PDF uploaded to a server?

No. The file is read in your browser with pdf-lib and never leaves the tab. The filled PDF is generated and downloaded locally as well.

Are the form fields still editable after I download the filled PDF?

Yes. This tool writes values into the existing fields without flattening them, so the downloaded PDF can still be opened and edited as a form in other PDF software.

What happens to my answers if I close the tab?

They are gone - nothing is saved. Download the completed PDF before closing or reloading the page, then reopen that file to keep working from where you left off.
