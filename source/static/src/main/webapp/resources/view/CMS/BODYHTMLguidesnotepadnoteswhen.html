Notepad / Notes App - When To Use It

Last reviewed: 2026-07-13
freetoolonline.com Editorial Team

Notepad / Notes is worth opening when you need a quick, one-off run without installing anything. This page covers when that fits and when it does not.


When it fits

Use it for a quick scratchpad - a phone number, a draft line, or a reminder you do not want to type into a permanent app.


What to expect

Each note has a Copy button to copy its text, and a Delete button to remove it; notes cannot be edited in place, so delete and re-add to change one.

Notes are saved in this browser via local storage, so they survive a page reload; export a backup (JSON) before clearing browser data or switching devices, since neither is synced automatically.

Each note can hold up to 20,000 characters across multiple lines, and pressing Ctrl/Cmd+Enter saves it without leaving the keyboard - plain Enter only starts a new line. To restore a backup, use Import backup and pick that JSON file; it merges the saved notes into your current list instead of replacing it, so any note already in the browser stays unless the imported file updates that same note.
