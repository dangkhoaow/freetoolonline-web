What the Barosaurus size numbers are not

What these size numbers are not is a blended composite outside published ranges. Soft-tissue color is still an artistic reconstruction. This model does not use ~45 m / ~60 t Supersaurus-reattributed estimates as Barosaurus figures; does not assert rearing-to-15-m head height as a measured fact.
