When to use Phobos Stickney Crater 3D Explorer

Use this explorer when you want to see how large Stickney crater is compared with tiny Phobos, and how groove streaks cross the moon.

  • Comparing Stickney's ~9 km bowl to Phobos's ~27 x 22 x 18 km body
  • Introducing the debated groove origins without forcing one answer
  • Giving Mars-system context beside Mars Terrain or tidal-locking pages

This is an educational visualization, not an impact or tidal physics solver.
