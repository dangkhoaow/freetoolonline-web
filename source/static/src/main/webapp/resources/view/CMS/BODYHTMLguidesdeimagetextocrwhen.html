Image To Text (ocr) Wann Verwenden

Last reviewed: 2026-07-16

Image to Text (OCR) lohnt sich zu oeffnen, wenn Sie ein Foto, einen Screenshot oder eine gescannte Seite haben und den Text daraus brauchen, ohne etwas zu installieren. Diese Seite zeigt, wann das passt und wann nicht.


Wann es passt

Ein Zitat oder einen Absatz aus einem Screenshot uebernehmen, statt ihn neu zu tippen. Text aus einem fotografierten Dokument, Schild, Beleg oder Whiteboard wiederherstellen. Eine gescannte Seite in bearbeitbaren, durchsuchbaren oder kopierbaren Text umwandeln, fuer eine einmalige Aufgabe, bei der die Installation einer Desktop-OCR-App mehr Aufwand waere als die Aufgabe selbst.


Was zu erwarten ist

Die Erkennung laeuft komplett lokal ueber eine browserseitige OCR-Engine - das Bild wird nie an einen Server gesendet. Der erste Durchlauf auf diesem Geraet laedt die Erkennungs-Engine herunter (ein paar Megabyte); spaetere Durchlaeufe auf diesem Geraet sind schneller, weil der Browser diesen Download zwischenspeichert.

Funktioniert am besten bei klarem, gedrucktem Text mit gutem Kontrast - die Genauigkeit sinkt bei Handschrift, niedrig aufgeloesten Fotos oder stark schraeg/gedrehtem Text, und die Ausgabe ist nur reiner Text (ohne erhaltenes Layout, Spalten oder Schriftarten).
