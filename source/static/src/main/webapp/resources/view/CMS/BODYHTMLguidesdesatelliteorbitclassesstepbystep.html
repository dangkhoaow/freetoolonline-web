Wie Sie Satellite Orbit Classes 3D Explorer Erkunden - Schritt fuer Schritt

Last reviewed: 2026-07-19

Der Satellite Orbit Classes 3D Explorer startet, sobald die Seite laedt: ziehen Sie, um die Ansicht zu drehen, scrollen oder zoomen Sie mit zwei Fingern, und sehen Sie allen fuenf Bahnklassen gleichzeitig beim Animieren zu. Alles laeuft auf Ihrem eigenen Geraet - keine Installation, kein Konto.


Sehen Sie den fuenf Bahnklassen zu

Sehen Sie den fuenf Bahnklassen ein paar Sekunden zu, bevor Sie etwas anklicken - der niedrige Erdorbit (LEO) umkreist den Planeten sichtbar viele Male, waehrend der geostationaere Orbit (GEO) sich kaum bewegt, weil jeder Satellitenmarker sich mit einer Geschwindigkeit bewegt, die durch seine echte veroeffentlichte Periode festgelegt ist.


Klicken Sie auf eine Klasse zum Hervorheben

Klicken Sie auf eine Klasse zum Hervorheben im Satellite Orbit Classes 3D Visualizer: druecken Sie LEO, MEO, GEO, Molniya oder Sun-sync, und dieser Ring leuchtet auf, waehrend sein Satellitenmarker groesser wird und das Faktenpanel sich aktualisiert.


Lesen Sie die Vergleichstabelle

Lesen Sie die Vergleichstabelle im Faktenpanel: die Zeile der ausgewaehlten Klasse ist unter allen fuenf fett markiert und listet Hoehe, Periode, Neigung und ein echtes Beispielsatelliten fuer jede auf.


Vergleichen Sie zwei Klassen direkt nacheinander

Vergleichen Sie zwei Klassen direkt nacheinander, indem Sie erst LEO, dann GEO klicken - die Tabelle bleibt die ganze Zeit auf dem Bildschirm, sodass die 92.68-Minuten-Periode von LEO und die 23h56m4s-Periode von GEO direkt nebeneinander zum Vergleich stehen.
