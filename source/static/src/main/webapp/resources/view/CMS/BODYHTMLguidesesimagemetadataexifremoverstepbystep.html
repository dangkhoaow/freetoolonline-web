Como quitar metadatos de una foto en el navegador

  1. Abra Image Metadata (EXIF) Remover y elija una imagen local.
  2. Opcional: pulse Read Metadata y revise Camera, Date / Time, GPS y Other.
  3. Si aparece aviso de GPS, abra el enlace OpenStreetMap y continue.
  4. Pulse Download Without Metadata para guardar un JPEG a 0.95 sin EXIF/XMP.
  5. Clear limpia la vista previa y las tablas. Nada se sube al servidor.
