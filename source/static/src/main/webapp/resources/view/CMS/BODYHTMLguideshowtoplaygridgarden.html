How to Play Grid Garden - Step by Step

The Grid Garden page loads a ~1.0 MB CSS grid teaching game in an iframe.

Last reviewed: 2026-07-21

Press Play, read the level prompt, type a CSS grid property in the editor, then click Next when carrots are watered.

Step 1 - press Play

Press Play to inject the iframe (~1.0 MB). The editor and garden appear side by side.

Step 2 - read the prompt

Each level names a grid property to practice (justify-content, align-items, flex-direction, and more).

Step 3 - type CSS

Enter the property in the code box. Frogs move live as your CSS applies to the garden grid.

Step 4 - Next / Reset

Click Next when water matches carrots. Use the level menu Reset to clear ftol:gridgarden localStorage progress.
