Delete PDF Pages

Last reviewed: 2026-07-13

Delete a page range from a PDF, right in the browser.

Cannot delete every page from a PDF - at least one page must remain in the output.

The file is processed in this browser tab and is never uploaded to a server.

  • This tool works on one PDF at a time and does not show page thumbnails, so check the page numbers in a PDF viewer first before typing a range like 1-3,5.
  • Leaving the range blank tells you how many pages the file actually has instead of deleting nothing silently.

Need to pull specific pages out into their own file instead of removing them from the original? Use split PDF by range.
