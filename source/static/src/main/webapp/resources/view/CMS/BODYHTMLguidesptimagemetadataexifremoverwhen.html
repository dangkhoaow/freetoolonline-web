Quando remover EXIF antes de partilhar

Use este removedor quando um JPEG ou HEIC ainda leva modelo da camera, hora ou GPS e vai publicar ou enviar a foto.

Escolha o ficheiro, opcionalmente clique Read Metadata e depois Download Without Metadata. A pagina regrava um JPEG no browser - o original nao sai do dispositivo.

Se so precisa de inspecionar tags, use o Image EXIF Metadata Viewer. Prefira este removedor quando o objetivo e o JPEG limpo.
