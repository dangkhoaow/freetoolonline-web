Jwt Decoder Passo A Passo

Last reviewed: 2026-07-14

JWT Decoder funciona em tres passos: abrir a ferramenta, colar o token e depois ler o resultado decodificado. Esta pagina mostra o que cada passo requer.


Passo 1 - abrir a ferramenta

Acesse JWT Decoder. Nao e necessario instalar nem fazer login antes de comecar.


Passo 2 - colar o token

Cole o token completo - as tres partes separadas por ponto (header, payload, signature). Um token faltando uma parte, ou com uma parte que nao e um JSON base64url valido, e rejeitado com uma mensagem clara em vez de travar.


Passo 3 - obter o resultado

O resultado mostra o header e o payload como JSON formatado. Se o payload tiver exp ou iat, eles aparecem como timestamps legiveis, e um exp ja no passado e marcado EXPIRED.

A decodificacao roda localmente neste navegador - sem upload e sem conta.
