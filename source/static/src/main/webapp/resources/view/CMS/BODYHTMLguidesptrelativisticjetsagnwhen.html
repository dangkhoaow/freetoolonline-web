Quando Usar o Explorador 3D de Jatos Relativísticos e Efeito Doppler - Melhores Sessões e Limites

Última revisão: 2026-07-24

Abra este explorador quando quiser ver por que o jato de um buraco negro parece unilateral vista da Terra, ou para ensinar o efeito Doppler e o movimento superluminal aparente com os números reais de M87 - não para executar uma simulação magnetohidrodinâmica de lançamento de jato.


Uma lição sobre o efeito Doppler

Uma lição começa na base de reinício (sem efeito Doppler) para ver dois jatos igualmente brilhantes, depois carrega o predefinido "M87" para observar um jato brilhar e o outro quase desaparecer, lendo o fator Doppler calculado e a velocidade aparente no painel de dados a cada etapa.


Sessões para as quais este explorador não serve

Sessões para as quais esta página não serve são explorar a geometria exata de órbita de fótons e ISCO de Kerr - use o Explorador 3D de Esfera de Fótons e ISCO para isso - ou visualizar um brilho artístico de disco de acreção sem física de efeito Doppler - use o Explorador 3D de Buraco Negro. Percorra os controles no guia passo a passo.
