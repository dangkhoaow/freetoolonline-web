Number Base Converter (binary/hex/octal/decimal) - Passo A Passo

Last reviewed: 2026-07-23

Number Base Converter (Binary/Hex/Octal/Decimal) funciona em três passos: abrir a ferramenta, digitar seu valor e depois copiar o resultado convertido. Esta página explica o que cada passo espera.


Passo 1 - abrir a ferramenta

Acesse Number Base Converter (Binary/Hex/Octal/Decimal). Não é preciso instalar nada nem fazer login antes de começar.


Passo 2 - informe sua entrada

Bases personalizadas de 2 a 36 são suportadas para conversões menos comuns além das quatro predefinições habituais.


Passo 3 - obtenha o resultado

A conversão usa aritmética exata dígito a dígito com BigInt, não o tipo Number nativo, então valores além de 2^53 - 1 permanecem precisos em vez de perder precisão silenciosamente.

Não trata números negativos em representação de complemento de dois para larguras de bits arbitrárias - apenas a semântica de inteiro com sinal do JavaScript para o sinal de menos.
