Quando Usar o Explorador 3D de Dilatação Temporal do GPS - Melhores Sessões e Limites

Última revisão: 2026-07-24

Abra o Explorador 3D de Dilatação Temporal do GPS quando quiser entender por que o relógio de um satélite GPS precisa de uma correção relativística e o que acontece se essa correção for desligada - não para verificar a telemetria real e atual do relógio de um satélite ao vivo, nem para executar uma projeção de mecânica orbital em escala real.


Uma lição de correção ligada versus desligada

Uma lição começa com o alternador de correção LIGADO, para ver a deriva mantida em zero por mais que o controle de dias seja ajustado, depois muda o alternador para DESLIGADO para ver a mesma contagem de dias gerar uma deriva de microssegundos crescente e uma estimativa de erro de posição em escala de quilômetros correspondente.


Sessões para as quais este explorador não serve

Sessões para as quais esta página não serve incluem verificar a telemetria real e atual do relógio de um satélite ao vivo, ou estudar isoladamente o efeito puro de taxa de relógio gravitacional sem o efeito de velocidade misturado - use o Explorador 3D de Desvio Gravitacional para o Vermelho para isso. Percorra os controles no guia passo a passo.
