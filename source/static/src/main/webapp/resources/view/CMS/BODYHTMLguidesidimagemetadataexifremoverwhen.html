Kapan hapus EXIF sebelum berbagi foto

Gunakan alat ini saat JPEG atau HEIC masih membawa model kamera, waktu, atau GPS dan Anda akan memposting atau mengirim foto.

Pilih file, opsional klik Read Metadata lalu Download Without Metadata. Halaman menulis ulang JPEG di browser - file asli tidak meninggalkan perangkat.

Jika hanya perlu memeriksa tag, gunakan Image EXIF Metadata Viewer. Pilih remover ini bila tujuan Anda adalah JPEG bersih.
