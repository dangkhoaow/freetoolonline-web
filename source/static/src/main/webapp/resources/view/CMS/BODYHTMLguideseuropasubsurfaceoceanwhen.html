When to Use Europa Subsurface Ocean 3D Explorer - Best Sessions and Limits

Last reviewed: 2026-07-23

Open Europa Subsurface Ocean 3D Explorer when you need ice-shell and ocean-depth literacy - about 15-25 km of ice over a 60-150 km ocean - for a Europa Clipper lesson, without a planetarium install or a live mission feed.


A five-minute classroom demo

A five-minute classroom demo starts when you open the explorer, reset Peel shell to 0, then slide past 25, 50, and 75 percent while the class calls out which layer just appeared. Pause on the ocean layer and read the "roughly 2x Earth's oceans" figure out loud - it is the number that tends to land.


What the panel is good for

The panel keeps the published depth ranges in view while you peel: ice shell thickness, ocean depth, and the Europa Clipper mission dates. It is built for the "how do we know that" moment right after a student asks whether the ocean is real.


Sessions this explorer does not fit

Sessions this explorer does not fit are live Europa Clipper telemetry checks, precise interior-density modeling, or plume-venting questions - for those, open Enceladus Geysers 3D Explorer for a different icy moon's real-time plume behavior instead. This page needs WebGL and is not a win/lose game.
