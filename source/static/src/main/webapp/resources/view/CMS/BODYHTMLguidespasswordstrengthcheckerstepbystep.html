Password Strength Checker Step By Step

Last reviewed: 2026-07-23

Password Strength Checker runs in three steps: open the tool, type or paste a password, then read the score. There is no file to upload and no result to download - the readout updates live while you type.


Step 1 - open the tool

Go to Password Strength Checker. Nothing needs to be installed or signed in before you start.


Step 2 - type or paste a password

Click into the password field and type, or paste, the password you want to check. The strength bar, the 0 to 4 score, and the estimated crack time update as you type - there is no separate "check" button to click. Use the Show button next to the field to reveal the characters and confirm you typed what you meant to.


Step 3 - read the score and suggestions

The score comes from zxcvbn, the same pattern-matching + dictionary-attack model used by Dropbox, not a simple length/character-class rule - it flags common passwords and predictable patterns that "P@ssw0rd1" style rules miss. Below the score, any warning and suggestions from zxcvbn are listed - for example, a suggestion to avoid a repeated pattern or a common word. Edit the password in the field and the score, crack-time estimate, and suggestions all update again immediately - there is nothing to save or download, so you can try as many variations as you want.

Everything runs in the browser - no upload, no account, and the password is never sent anywhere, logged, or checked against a breach-database API.

This tool does not store or recover passwords; it only scores whatever is currently typed into the field.
