Gamepad / Controller Test Quando Usar

Last reviewed: 2026-07-21

Abra o Gamepad / Controller Test quando precisar confirmar se um comando USB ou Bluetooth reporta botoes e sticks - sem instalar utilitario de secretaria.


Quando se aplica

Um comando novo parece morto num botao. Um stick deriva e quer ver os valores do eixo. Prefere um teste no separador, sem conta.


O que esperar

Ative a leitura, pressione um botao para o navegador expor o pad, veja indicadores e eixos. Reset limpa a sessao. Nada e enviado.

Passos exactos: guia passo a passo.
