Add Watermark to PDF

Last reviewed: 2026-07-12

Add a watermark to every page of a PDF, right in the browser.

Applies one fixed diagonal text style (gray, semi-transparent, centered) - does not support custom position, color, size, image watermarks, or a different watermark per page.

The watermark text size adjusts automatically to fit each page's width, so the same text stays readable whether the PDF is portrait or landscape, letter or A4.

The same watermark text is stamped on every page of a multi-page PDF in one pass - no repeated uploads or per-page steps.

The file is processed in this browser tab and is never uploaded to a server.
