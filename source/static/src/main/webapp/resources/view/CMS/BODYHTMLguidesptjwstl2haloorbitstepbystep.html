Como usar o Explorador 3D da Órbita Halo L2 do JWST

  1. Abra o Explorador 3D da Órbita Halo L2 do JWST e espere o canvas ser desenhado.
  2. Arraste para orbitar a visão e role para ampliar o laço halo em torno do marcador L2.
  3. Arraste o controle de dia halo de 0 a 168 para mover o Webb para qualquer ponto do trajeto.
  4. Pressione Reproduzir órbita para animar um ciclo halo completo continuamente.
  5. Alterne os marcadores de manobra para ver aproximadamente onde ocorrem as manobras de manutenção a cada 21 dias.
