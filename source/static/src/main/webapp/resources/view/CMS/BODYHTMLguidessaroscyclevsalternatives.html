Saros Cycle Eclipse Explorer 3D vs alternatives

The Saros Cycle Eclipse Explorer 3D shows the westward-stepping ground-track pattern across a whole series in the browser - eclipse almanacs win for exact historical and future dates; astronomy textbooks win for the full orbital-mechanics derivation.
