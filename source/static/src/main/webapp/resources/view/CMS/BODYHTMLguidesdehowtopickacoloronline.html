Farbe online auswaehlen - HEX, RGB und HSL

Ein Farbwaehler laesst Sie eine Farbe waehlen und sofort ihre HEX-, RGB- und HSL-Werte ablesen. Waehlen Sie einen Ton, kopieren Sie den Wert in Ihr CSS und sehen Sie komplementaere, analoge und Ton-Paletten. Eine Kontrastanzeige sagt, ob Text auf der Farbe lesbar ist. Alles laeuft im Browser, ohne Upload und ohne Konto.

Schritte

  1. Oeffnen Sie das Farbfeld und waehlen Sie einen Ton.
  2. Lesen Sie die HEX-, RGB- und HSL-Werte; klicken Sie auf Kopieren, um einen in die Zwischenablage zu legen.
  3. Nutzen Sie ein komplementaeres, analoges oder Ton-Feld als naechste Farbe, indem Sie darauf klicken.
  4. Speichern Sie wiederverwendete Farben in den Favoriten; die letzten Farben werden automatisch gemerkt.

HEX, RGB und HSL

HEX ist ein sechsstelliger Code wie #3366FF. RGB listet die Kanaele Rot, Gruen und Blau von 0 bis 255. HSL gibt Farbton (0 bis 360 Grad), Saettigung und Helligkeit in Prozent an. Das Werkzeug zeigt alle drei fuer die gewaehlte Farbe, plus eine Kontrastanzeige nach der WCAG-Formel fuer relative Leuchtdichte, um die Lesbarkeit zu beurteilen.
