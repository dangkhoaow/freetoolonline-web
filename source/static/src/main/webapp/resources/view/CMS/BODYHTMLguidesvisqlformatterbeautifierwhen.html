SQL Formatter & Beautifier - Khi Nào Dùng

Last reviewed: 2026-07-21

Mở SQL Formatter & Beautifier khi câu lệnh khó đọc và bạn muốn xuống dòng theo mệnh đề cùng từ khóa viết hoa mà không dán SQL lên trang từ xa.


Khi phù hợp

Dùng cho bản nháp SELECT / INSERT / UPDATE / DELETE, câu lệnh chép từ log, và đọc nhanh trước khi review.


Khi nên bỏ qua

Bỏ qua nếu cần kiểm tra dialect, EXPLAIN hoặc viết lại theo schema - trang này chỉ định dạng lại văn bản trong tab.
