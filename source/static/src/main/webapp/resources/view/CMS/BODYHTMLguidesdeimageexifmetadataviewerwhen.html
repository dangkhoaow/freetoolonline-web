Image EXIF Metadata Viewer Wann Nutzen

Last reviewed: 2026-07-21

Oeffnen Sie Image EXIF Metadata Viewer, wenn Sie Kamera-, Datums- und GPS-Tags eines Fotos vor dem Teilen pruefen wollen - ohne Upload und ohne ExifTool-Installation.


Wann es passt

Handyfoto vor dem oeffentlichen Posten pruefen. Bestaetigen, ob ein JPEG noch ein Aufnahmedatum traegt. GPS in einem Reisefoto finden. Lokales Parsen bevorzugen, wenn die Datei auf diesem Geraet bleiben muss.


Was zu erwarten ist

Bild waehlen, Read Metadata klicken, dann Camera, Date / Time, GPS und Other pruefen. Bei GPS erscheinen Privatsphaere-Hinweis und OpenStreetMap-Link; optional JPEG ohne Metadaten laden. Alles im Browser - kein Upload, kein Konto.
