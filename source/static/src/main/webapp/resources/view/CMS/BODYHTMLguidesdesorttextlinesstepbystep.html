Textzeilen Schritt fuer Schritt sortieren: eine Liste im Browser ordnen

Das kostenlose Tool Sort Text Lines auf dieser Seite ordnet einen Textblock zeilenweise neu, A-Z oder Z-A, vollstaendig in Ihrem Browser. Diese Anleitung geht die genauen Schritte durch und erklaert, wie das Tool die Reihenfolge bestimmt.

Antwort in 30 Sekunden. Oeffnen Sie Sort Text Lines, fuegen Sie Ihre Liste mit einem Eintrag pro Zeile in das obere Feld ein und klicken Sie auf Sort A-Z fuer aufsteigende oder Sort Z-A fuer absteigende Reihenfolge. Die sortierte Liste erscheint im Ausgabefeld darunter, bereit zum Kopieren.

So sortieren Sie Ihre Zeilen

  1. Oeffnen Sie das Tool Sort Text Lines.
  2. Tippen oder fuegen Sie Ihren Text in das Eingabefeld ein und halten Sie einen Eintrag pro Zeile - das Tool behandelt jeden Zeilenumbruch als Grenze zwischen zwei Eintraegen.
  3. Klicken Sie auf Sort A-Z, um die Zeilen aufsteigend zu sortieren, oder auf Sort Z-A, um sie absteigend zu sortieren.
  4. Lesen Sie die sortierte Liste im Ausgabefeld darunter und kopieren Sie sie heraus. Klicken Sie jederzeit auf die andere Schaltflaeche, um die Reihenfolge umzukehren.

Wie die Reihenfolge bestimmt wird

Die Sortierung verwendet den im Browser eingebauten, sprachbewussten Textvergleich, sodass die Zeilen Buchstabe fuer Buchstabe geordnet werden. Zwei Dinge sollten Sie wissen, bevor Sie dem Ergebnis vertrauen:

  • Gross- und Kleinschreibung bleiben zusammen. Die Gross- und Kleinschreibvariante desselben Wortes wird nebeneinander sortiert, sodass "apple" und "Apple" nebeneinander stehen und nicht in zwei getrennten Bloecken.
  • Zahlen werden als Text verglichen, nicht als Werte. Da jede Zeile Zeichen fuer Zeichen verglichen wird, stehen "10" und "100" vor "2" (das Tool liest die fuehrende "1" vor der "2"). Fuellen Sie bei einer Liste einfacher Zahlen alle auf dieselbe Breite auf - "02", "10", "100" - um die erwartete numerische Reihenfolge zu erhalten.

Haeufige Anwendungen

Eine Liste von Namen oder E-Mail-Adressen alphabetisch ordnen; eine Reihe von Tags, Schluesselwoertern oder CSS-Klassen sortieren; die Zeilen einer Konfigurationsdatei oder einer Wortliste vor einem Diff ordnen, damit beide Seiten uebereinstimmen; oder einfach eine eingefuegte Liste aufraeumen, damit Duplikate und Luecken leichter mit dem Auge zu erkennen sind.

Sort Text Lines im Vergleich zu einer Tabellenkalkulation oder einem Desktop-Editor

Eine Tabellenkalkulation kann eine Spalte sortieren, aber Sie muessen die Liste zuerst in einen Zellbereich einfuegen, sie auswaehlen und ein Sortierdialogfeld oeffnen; ein Desktop-Code-Editor kann ausgewaehlte Zeilen sortieren, braucht dafuer aber meist ein Plugin oder einen Befehl. Sort Text Lines ist ein einziges Feld mit zwei Schaltflaechen - ohne Upload, ohne Importschritt und ohne Installation - was es schneller macht, wenn die Liste noch nicht in einem Dokument steht. Fuer schwere Arbeit mit mehreren Spalten oder numerischer Sortierung ist eine Tabellenkalkulation weiterhin die bessere Wahl.

Datenschutz

Die Sortierung laeuft vollstaendig im Tab Ihres Browsers mit einfachem JavaScript. Nichts, was Sie tippen oder einfuegen, wird hochgeladen, an einen Analysedienst gesendet oder auf einem Server gespeichert - ein Neuladen oder Verlassen der Seite leert das Feld, ohne Verlauf und ohne Konto.

Haeufig gestellte Fragen

Woher weiss das Tool, wo ein Eintrag endet und der naechste beginnt?

Jede Zeile ist ein Eintrag. Das Tool teilt Ihren Text an den Zeilenumbruechen, sortiert die entstandene Liste und fuegt sie mit Zeilenumbruechen wieder zusammen, setzen Sie also genau einen Eintrag pro Zeile.

Warum stand "10" in meiner sortierten Liste vor "2"?

Zeilen werden als Text verglichen, Zeichen fuer Zeichen, nicht als Zahlen, daher steht "10" vor "2", weil "1" vor "2" kommt. Um einfache Zahlen in numerischer Reihenfolge zu sortieren, fuellen Sie sie zuerst auf dieselbe Anzahl Ziffern auf (02, 10, 100).

Kann ich eine Sortierung rueckgaengig machen oder die urspruengliche Reihenfolge zurueckbekommen?

Das Ausgabefeld haelt das sortierte Ergebnis; Ihr Originaltext bleibt im Eingabefeld, bis Sie ihn aendern. Das Sortieren ueberschreibt die Eingabe nicht, also koennen Sie so oft Sie moechten auf Sort A-Z und Sort Z-A klicken, um beide Reihenfolgen zu vergleichen.

Wird mein Text irgendwo hochgeladen oder gespeichert?

Nein. Die Sortierung laeuft vollstaendig in Ihrem Browser; nichts, was Sie tippen oder einfuegen, wird an einen Server gesendet, und nichts wird gespeichert, sobald Sie die Seite neu laden oder verlassen.
