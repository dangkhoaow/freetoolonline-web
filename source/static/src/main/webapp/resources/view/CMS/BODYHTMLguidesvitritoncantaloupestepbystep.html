How to use Triton Cantaloupe Terrain 3D Explorer

  1. Open the Triton Cantaloupe Terrain 3D Explorer and wait for the canvas to paint.
  2. Press Play if paused, then drag to face the dimpled hemisphere.
  3. Toggle Dimples Off and On to see the schematic cavi grid alone.
  4. Move Speed while you read the size table in the facts panel.
  5. Use Fullscreen when teaching Voyager-era terrain on a large display.
