Choose a photo, pick the country/document size preset (or enter a custom size in millimeters), drag to position the face inside the frame, and use the zoom slider to fill it - then click Generate photo.

Cropping a photo to a passport or visa size takes four steps:

  1. Choose a photo file - it opens in the browser and is never uploaded to a server.
  2. Pick a country/document size preset (US, India, Schengen/EU, UK, Australia, Canada, China visa) or switch to "Custom size (mm)" and enter your own width and height in millimeters.
  3. Drag the photo to position the face inside the frame, and use the Zoom slider to fill it; check "Also generate a 4x6in print sheet" if you want multiple copies tiled onto one printable page.
  4. Click Generate photo, then use the Download links to save the cropped photo (and the print sheet, if you checked that box) - both are composited onto a plain white background at the preset's exact pixel size for 300 DPI printing.
