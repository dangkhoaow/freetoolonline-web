TDEE & Calorie Calculator When To Use

Last reviewed: 2026-07-24

TDEE & Calorie Calculator is worth opening when you need a quick, one-off run without installing anything. This page covers when that fits and when it does not.


When it fits

Use it to set a starting calorie target for maintaining, losing, or gaining weight.


What to expect

The calculation runs locally in the browser - no upload and no account.

Uses the Mifflin-St Jeor formula for BMR, then applies an activity multiplier to estimate total daily energy expenditure (TDEE), with a loss and gain target 500 kcal/day either side of maintenance.
