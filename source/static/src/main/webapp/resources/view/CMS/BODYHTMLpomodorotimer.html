Click Start to begin the Work interval; the timer switches to Break automatically when it reaches zero, then back to Work when the Break ends.

  1. Set Work minutes (1-120, default 25) and Break minutes (1-60, default 5), or leave the defaults for the classic 25/5 split.
  2. Click Start. The phase label reads "Work" and the countdown ticks down.
  3. Let it flip on its own: "Break" starts when Work hits zero, "Work" starts again when Break hits zero, repeating for as long as the tab stays open.
