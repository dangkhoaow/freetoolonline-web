Eine fuenfminuetige Unterrichtsdemo beginnt, wenn Sie Gravity Well 3D Explorer oeffnen, Mond dann Sonne antippen und lesen, wie die Fluchtgeschwindigkeit im Faktenpanel von 2.38 km/s auf 617.5 km/s springt - schneller als vier Zahlen allein auf einer Folie.


Schwarzschild-Radius-Literacy

Schwarzschild-Radius-Literacy bleibt lesbar, wenn Sie Show r_s (Sun) umschalten und lesen, dass der Bildschirmring fuer Unterricht uebertrieben ist, waehrend das Panel den veroeffentlichten Wert etwa 2.95 km aus r_s = 2GM/c^2 behaelt. Das Blatt ist ein Gummituch-Lehrproxy, kein physikalischer GR-Metrik-Loeser.
