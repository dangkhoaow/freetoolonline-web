CSS Box-Shadow Generator - Passo a Passo

Last reviewed: 2026-07-24

O CSS Box-Shadow Generator funciona em três passos: abrir a ferramenta, arrastar os controles, depois copiar o resultado. Esta página detalha o que cada passo faz.


Passo 1 - abra a ferramenta

Acesse o CSS Box-Shadow Generator. A caixa de pré-visualização e uma sombra inicial aparecem imediatamente - nada precisa ser instalado ou logado antes.


Passo 2 - molde a sombra

Arraste o deslocamento horizontal e vertical para mover a sombra, o raio de desfoque para suavizar sua borda, e o raio de espalhamento para aumentar ou diminuir seu tamanho. Escolha uma cor e opacidade para a sombra, e marque Inset se quiser uma sombra interna em vez de projetada. A pré-visualização atualiza a cada mudança.


Passo 3 - copie o resultado

Clique em Copy CSS para copiar a linha box-shadow gerada para a área de transferência, ou selecione o texto na caixa de saída manualmente. Tudo acontece na aba do seu navegador; nada é enviado.
