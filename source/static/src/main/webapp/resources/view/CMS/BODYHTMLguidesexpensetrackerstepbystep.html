Expense Tracker Step By Step

Last reviewed: 2026-07-11

Expense Tracker logs a description and an amount for each expense, then keeps a running total as you add entries. This page walks through logging your first expense and reading the total.


Step 1 - open the tool

Go to Expense Tracker. Nothing needs to be installed or signed in before you start.


Step 2 - log an expense

Type a short description (for example "Coffee") and an amount (for example 4.50), then press Enter on either field or click Add expense. An amount that is left blank, is not a number, or is negative is rejected with an inline message instead of being added.


Step 3 - read the total and manage entries

The running total above the list adds up every expense currently in the list. Use the checkbox next to an entry to mark it reviewed, or Delete to remove it - removing an entry updates the total right away.

The list is saved in this browser via local storage, so it survives a page reload; export a JSON backup before clearing your browser data, since the list only exists in the browser that added it.
