Khi nao phu hop

Dan ban nhap README de xac nhan tieu de, danh sach va lien ket truoc khi commit. Doi ghi chu Markdown thanh HTML cho CMS chi nhan HTML. Xem truoc HTML email tren may. Phat hien HTML nhung rui ro vi ban xem truoc di qua DOMPurify.


Nen mong doi gi

Go hoac dan vao o ben trai; ban xem truoc cap nhat khi ban go. Chuyen sang HTML source de copy markup da loc. Dong tab la mat het; khong luu.
