Galilean Moons 3D Explorer langkah demi langkah

Buka Galilean Moons 3D Explorer, lihat empat bulan mengorbit dengan rasio periode nyata dan klik tiap bulan untuk angka NASA JPL.

  1. Buka /space-3d/galilean-moons.html dan tunggu Jupiter plus empat bulan dan cincin orbit.
  2. Klik Io dulu - periode 1.769 hari, diameter sekitar 3,643 km - lalu Europa (3.551 d).
  3. Klik Ganymede (7.155 d) dan cek mendekati 4x periode Io.
  4. Klik Callisto (16.689 d) lebih jauh - di luar kunci 4:2:1.
  5. Pause orbits membekukan jarak; Overview table menampilkan empat baris.
