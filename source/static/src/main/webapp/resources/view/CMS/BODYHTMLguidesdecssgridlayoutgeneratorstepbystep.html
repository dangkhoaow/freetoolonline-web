CSS-Grid-Layout-Generator - Schritt für Schritt

Last reviewed: 2026-07-23

Der CSS-Grid-Layout-Generator erstellt in drei Schritten einen einfügbaren display: grid-CSS-Block: Spalten- und Zeilenanzahl festlegen, Abstände festlegen, Ergebnis kopieren. Es wird nichts hochgeladen und nichts installiert.


Schritt 1 - Spalten und Zeilen festlegen

Öffnen Sie den CSS-Grid-Layout-Generator. Ziehen Sie den Regler Columns (1 bis 12) und den Regler Rows (1 bis 6), um dem gewünschten Layout zu entsprechen - das Vorschauraster rechts zeichnet die Zellen sofort neu.


Schritt 2 - Abstände festlegen

Ziehen Sie die Regler Column gap und Row gap (jeweils 0px bis 50px), um den Abstand zwischen den Zellen zu steuern. Das Vorschauraster und der darunter erzeugte CSS-Text aktualisieren sich live mit den exakten Pixelwerten.


Schritt 3 - Ergebnis kopieren

Klicken Sie auf "Copy CSS", um den erzeugten Block in die Zwischenablage zu kopieren. Falls der Browser den Zugriff auf die Zwischenablage blockiert, fordert eine Statusmeldung Sie auf, den Text manuell auszuwählen und zu kopieren. Die gesamte Seite läuft im Browser, daher wird kein Reglerwert an einen Server gesendet.
