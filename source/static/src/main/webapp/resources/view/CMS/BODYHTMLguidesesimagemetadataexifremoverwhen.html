Cuando quitar EXIF antes de compartir

Use este removedor cuando un JPEG o HEIC aun lleva modelo de camara, hora o GPS y va a publicar o enviar la foto.

Elija el archivo, opcionalmente pulse Read Metadata y luego Download Without Metadata. La pagina reescribe un JPEG en el navegador - el original no sale del dispositivo.

Si solo necesita inspeccionar etiquetas, use Image EXIF Metadata Viewer. Prefiera este removedor cuando el objetivo es el JPEG limpio.
