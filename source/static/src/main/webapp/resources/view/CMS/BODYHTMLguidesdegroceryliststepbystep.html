Grocery List Schritt Fuer Schritt

Last reviewed: 2026-07-09

Grocery List laeuft in drei Schritten: das Tool oeffnen, die Eingabe machen, dann das Ergebnis herunterladen. Diese Seite erklaert, was jeder Schritt erwartet.


Schritt 1 - das Tool oeffnen

Gehe zu Grocery List. Vor dem Start muss nichts installiert oder angemeldet werden.


Schritt 2 - die Eingabe machen

Nutze es fuer eine schnelle Checkliste, wenn du keine App installieren oder dich anmelden willst.


Schritt 3 - das Ergebnis erhalten

Jeder Eintrag hat ein Kaestchen, um ihn als erledigt zu markieren, und eine Schaltflaeche zum Entfernen.

Die Liste wird in diesem Browser per local storage gespeichert, sie bleibt also nach einem Neuladen der Seite bestehen; das Loeschen der Browserdaten dieser Seite entfernt sie.
