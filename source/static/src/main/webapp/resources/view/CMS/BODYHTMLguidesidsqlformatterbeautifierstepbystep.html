SQL Formatter & Beautifier - Langkah demi Langkah

Last reviewed: 2026-07-21

Buka SQL Formatter & Beautifier, tempel SQL di kiri, klik Format, lalu salin hasil yang sudah diindentasi di kanan.


Langkah

  1. Tempel atau ketik SQL di kotak input.
  2. Klik tombol Format merah.
  3. Periksa keluaran: klausa utama baris baru, kata kunci huruf besar.
  4. Klik Copy Output dan tempel ke editor.

Privasi

Pemformatan berjalan di peramban. Teks kueri tidak diunggah.
