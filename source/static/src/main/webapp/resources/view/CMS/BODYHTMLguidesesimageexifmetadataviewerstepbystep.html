Image EXIF Metadata Viewer Paso A Paso

Last reviewed: 2026-07-21

Image EXIF Metadata Viewer funciona en tres pasos: abrir, elegir imagen local, leer etiquetas (strip opcional).


Paso 1 - abrir la herramienta

Ve a Image EXIF Metadata Viewer. No necesitas instalar nada ni iniciar sesion.


Paso 2 - elegir una imagen

Usa el selector para JPEG, PNG, WebP, HEIC o TIFF de este dispositivo. Aparece una vista previa. Read Metadata queda desactivado hasta elegir archivo.


Paso 3 - leer etiquetas (strip opcional)

Pulsa Read Metadata. Las tablas Camera, Date / Time, GPS y Other llenan la pagina. Con GPS, el aviso rojo muestra coordenadas y enlace OpenStreetMap. Download Without Metadata guarda JPEG a 0.95 sin EXIF. Clear limpia la sesion - no se subio nada.
