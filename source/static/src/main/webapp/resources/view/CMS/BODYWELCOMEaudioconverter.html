Audio Converter

Last reviewed: 2026-07-02

Convert audio files between common formats in the browser. Pick the target format from the dropdown, upload or drop your file, and save the result - conversion runs in the tab using FFmpeg.wasm and nothing reaches a server. The working copy is released when you navigate away.

Supported input and output formats include MP3, WAV, AAC, OGG, FLAC, and M4A. The bitrate dropdown lets you choose 128, 192, 256, or 320 kbps for audio output. Your input file is processed locally and discarded when the page closes - nothing is stored or uploaded at any step.
