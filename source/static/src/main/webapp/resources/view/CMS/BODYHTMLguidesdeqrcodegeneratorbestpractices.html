30-Sekunden-Antwort. Ein QR-Code, der immer scannt, haengt von fuenf Einstellungen ab. Fehlerkorrektur auf Stufe M oder hoeher (laesst den Code leichte Schaeden ueberleben). Hoher Kontrast (schwarz auf weiss, keine hellen Gradienten). Mindestens 2 cm physische Groesse (kleiner scheitert bei Handykameras). Ruhezone von mindestens vier Modulen um den Rand. Und eine kurze Payload - lange URLs erzeugen dichte, schwer scanbare Codes; verwenden Sie zuerst einen URL-Verkuerzer.

Fehlerkorrektur - die am meisten uebersprungene Einstellung

QR-Codes haben vier Fehlerkorrekturstufen. Sie sagen dem Encoder, wie viel des Codes beschaedigt oder verdeckt werden kann, bevor das Scannen fehlschlaegt:

  • L (Niedrig). Stellt bis zu ~7% Schaden wieder her. Kleinster Code, geringste Fehlertoleranz. Nur richtig, wenn der Code auf einem Bildschirm ist und nicht zerkratzt wird.
  • M (Mittel). Stellt ~15% Schaden wieder her. Standard in den meisten Generatoren. Gut fuer die meisten Druckkontexte.
  • Q (Quartil). Stellt ~25% Schaden wieder her. Richtig, wenn der Code ein Logo ueberlappen oder teilweise bedeckt werden koennte.
  • H (Hoch). Stellt ~30% Schaden wieder her. Die richtige Wahl fuer Codes auf Aussen-Beschilderung, auf Verpackungen oder ueberall, wo Verschleiss erwartet wird.

Der Trade-off: hoehere Fehlerkorrektur macht den Code dichter (mehr schwarz-weisse Quadrate pro Flaeche) fuer dieselbe Payload. Waehlen Sie die niedrigste Stufe, die Ihre Umgebung toleriert.

Kontrast und Farbe

Der Standard ist dunkler Vordergrund auf hellem Hintergrund. Schwarz auf weiss scannt am besten. Invertiert (hell auf dunkel) scheitert oft, weil Handykamera-Scanner die Standardpolaritaet annehmen.

Benutzerdefinierte Farben funktionieren, wenn das Kontrastverhaeltnis ueber 4:1 bleibt - ein dunkles Blau auf Weiss ist in Ordnung, ein mittleres Grau auf Weiss nicht. Hell-auf-Dunkel ist unzuverlaessig; verwenden Sie niemals weiss-auf-schwarz QR fuer Druckmaterialien. Gradienten-Hintergruende brechen das Scannen vollstaendig; der Code-Bereich muss eine einzelne Volltonfarbe sein.

Physische Groesse und Ruhezone

Die mindestscanbare Groesse haengt von der Entfernung ab. Faustregel: die Seitenlaenge des Codes sollte mindestens 1/10 der Scan-Distanz betragen. Ein Code, der aus 30 cm gescannt wird (ein Telefon in der Hand), muss mindestens 3 cm sein. Ein Code, der aus 3 m gescannt wird (ein Poster), muss mindestens 30 cm sein.

Die Ruhezone ist der weisse Raum um den Code. Die QR-Spezifikation erfordert mindestens vier Module (die kleinen Quadrate) Rand auf allen vier Seiten. Den Code direkt an Text, einen Rahmen oder eine andere Grafik zu setzen, verursacht Scanner-Fehler, auch wenn der Code selbst intakt ist.

Payload-Laenge ist wichtiger, als Sie denken

Die Dichte eines QR-Codes waechst mit der Payload-Groesse. Eine 10-Zeichen-URL kodieren erzeugt einen sparsamen Code, der aus einem Meter Entfernung scannt. Eine 200-Zeichen-URL mit Tracking-Parametern kodieren erzeugt einen dichten Code, der physisch groesser sein muss und aus naeherer Entfernung scannt.

Verkuerzen Sie die URL immer zuerst. example.com/p ist besser als example.com/products/winter-collection?utm_source=poster&utm_campaign=2026-q1. Die Weiterleitung kann die Tracking-Parameter serverseitig tragen; der QR traegt nur den kurzen Link. Das Ergebnis ist ein kleinerer, zuverlaessigerer Code.

Testen vor dem Drucken

Drucken Sie den Code in der Groesse, die Sie verwenden moechten, scannen Sie aus der Entfernung und dem Winkel, den Sie erwarten, und probieren Sie mindestens zwei Telefone aus - ein neues, eines von vor ein paar Jahren. Ein Code, der aus 5 cm scannt, aber aus 50 cm scheitert, hat die falsche Groesse. Der QR-Code-Generator zeigt eine Vorschau des Codes in mehreren Groessen vor dem Herunterladen. Ein zentrales Logo hinzufuegen

Der QR-Code-Generator kann ein Logo in die Mitte des Codes setzen. Waehlen Sie ein Bild von Ihrem Geraet, und es sitzt auf einer kleinen weissen Flaeche in der Mitte; das Bild wird in Ihrem Browser gelesen und nie hochgeladen. Wenn Sie ein Logo hinzufuegen, hebt das Tool die Fehlerkorrektur automatisch auf High an, damit die vom Logo verdeckten Quadrate weiterhin rekonstruiert werden koennen.

Halten Sie das Logo klein - das Tool legt es auf etwa ein Fuenftel der Codebreite fest, was ein Code mit High-Korrektur vertraegt. Ein groesseres Logo kann das Scannen trotzdem stoeren, erzeugen Sie also den Code und scannen Sie das fertige Bild aus der erwarteten Entfernung, bevor Sie es drucken oder veroeffentlichen. Sie koennen ausserdem die Vorder- und Hintergrundfarbe an eine Marke anpassen, solange der Kontrast hoch bleibt.
