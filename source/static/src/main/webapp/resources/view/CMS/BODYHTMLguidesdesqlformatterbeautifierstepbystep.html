SQL Formatter & Beautifier - Schritt für Schritt

Last reviewed: 2026-07-21

Öffne SQL Formatter & Beautifier, füge links SQL ein, klicke Format und kopiere rechts das eingerückte Ergebnis.


Schritte

  1. SQL in das Eingabefeld einfügen oder tippen.
  2. Den roten Format-Button klicken.
  3. Ausgabe prüfen: Hauptklauseln in neuen Zeilen, Schlüsselwörter in Großbuchstaben.
  4. Copy Output klicken und in den Editor einfügen.

Privatsphäre

Die Formatierung läuft im Browser. Der Abfragetext wird nicht hochgeladen.
