2048 Spielen - Schritt fuer Schritt

Last reviewed: 2026-07-11

Das 2048 Game laeuft komplett auf dieser Seite: das 4x4-Feld mit zwei Startsteinen erscheint, sobald die Seite geladen ist, und Ihre erste Pfeiltaste oder Wischgeste ist bereits Ihr erster Zug. Hier ist jeder Schritt von diesem ersten Schieben bis zu einer gespeicherten Bestpunktzahl.


Schritt 1 - die Steine schieben

Nutzen Sie die Pfeiltasten oder WASD auf der Tastatur, wischen Sie irgendwo auf dem Feld auf dem Handy, oder tippen Sie auf das Steuerkreuz auf dem Bildschirm. Jeder Zug schiebt ALLE Steine so weit wie moeglich in diese Richtung.


Schritt 2 - gleiche Steine verschmelzen

Zwei gleiche Steine, die zusammenstossen, verschmelzen zu ihrer Summe - aus zwei 4ern wird eine 8, aus zwei 8ern eine 16, und so weiter. Jeder Stein verschmilzt hoechstens einmal pro Zug (zwei Paare von 2ern werden zu zwei 4ern, niemals zu einer 8), und verschmolzene Steine machen einen kleinen Sprung, damit Sie sehen, woher die Punkte kamen.


Schritt 3 - den neuen Stein einplanen

Nach jedem Zug, der das Feld veraendert hat, erscheint ein neuer Stein mit der 2 auf einem zufaelligen leeren Feld - etwa einmal von zehn ist es stattdessen eine 4. Zuege, die nichts veraendern, werden ignoriert und erzeugen keinen neuen Stein, Sie koennen also keinen Zug auslassen.


Schritt 4 - das Ende, oder auch nicht

Das Spiel endet erst, wenn kein Zug mehr etwas veraendern wuerde: kein leeres Feld und kein gleiches Paar direkt daneben. Druecken Sie die Leertaste oder tippen Sie auf das Feld, um neu zu starten. Erreichen Sie den Stein mit der 2048, wechselt die Statuszeile zu einer Weiterspielen-Meldung - das Spiel geht fuer eine hoehere Punktzahl weiter. Es gibt kein Rueckgaengig.


Schritt 5 - Vollbild und Ihre Bestpunktzahl

Die Schaltflaeche Vollbild bringt das Feld auf Ihren ganzen Bildschirm, die Punktzeile bleibt sichtbar; Esc bringt die Seite zurueck. Ihre Bestpunktzahl wird im lokalen Speicher dieses Browsers gespeichert, sie bleibt also nach einem Neuladen erhalten - das Loeschen der Websitedaten loescht sie ebenfalls.
