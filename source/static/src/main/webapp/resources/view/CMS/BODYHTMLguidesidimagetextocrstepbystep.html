Image To Text (ocr) Langkah Demi Langkah

Last reviewed: 2026-07-16

Image to Text (OCR) berjalan dalam tiga langkah: buka alat, pilih gambar, lalu salin atau unduh teks yang dikenali. Halaman ini menjelaskan apa yang diharapkan di setiap langkah.


Langkah 1 - buka alat

Buka Image to Text (OCR). Tidak perlu memasang apa pun atau masuk akun sebelum mulai.


Langkah 2 - pilih gambar dan klik Extract text

Pilih file gambar dengan pemilih file - gambar terbuka di browser dan tidak pernah diunggah ke server - lalu klik Extract text. Proses pertama di perangkat ini mengunduh mesin OCR di latar belakang; proses berikutnya di perangkat yang sama dimulai lebih cepat.


Langkah 3 - salin atau unduh teks hasil pengenalan

Teks hasil pengenalan muncul di kotak baca-saja setelah pemindaian selesai. Salin dengan satu klik atau unduh sebagai file .txt. OCR tidak sempurna, jadi periksa kembali nama, angka, dan tanda baca sebelum mengandalkan hasilnya.
