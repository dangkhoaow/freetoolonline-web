Color Contrast Checker (WCAG) Kapan Dipakai

Last reviewed: 2026-07-20

Buka Color Contrast Checker (WCAG) saat Anda butuh cek kontras cepat antara dua warna tanpa memasang suite aksesibilitas atau mengunggah file.


Kapan cocok

Cek label tombol terhadap isian sebelum ship CSS. Bandingkan teks body dengan latar palette merek. Pastikan judul besar masih lulus AA dengan latar lembut. Simpan HEX di perangkat ini karena hitungan berjalan di tab.


Yang diharapkan

Pilih warna foreground dan background; rasio serta hasil AA / AAA diperbarui saat Anda mengubahnya. Gunakan preview untuk membaca teks normal dan besar. Semua di browser - tanpa upload dan tanpa akun. Rasio memakai rumus luminansi relatif W3C (WCAG 2.1).

Langkah tepat: panduan langkah demi langkah.
