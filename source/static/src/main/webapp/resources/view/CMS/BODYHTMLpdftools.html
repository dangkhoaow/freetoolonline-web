Split by range vs split to each page

Split by range keeps a section together (pages 3-7). Split to each page creates one file per page - useful for uploading forms or sharing single pages individually.

A note on accessibility and searchable text

A PDF that looks fine to a human can still be unreadable to screen readers and search. The difference usually comes down to whether the file has a real text layer or is only a picture of text.

  • Born-digital PDFs (exported from Word, Pages, or a design tool) carry a text layer by default. Our PDF to Text tool can extract that text directly and preserves paragraph order.
  • Scanned PDFs are photographs of pages. They need OCR before anything can extract the words. If PDF to Text returns a blank result, that is usually the reason.
  • Tagged PDFs carry extra structure (headings, lists, reading order) for screen readers. Tagging is preserved when you merge, split, or compress, but can be lost when you convert to images.

When you need a form or contract to stay legally accessible, keep the original export rather than a re-scanned copy, and prefer Compose PDF for edits so the text layer survives.

Page-size catalog shared by Preflight and Images to PDF

Two tools on this hub - Preflight PDF and Images to PDF - share the same trim-size dropdown so a document can be authored at one size and then validated against that same size without converting between catalogs. The supported sizes group into four print families:

  • Paper. A4 (the international office standard for letters, reports, and forms) and 8.5x11 Magazines (the US letter equivalent, also used for short-run magazines).
  • Square photo. 9x9, 7x7, and 5x5 - common for square photo prints, social-media album exports, and small gift cards where the layout is symmetrical.
  • Trade book. 6x9 and 5x8 - the two trim sizes most book printers list as defaults for paperback novels, manuals, and self-published trade titles.
  • Photo book. 12x12, 13x11, 10x8, 8x10, and 8.25x10.25 - rectangular and large-square trims used by photo-book printers; 12x12 is the most common wedding-album trim, 8x10 is the standard photo-print rectangle.

Open Images to PDF when the document does not yet exist - the converter writes each image to the chosen trim. Open Preflight when the document already exists and you need a pass/fail report against the chosen trim. Both tools route the file through the same short-lived HTTPS processing service, so the page-size selection on one tool is interchangeable with the same selection on the other.

How processing works across the hub

Every transform on this hub - Split, Protect, Flatten, Compose, Join, PDF to Text, PDF to HTML, and Images to PDF - routes through a short-lived HTTPS processing service. The service receives the upload, runs the operation, streams the result back, and releases the working copy. The file is not retained and no account is required. Lightweight checks (file size, page count) stay local in the browser. If a specific document cannot leave your device, use a desktop tool for that file; for everything else, the service handles the transform in seconds with no install.

Merge PDFs or extract their text - free, online

Merge PDFs. The Join tool combines as many PDFs as you upload into one document over a short-lived HTTPS processing service, with no account or install. Drop the source files in the order you want them stacked, press Join, and the merged PDF downloads back to your device once the merge completes - no watermark, no signup, no email required.

Convert PDF to plain text. The To Text tool extracts every readable string from a PDF and hands it back as a plain `.txt` file - useful when you need the words but not the layout (paste into a doc, search a contract, feed a spreadsheet column). Scanned PDFs whose pages are image bitmaps will pass through with no extracted text; for those, run the PDF through an OCR tool first because To Text only reads characters the PDF already exposes as text.

Split a PDF online - free, no install

Two splitters live here for two different reader tasks. Split PDF By Range pulls a contiguous section - pages 3 to 7, or just the appendix - out of a longer document and hands back a single smaller PDF you can email or upload. Split PDF To Each Pages goes the other way: every page leaves as its own file, which is what you want for per-page signatures or single-page form uploads. Both run over a short-lived HTTPS processing service with no account, no install, and no watermark.

PDF to Text, PDF to Images, and Flatten PDF - what each does

PDF to Text extracts the text layer from a born-digital PDF and hands it back as a plain .txt file - useful when you need the words without the layout. Scanned PDFs return blank; for those, run OCR first. PDF to Images converts each page to a PNG or JPG - use it when the destination is a slide deck or image gallery, not a printable document. Flatten PDF bakes form fields and annotations into the page so an archive copy renders identically on every viewer.
