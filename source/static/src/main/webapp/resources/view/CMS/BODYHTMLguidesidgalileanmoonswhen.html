Kapan memakai Galilean Moons 3D Explorer

Pakai Galilean Moons 3D Explorer untuk melihat empat bulan besar Jupiter di browser - dan lewati jika Anda perlu ephemeris semua bulan kecil.

  • Demo kelas kunci periode 4:2:1 Io-Europa-Ganymede
  • Penyegaran cepat sebelum mengamati Jupiter
  • Membandingkan ukuran (Ganymede lebih besar dari Merkurius) di browser

Halaman ini tidak menggantikan ephemeris planetarium, tidak mendaftar puluhan bulan kecil, dan tidak menyelesaikan gravitasi n-benda.
