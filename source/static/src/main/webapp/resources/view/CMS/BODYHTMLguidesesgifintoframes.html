La eleccion del formato de salida tiene consecuencias reales para los frames guardados. PNG mantiene cada pixel exacto, preserva transparencia, y es la eleccion mas segura cuando el frame se re-editara o compondra despues. JPG descarta el canal alfa y aplica compresion con perdidas, asi que el archivo es mas pequeno y mas amigable para una publicacion de feed o miniatura, al costo de fidelidad despues de editar.
