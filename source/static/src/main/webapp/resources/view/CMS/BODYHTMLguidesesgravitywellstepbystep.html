Como Usar Gravity Well 3D Explorer - Paso a Paso

Last reviewed: 2026-07-14

Empieza en Tierra (11.2 km/s), cambia a Luna / Jupiter / Sol para ver como se profundiza la hoja, y luego activa Show r_s (Sun) para leer el dato Schwarzschild de ~2.95 km sin salir del navegador.

  1. Abre Gravity Well 3D Explorer y espera a que se dibuje la hoja.
  2. Arrastra para orbitar, desplaza o pellizca para hacer zoom, y manten el panel de datos visible en pantallas anchas.
  3. Toca Luna 2.38, Tierra 11.2, Jupiter 59.5 y Sol 617.5 para comparar la profundidad del pozo.
  4. Opcional: Show r_s (Sun) y Exaggerated depth - recuerda que el tamano en pantalla es escala didactica.
  5. Lee los valores de la tabla en voz alta para que las velocidades de escape sigan siendo lo importante.
