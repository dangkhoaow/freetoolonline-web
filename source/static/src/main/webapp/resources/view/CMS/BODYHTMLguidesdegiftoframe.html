GIF zu Frame

GIF animiert hochladen und das Tool extrahiert jeden Frame als PNG oder JPG. PNG behaelt die urspruengliche Palettentransparenz; JPG verwirft sie, erzeugt aber kleinere Dateien. Nuetzlich wenn die Frames in einen Bildeditor, eine Sprite-Tabelle oder als einzelne statische Bilder verwendet werden sollen.

Das Tool dekodiert das GIF auf dem Server und gibt jeden Animationsframe als eigene Bilddatei zurueck. Dateien werden nach einem kurzen Download-Fenster geloescht. Die Seite unter /extract-gif-to-image-frames.html uebernimmt den Upload und die Extraktion.

Das richtige Ausgabeformat waehlen

PNG ist die richtige Wahl wenn das GIF Palettentransparenz verwendet - die extrahierten PNG-Frames tragen diesen Transparenzkanal. JPG ist die richtige Wahl wenn die Dateigroesse wichtiger ist als die Transparenzgenauigkeit: jeder JPG-Frame ist deutlich kleiner.

Was mit den extrahierten Frames tun

Der haeufigste Workflow ist das Bearbeiten eines Frames: Set extrahieren, den spezifischen Frame in einem Bildeditor oeffnen, Aenderung vornehmen und wieder zusammenfuehren. Ein zweiter Workflow ist das Erstellen einer Sprite-Tabelle. Ein dritter ist die Auswahl eines repraesentativen Frames als statisches Fallback-Bild fuer Kontexte wo ein schleifendes GIF Bandbreite verschwenden wuerde.
