CSS Box-Shadow Generator - Langkah Demi Langkah

Last reviewed: 2026-07-24

CSS Box-Shadow Generator berjalan dalam tiga langkah: buka alat, geser slider, lalu salin hasilnya. Halaman ini merinci apa yang dilakukan setiap langkah.


Langkah 1 - buka alat

Buka CSS Box-Shadow Generator. Kotak pratinjau dan bayangan awal langsung muncul - tidak perlu instalasi atau login lebih dulu.


Langkah 2 - bentuk bayangan

Geser offset horizontal dan vertikal untuk memindahkan bayangan, radius blur untuk melembutkan tepinya, dan radius spread untuk membesarkan atau mengecilkan bayangan. Pilih warna dan opacity untuk bayangan, dan centang Inset jika ingin bayangan dalam bukan bayangan jatuh. Pratinjau memperbarui setiap kali ada perubahan.


Langkah 3 - salin hasilnya

Klik Copy CSS untuk menyalin baris box-shadow yang dihasilkan ke clipboard, atau pilih teks di kotak hasil secara manual. Semuanya terjadi di tab browser Anda; tidak ada yang diunggah.
