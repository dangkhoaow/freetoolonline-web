Schritte

  1. Waehlen Sie eine Kategorie - Laenge, Gewicht, Temperatur und sieben weitere.
  2. Waehlen Sie die Ausgangseinheit und die Zieleinheit.
  3. Geben Sie eine Zahl in das Wertfeld ein; das Ergebnis erscheint waehrend der Eingabe.
  4. Nutzen Sie die Schaltflaeche Tauschen, um Ausgangs- und Zieleinheit zu vertauschen.

Wie die Umrechnungen funktionieren

Die meisten Kategorien nutzen Standard-Umrechnungsfaktoren: der Wert wird in die Basiseinheit der Kategorie und dann in die gewaehlte Einheit umgerechnet. Temperatur ist anders, weil sie kein einfaches Verhaeltnis ist, daher nutzen Celsius, Fahrenheit und Kelvin Offset-Formeln, die das Werkzeug auf dem Bildschirm zeigt. Ihre letzten Umrechnungen werden in diesem Browser gespeichert, damit Sie sie nachschlagen koennen, und Sie koennen sie jederzeit loeschen.
