Cara main Particle Clicker - panduan langkah demi langkah

Halaman Particle Clicker memuat clicker fisika CERN Webfest (~2 MB) di iframe.

Last reviewed: 2026-07-21

Tekan Play, klik detector untuk data, lalu riset, rekrut, dan beli upgrade.

Langkah 1 - tekan Play

Tekan Play untuk memuat iframe CDN (~2 MB). UI lab terbuka dengan canvas detector.

Langkah 2 - klik detector

Klik atau ketuk detector untuk membuat event dan mengumpulkan data. Data adalah mata uang riset utama.

Langkah 3 - riset penemuan

Buka Research dan belanjakan data untuk menaikkan level penemuan. Level baru menaikkan reputasi.

Langkah 4 - rekrut dan upgrade

Pakai HR untuk merekrut worker data otomatis. Belanjakan funding untuk upgrade dan simpan dari menu.
