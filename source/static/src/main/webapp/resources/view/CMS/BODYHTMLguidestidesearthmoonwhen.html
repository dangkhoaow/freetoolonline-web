When to use Tides: Earth-Moon 3D Explorer

Use the Tides: Earth-Moon 3D Explorer for a quick browser view of why coasts get two tides a day and how Spring tides differ from Neap tides - skip it when you need a real hydrodynamic tide-prediction model for a specific harbor.

  • Classroom demos of the two-bulge equilibrium tide model
  • Comparing Spring tide (about 1.5x) vs Neap tide (about 0.5x) amplitude
  • One-screen refresher before reading the tidal-locking or moon-phases-3d siblings

This page does not replace a real tide table for a specific port, does not model harbor resonance or storm surge, and does not simulate ocean currents.
