Morse Code Translator - Paso a Paso

Last reviewed: 2026-07-21

Abre Morse Code Translator, pega texto o Morse a la izquierda, haz clic en convertir y copia el resultado a la derecha.


Pasos

  1. Pega texto plano (codificar) o puntos y rayas (decodificar) en el cuadro.
  2. Haz clic en el botón rojo de convertir.
  3. Lee la salida: las letras se vuelven códigos Morse; las palabras usan barra; el Morse se vuelve texto.
  4. Haz clic en Copy Output y pégalo donde lo necesites.

Privacidad

La traducción se ejecuta en el navegador. Tu texto no se sube.
