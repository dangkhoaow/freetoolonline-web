Egal was Sie waehlen, der Digest aktualisiert sich live unter dem Formular und Sie kopieren die resultierende Hex-Zeichenkette mit einem Klick - kein Download-Schritt, keine Datei zu verwalten.

Das Hashen laeuft komplett clientseitig ueber die WebCrypto-API des Browsers - es wird nichts an einen Server gesendet. MD5 wird hier nicht angeboten, da moderne Browser es als unsicher weglassen; nutzen Sie das verwandte md5-converter-Tool fuer diesen speziellen Algorithmus.
