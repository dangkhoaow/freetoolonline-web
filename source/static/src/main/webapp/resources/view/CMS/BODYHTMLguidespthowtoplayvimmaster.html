Como jogar Vim Master - passo a passo

The Vim Master page loads a ~584 KB Vim lesson game in an iframe.

Last reviewed: 2026-07-21

Press Play, Start Learning, then type Vim keys to clear each lesson objective.

Step 1 - press Play

Press Play to inject the iframe (~584 KB). The lesson UI boots inside the frame.

Step 2 - start learning

Click Start Learning to dismiss onboarding (#start-learning-btn). The editor (#editor-container) becomes active.

Step 3 - clear lessons

Follow the instructions line. Type real Vim motions and operators until the win condition passes, then advance.

Step 4 - save and drill

XP and badges auto-save under ftol:vimmaster:* keys. Optional: export / import progress, cheat-mode practice, and timed challenges.
