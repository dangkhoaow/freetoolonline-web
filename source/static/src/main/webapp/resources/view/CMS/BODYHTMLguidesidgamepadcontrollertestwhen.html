Gamepad / Controller Test Kapan Digunakan

Last reviewed: 2026-07-21

Buka Gamepad / Controller Test saat Anda perlu memastikan stik USB atau Bluetooth melaporkan tombol dan sumbu - tanpa memasang utilitas desktop.


Kapan cocok

Stik baru terasa mati di satu tombol. Stick drift dan ingin melihat nilai sumbu. Lebih suka uji di tab tanpa akun.


Apa yang diharapkan

Aktifkan polling, tekan tombol agar browser mengekspos pad, lihat indikator dan sumbu. Reset membersihkan sesi. Tidak ada yang diunggah.

Langkah tepat: panduan langkah demi langkah.
