Add Watermark to Image - Passo a Passo

Last reviewed: 2026-07-22

Add Watermark to Image funciona em três etapas: abrir a ferramenta, informar a entrada e baixar o resultado. Esta página detalha o que cada etapa espera.


Passo 1 - abra a ferramenta

Acesse Add Watermark to Image. Não é preciso instalar nada nem fazer login antes de começar.


Passo 2 - informe a entrada

Escolha uma das oito posições fixas ou distribua o texto na diagonal por toda a imagem para uma proteção de cópia mais forte.


Passo 3 - obtenha o resultado

Tudo roda localmente via canvas - a foto nunca é enviada a um servidor.

Aplicar marca de água em várias imagens de uma vez e usar uma imagem de logotipo como marca de água não estão disponíveis aqui - apenas texto, uma imagem por vez.
