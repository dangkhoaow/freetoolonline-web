Sprache zu Text vs Alternativen

Sprache zu Text ist eine von drei gaengigen Moeglichkeiten zur Sprachtranskription: ein browserbasiertes Werkzeug wie dieses, eine Desktop-Anwendung oder ein Online-Dienst mit Upload. Jede bietet unterschiedliche Kompromisse bei Datenschutz und Installation.

Haeufige Fragen

Warum ist es auf Chrome und Safari beschraenkt?

Die Web Speech Recognition API ist in Firefox nicht verfuegbar. Chrome, Edge (Chromium) und Safari unterstuetzen die API nativ.

Kann ich es zum Transkribieren vorab aufgezeichneter Audiodateien verwenden?

Nein. Dieses Werkzeug nimmt Live-Audio vom Mikrofon auf. Fuer vorab aufgezeichnete Dateien waere ein spezialisierter Transkriptionsdienst geeigneter.
