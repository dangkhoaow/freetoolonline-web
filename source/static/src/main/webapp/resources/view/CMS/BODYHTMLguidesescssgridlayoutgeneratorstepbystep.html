Generador de Diseño CSS Grid - Paso a Paso

Last reviewed: 2026-07-23

El Generador de Diseño CSS Grid crea un bloque CSS display: grid listo para pegar en tres pasos: definir el numero de columnas y filas, definir los espacios, y copiar el resultado. No se sube nada y no se instala nada.


Paso 1 - define columnas y filas

Abre el Generador de Diseño CSS Grid. Arrastra el control deslizante Columns (1 a 12) y el control Rows (1 a 6) para que coincidan con el diseño que quieres - la cuadricula de vista previa a la derecha redibuja las celdas de inmediato.


Paso 2 - define los espacios

Arrastra los controles Column gap y Row gap (0px a 50px cada uno) para controlar el espacio entre celdas. La cuadricula de vista previa y el texto CSS generado debajo se actualizan en vivo con los valores exactos en pixeles.


Paso 3 - copia el resultado

Haz clic en "Copy CSS" para copiar el bloque generado al portapapeles. Si el navegador bloquea el acceso al portapapeles, un mensaje de estado te pide que selecciones y copies el texto manualmente. Toda la pagina funciona en el navegador, asi que ningun valor de los controles deslizantes se envia a un servidor.
