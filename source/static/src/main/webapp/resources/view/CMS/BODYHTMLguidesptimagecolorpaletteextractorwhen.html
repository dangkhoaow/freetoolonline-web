Quando extrair uma paleta de cores de uma foto

Use este extrator quando tem um JPEG, PNG ou WebP local e precisa das cores HEX dominantes para um design system, mood board ou tema CSS.

Escolha o ficheiro, defina quantas cores quer e clique Extract Palette. A amostragem fica no browser - a imagem nunca e enviada.

Se so precisa de escolher um tom a mao, use o Color Picker. Para legibilidade texto-sobre-fundo, use o WCAG Contrast Checker.
