Quando usar o Algol Eclipsing Binary 3D Explorer - Melhores momentos e limites

Última revisão: 2026-07-22

Abra o Algol Eclipsing Binary 3D Explorer quando precisar de uma lição sobre binárias eclipsantes e curva de luz com dados publicados de magnitude, período, massa e raio - sem um laboratório de fotometria.


Uma lição sobre eclipses e curva de luz

A lição começa quando você move o controle de fase por 0% (eclipse primário profundo) e 50% (eclipse secundário raso), ou pressiona Play orbit para ver os dois automaticamente enquanto a curva de luz plota a queda de brilho e o painel mantém os dados publicados.


Sessões que este explorador não atende

Este explorador não é indicado para lições gerais de dinâmica orbital ou baricentro sem eclipse. Use o Binary Star System 3D Explorer para isso. Siga os controles no guia passo a passo.
