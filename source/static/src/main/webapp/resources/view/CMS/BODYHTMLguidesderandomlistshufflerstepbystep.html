Zufaelliger Listen-Mischer: Schritt-fuer-Schritt-Anleitung

Das Tool Zufaelliger Listen-Mischer bringt eine eingefuegte Liste direkt im Browser in eine faire, zufaellige Reihenfolge. Diese Anleitung zeigt, wie Sie die Liste einfuegen, was beim Klick auf Shuffle passiert, und warum die neue Reihenfolge keinen Eintrag bevorzugt.

Last reviewed: 2026-07-02

freetoolonline.com Editorial Team

Antwort in 30 Sekunden. Oeffnen Sie den Zufaelligen Listen-Mischer, ersetzen Sie den Beispieltext durch je einen Eintrag pro Zeile, und klicken Sie auf Shuffle - das Feld darunter ordnet sich sofort neu. Klicken Sie jederzeit erneut auf Shuffle fuer eine neue Reihenfolge.

Ihre Liste einfuegen

Das Eingabefeld akzeptiert einen Eintrag pro Zeile - Namen, Aufgaben, Loszettel, alles, was eine zufaellige Reihenfolge braucht. Es startet mit vier Beispielnamen, damit die Seite sofort ein funktionierendes Ergebnis zeigt; leeren Sie das Feld und fuegen Sie Ihre eigene Liste ein, um sie zu ersetzen. Leere Zeilen werden ignoriert, sodass eine mit zusaetzlichen Leerzeichen eingefuegte Liste trotzdem saubet gemischt wird.

Was beim Klick auf Shuffle passiert

Ein Klick auf Shuffle fuehrt eine Fisher-Yates-Mischung der aktuellen Liste aus und gibt die neue Reihenfolge als nummerierte Liste darunter aus. Beim Laden der Seite laeuft ausserdem automatisch eine Mischung, sodass immer ein Ergebnis zu sehen ist, bevor Sie ueberhaupt etwas eintippen. Jeder Klick erzeugt eine unabhaengige neue Reihenfolge - das Tool speichert keine fruehere Ergebnisse.

Warum die Reihenfolge nicht verzerrt ist

Die Mischung tauscht jede Position mit einer zufaellig gewaehlten vorherigen Position, indem die Liste einmal von hinten nach vorn durchlaufen wird - die Standardmethode Fisher-Yates. Jeder Tausch nutzt crypto.getRandomValues des Browsers (mit Rueckfall auf Math.random nur falls diese API nicht verfuegbar ist), dieselbe Fairness-Quelle, die auch der Wuerfel und der Muenzwurf dieser Seite nutzen, sodass kein Eintrag wegen seiner Laenge oder seiner Startposition in der Liste eine hoehere Chance auf eine bestimmte Position hat.

Datenschutz

Die Mischung laeuft vollstaendig in Ihrem Browser-Tab. Die eingefuegte Liste wird nirgendwo hochgeladen, es ist kein Konto erforderlich, und die Seite macht waehrend des Mischens keinen Serveraufruf.

Haeufig gestellte Fragen

Bevorzugt die Mischung Eintraege am Anfang oder Ende meiner Liste?

Nein. Fisher-Yates gibt jeder Position eine gleiche Chance, unabhaengig davon, wo der Eintrag begann, und der Tauschindex stammt aus crypto.getRandomValues, also gibt es keine Neigung zum Anfang, Ende oder einer anderen Position.

Wird meine Liste gespeichert, wenn ich spaeter wiederkomme?

Nein. Die Liste existiert nur waehrend dieses Seitenbesuchs - ein Neuladen oder Verlassen setzt sie auf die vier Beispielnamen zurueck, und nichts wird auf einem Server gespeichert.

Kann ich dieselbe Liste mehrmals mischen?

Ja. Klicken Sie so oft wie gewuenscht auf Shuffle; jeder Klick erzeugt eine unabhaengige neue Reihenfolge, ohne dass Sie etwas neu eintippen muessen.

Was passiert, wenn meine Liste nur einen Eintrag hat?

Die Liste wird unveraendert erneut ausgegeben - mit nur einem Eintrag gibt es nichts neu zu ordnen, und es wird kein Fehler angezeigt.
