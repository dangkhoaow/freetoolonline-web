Ferramenta que implementa/zip-file.html

A eficacia da compressao depende do que esta dentro do arquivo. Entradas de texto, codigo, JSON e CSV normalmente encolhem 60-80%; documentos do Office 10-25%; PDFs 5-15%; fotos ja comprimidas (JPG, PNG, HEIC) apenas 0-5% - para essas, comprima primeiro com uma ferramenta de imagem e depois compacte. A tabela "Reduce ZIP file size - what to expect" na pagina da ferramenta mostra as contas para atingir alvos comuns como o limite de 25 MB para anexo de e-mail. Para conteudo sensivel, abra o painel Settings no criador e adicione uma senha antes de compactar - tres niveis de criptografia estao disponiveis (Standard, AES-128, AES-256). Envie a senha por um canal diferente do arquivo.
