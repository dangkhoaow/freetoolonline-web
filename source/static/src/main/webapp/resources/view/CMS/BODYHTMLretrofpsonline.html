Two full campaigns of Freedoom: Campaign 1 with four episodes (36 levels), Campaign 2 with 32 levels in one long run - each a one-click start card that states its one-time download before anything is fetched.

Campaign 1 - four episodes
The first Freedoom campaign: 36 levels across four episodes. One-time download (exact size shown once it starts); cached in this browser afterwards, so later visits work offline.
Campaign 2 - one long haul
The second Freedoom campaign: 32 levels in a single continuous run with a wider bestiary and the double-barrelled shotgun. Same one-time download and offline caching.
