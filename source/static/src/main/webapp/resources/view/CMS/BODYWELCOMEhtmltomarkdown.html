HTML to Markdown Converter

Last reviewed: 2026-07-20

Paste HTML and get Markdown you can copy - entirely in the browser, no upload and no account.

Typical use: when a README, wiki, or CMS field accepts Markdown but your source is HTML from email, a docs export, or a rich-text editor.

Turndown runs on this page after the vendor script loads from this origin. Clear resets both boxes. Closing the tab discards the text - nothing is stored on a server.

Need the reverse direction? Use the related Markdown to HTML previewer linked from the related-tools section.
