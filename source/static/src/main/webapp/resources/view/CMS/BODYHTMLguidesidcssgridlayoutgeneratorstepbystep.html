Generator Tata Letak CSS Grid - Langkah demi Langkah

Last reviewed: 2026-07-23

Generator Tata Letak CSS Grid membuat blok CSS display: grid yang siap ditempel dalam tiga langkah: atur jumlah kolom dan baris, atur jarak antar sel, lalu salin hasilnya. Tidak ada yang diunggah dan tidak perlu instalasi.


Langkah 1 - atur kolom dan baris

Buka Generator Tata Letak CSS Grid. Geser penggeser Columns (1 sampai 12) dan penggeser Rows (1 sampai 6) sesuai tata letak yang Anda inginkan - grid pratinjau di sebelah kanan langsung menggambar ulang sel-selnya.


Langkah 2 - atur jarak antar sel

Geser penggeser Column gap dan Row gap (masing-masing 0px sampai 50px) untuk mengatur jarak antar sel. Grid pratinjau dan teks CSS yang dihasilkan di bawahnya diperbarui secara langsung dengan nilai piksel yang tepat.


Langkah 3 - salin hasilnya

Klik "Copy CSS" untuk menyalin blok yang dihasilkan ke clipboard. Jika browser memblokir akses clipboard, pesan status akan meminta Anda memilih dan menyalin teks secara manual. Seluruh halaman berjalan di browser, jadi tidak ada nilai penggeser yang dikirim ke server.
