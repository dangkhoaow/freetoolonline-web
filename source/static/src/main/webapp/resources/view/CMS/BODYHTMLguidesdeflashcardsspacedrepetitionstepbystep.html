Karteikarten (Spaced Repetition) Schritt Fuer Schritt

Last reviewed: 2026-07-13

Karteikarten (Spaced Repetition) laeuft in drei Schritten ab: das Tool oeffnen, deine Karten hinzufuegen, dann lernen. Diese Seite erklaert, was jeder Schritt erwartet.


Schritt 1 - das Tool oeffnen

Gehe zu Karteikarten (Spaced Repetition). Vor dem Start muss nichts installiert oder eingeloggt werden.


Schritt 2 - deine Karten hinzufuegen

Tippe die Frage in "Front (question)", tippe die Antwort in "Back (answer)", dann klicke auf "Add card" (oder druecke Enter in einem der beiden Felder). Wiederhole das fuer jede Karte, die du im Stapel haben willst.


Schritt 3 - den Stapel lernen

Der Lernbereich zeigt jeweils die Vorderseite einer Karte. Klicke auf "Show answer", um sie umzudrehen, markiere sie dann mit "I knew it", um sie fuer diesen Durchgang zur Seite zu legen, oder mit "Still learning", um sie in der Warteschlange zu behalten; Shuffle mischt die Lernreihenfolge.

Karten werden in diesem Browser ueber lokalen Speicher gesichert, sodass sie ein Neuladen der Seite ueberstehen; exportiere ein Backup (JSON), bevor du Browserdaten loeschst oder das Geraet wechselst, da keines der beiden automatisch synchronisiert wird.

Das ist ein einfaches Lernhilfsmittel, kein Spaced-Repetition-Planer - es gibt keinen Algorithmus, der festlegt, wann eine Karte zurueckkommt; "Study all again" loescht nur die "bekannt"-Markierung jeder Karte, sodass der komplette Stapel wieder in die Warteschlange kommt.
