Namens-Gluecksrad Online: Schritt-fuer-Schritt-Anleitung

Das Tool Wheel Spinner Online verwandelt eine Liste von Namen oder Optionen in eine faire Zufallsauswahl. Diese Anleitung zeigt, wie Sie die Liste aufbauen, was waehrend der Drehung passiert, wie der Gewinner bestimmt wird, und wie es sich mit einem Wuerfelwurf, einem Muenzwurf oder einer manuellen Auslosung vergleicht.

Antwort in 30 Sekunden. Oeffnen Sie das Tool Wheel Spinner Online, geben Sie einen Namen oder eine Option pro Zeile in das Listenfeld ein, und klicken Sie dann auf Spin. Das Rad teilt sich in ein gleich grosses Segment pro Eintrag, dreht sich einige Sekunden lang und stoppt mit einem Gewinner, der unter der Anzeige genannt wird - bearbeiten Sie die Liste und drehen Sie erneut fuer eine weitere Runde.

Die Liste erstellen

Das Listenfeld nimmt einen Eintrag pro Zeile - Namen, Aufgaben, Optionen, alles, was eine zufaellige Auswahl braucht. Das Rad startet mit sechs bereits eingetragenen Beispielnamen, damit Sie sofort ein vollstaendiges Rad sehen; loeschen Sie diese und geben Sie Ihre eigene Liste ein, um sie zu ersetzen. Das Rad zeichnet sich live neu, waehrend Sie tippen, sodass Sie die Anzahl der Segmente vor dem Drehen sich aendern sehen.

Was waehrend einer Drehung passiert

Ein Klick auf Spin braucht mindestens zwei Eintraege in der Liste - bei weniger zeigt das Tool "Add at least two entries." an und dreht sich nicht. Mit zwei oder mehr Eintraegen beschleunigt das Rad ueber mehrere volle Umdrehungen plus eine zufaellige zusaetzliche Strecke und wird ueber einige Sekunden langsamer, bis es stoppt. Das Segment, das unter dem festen Zeiger oben landet, ist der Gewinner, und dessen Name wird direkt unter dem Rad angezeigt, sobald es stoppt.

Warum jeder Eintrag die gleichen Chancen hat

Das Rad teilt den vollen Kreis in genauso viele gleiche Segmente, wie es Eintraege gibt, sodass eine Liste mit sechs Namen sechs gleiche Segmente erhaelt und eine Liste mit drei Namen drei groessere, aber gleiche Segmente - die Segmentgroesse haengt nie von der Textlaenge eines Eintrags oder seiner Position in der Liste ab. Das Gewinnersegment wird mit dem im Browser integrierten crypto.getRandomValues ausgewaehlt, derselben Fairness-Quelle, die auch fuer Wuerfelwuerfe und Muenzwuerfe an anderer Stelle dieser Website verwendet wird, sodass keine Position in der Liste bevorzugt wird.

Namens-Gluecksrad im Vergleich zu Wuerfelwurf, Muenzwurf, oder manueller Auslosung

Ein Wuerfelwurf oder Muenzwurf ist fair, bietet aber nur eine kleine, feste Menge an Ergebnissen - sechs Seiten oder zwei Seiten. Das Namens-Gluecksrad akzeptiert eine beliebige Anzahl benannter Eintraege, sodass es fuer Situationen passt, die ein Wuerfel oder eine Muenze nicht abdecken: einen Namen aus zehn Teammitgliedern auszuwaehlen, oder einen Preisgewinner aus dreissig Verlosungseintraegen. Im Vergleich zu einer manuellen Auslosung - Namen auf Papier schreiben und einen herausziehen - entfernt das Rad jedes Risiko, dass die ziehende Person das Ergebnis beeinflusst, da die Drehstrecke vom Browser erzeugt wird und nicht von einer Person gewaehlt wird.

Haeufige Anwendungen

Menschen nutzen das Namens-Gluecksrad fuer Verlosungen und Gewinnspiele, um zu entscheiden, wer bei einem Spiel oder Meeting zuerst dran ist, um einem Teammitglied eine zufaellige Aufgabe zuzuweisen, oder um einen zufaelligen Gewinner aus einer Kurzliste auszuwaehlen, ohne dass jemand das Ergebnis durch die Art beeinflussen kann, wie die Liste eingegeben wurde.

Was dieses Tool nicht tut

Es speichert die Liste oder den Gewinner nicht zwischen Besuchen - ein Neuladen der Seite setzt die Liste auf die sechs Beispielnamen zurueck. Es gewichtet Eintraege nicht nach Textlaenge, Position, oder irgendeinem anderen Faktor als der Anzahl - jeder Eintrag erhaelt genau ein gleiches Segment. Es erfordert kein Konto, und es sendet die Liste nirgendwohin; die Drehung findet vollstaendig auf Ihrem Geraet statt.

Datenschutz

Das Namens-Gluecksrad laeuft vollstaendig in Ihrem Browser-Tab. Die Namen oder Optionen, die Sie eingeben, werden nie irgendwohin hochgeladen, es ist kein Konto erforderlich, und die Seite fuehrt waehrend der Drehung keine Serverkommunikation durch.

Haeufig gestellte Fragen

Aendert die Reihenfolge, in der ich Namen eingebe, ihre Chancen?

Nein. Jeder Eintrag erhaelt ein gleich grosses Segment, unabhaengig davon, wo er in der Liste steht, und das Gewinnersegment wird mit dem crypto.getRandomValues des Browsers ausgewaehlt, sodass die Listenreihenfolge das Ergebnis nicht beeinflusst.

Was passiert, wenn ich nur einen Namen habe?

Ein Klick auf Spin mit null oder einem Eintrag zeigt "Add at least two entries." an, und das Rad dreht sich nicht - Sie brauchen mindestens zwei Eintraege fuer eine sinnvolle Auswahl.

Wird die Liste gespeichert, wenn ich spaeter zurueckkomme?

Nein. Die Liste existiert nur waehrend dieses Besuchs auf der Seite - ein Neuladen oder Verlassen setzt sie auf die sechs Beispielnamen zurueck, und nichts wird auf einem Server gespeichert.

Kann ich das fuer mehr als Verlosungen nutzen?

Ja. Alles, was eine faire Zufallsauswahl aus einer Liste braucht, funktioniert - Aufgaben zuweisen, die Reihenfolge festlegen, oder einen Gewinner aus einer Kurzliste auswaehlen.
