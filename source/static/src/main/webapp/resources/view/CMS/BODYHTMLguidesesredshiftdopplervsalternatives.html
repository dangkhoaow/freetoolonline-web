Redshift and Doppler Shift 3D Explorer vs Alternativas

Aprender z = v/c toma una carga de pagina en Redshift and Doppler Shift 3D Explorer - 0 MB instalados, USD 0, sin cuenta. Milky Way Map 3D Explorer ensena estructura galactica, no medicion de lineas espectrales. Parallax Distance 3D Explorer ensena distancia a partir de geometria, no de un desplazamiento de longitud de onda.


vs un diagrama estatico de libro

Un libro puede imprimir un diagrama de onda a una sola velocidad. Redshift and Doppler Shift 3D Explorer agrega preconfiguraciones de Acercandose, Alejandose y Ley de Hubble que recalculan el desplazamiento en vivo.


vs software de planetario de escritorio

El software de planetario de escritorio necesita instalacion y a menudo una licencia de pago para modulos de espectroscopia; esta pagina no necesita ninguno de los dos.
