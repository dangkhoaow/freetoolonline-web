Lagrange Points 3D Explorer Langkah demi Langkah

Last reviewed: 2026-07-14

Lagrange Points 3D Explorer mulai begitu halaman termuat: seret untuk mengorbit, gulir untuk memperbesar, dan tekan Play orbit. Semua dirender di perangkat Anda sendiri - tanpa instalasi, tanpa akun.


Langkah 1 - titik kolinear

Tekan L1 dan L2 agar panel menunjukkan sekitar 1.5 juta km dari Bumi pada garis Matahari-Bumi.


Langkah 2 - titik segitiga

Lompat ke L4 dan L5 untuk membaca geometri segitiga sama sisi 60 derajat di depan dan di belakang.


Langkah 3 - penanda wahana

Tekan SOHO dan JWST, lalu layar penuh agar kontrol tetap terlihat saat kerangka korotasi berputar.
