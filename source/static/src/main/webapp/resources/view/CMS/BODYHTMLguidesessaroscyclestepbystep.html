Cómo Explorar el Saros Cycle Eclipse Explorer 3D

Última revisión: 2026-07-23

Revisado por Equipo editorial de FreetoolOnline

Abre el Saros Cycle Eclipse Explorer 3D, avanza por la serie con el control deslizante o pulsa Play stepping, y observa cómo la trayectoria de cada eclipse se desplaza unos 120 grados al oeste respecto a la anterior.

  1. Abre /space-3d/saros-cycle.html y espera a que se renderice la Tierra.
  2. Arrastra el control de índice de eclipse de 0 a 1 y observa la trayectoria brillante desplazarse unos 120 grados al oeste de la trayectoria inicial atenuada.
  3. Pulsa "Show all tracks so far" para ver todos los pasos de la serie a la vez, avanzando juntos hacia el oeste.
  4. Pulsa "Jump 3 steps (exeligmos)" para ver la trayectoria regresar cerca de la longitud inicial tras tres pasos.
  5. Pulsa "Play stepping" para recorrer automáticamente toda la serie.
  6. Lee el panel de datos para el tiempo transcurrido, el desplazamiento acumulado hacia el oeste y cuánto dura una serie completa.
