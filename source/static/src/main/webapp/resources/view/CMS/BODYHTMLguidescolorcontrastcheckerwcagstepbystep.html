Color Contrast Checker (WCAG) Step By Step

Last reviewed: 2026-07-20

Color Contrast Checker (WCAG) runs in three steps: open the page, pick text and background colors, then read the ratio and AA / AAA results. This page walks through each step.


Step 1 - open the tool

Go to Color Contrast Checker (WCAG). Nothing needs to be installed or signed in before you start.


Step 2 - pick foreground and background

Pick a foreground (text) color and a background color with the color boxes, or type HEX values. Use Swap colors if you want to flip them. The live preview shows sample normal and large text in those colors.


Step 3 - read the ratio and verdicts

The contrast ratio updates as you change colors. AA and AAA pass or fail badges appear for normal text and large text. Ratios use the W3C relative-luminance formula from WCAG 2.1. Closing the tab discards your picks - nothing is uploaded.
