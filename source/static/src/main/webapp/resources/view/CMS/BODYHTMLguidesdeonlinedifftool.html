Online-Diff-Tool

Ein Online-Diff-Tool liest zwei Textbloecke nebeneinander und hebt hervor, welche Zeilen hinzugefuegt, welche entfernt und welche zwischen den beiden Versionen geaendert wurden. Die Leseraufgabe ist die schnelle Aenderungspruefung: alte Version links einfuegen, neue Version rechts einfuegen, und das Tool kennzeichnet jede Zeile, sodass eine Konfigurationsabweichung, eine Release-Notiz oder ein Korrekturdurchgang auf einen Blick sichtbar wird, ohne einen der Texte in einem Editor zu oeffnen.

Umsetzendes Tool/developer-tools/text-diff.html

Das In-Browser-Diff-Tool unter /developer-tools/text-diff.html ist das richtige, wenn der Vergleich zwei Textbloecke umfasst - Konfigurationsdateien, JSON-Dumps, Quellcode-Revisionen, Textentwuerfe, Log-Ausschnitte. Es erledigt die Arbeit lokal im Browser-Tab; eingefuegter Text wird nicht an einen entfernten Server hochgeladen, und die Seite behaelt die beiden Eingaben nach dem Schliessen des Tabs nicht. Kein Konto, keine Installation, kein Abo. Fuer fortgeschrittenere Vergleiche (Zeilenebene vs Wortebene vs Zeichenebene vs Merge-Ansichten im Git-Stil) erklaert der Schwester-Leitfaden unter /guides/en/text-diff-vs-line-diff-vs-word-diff-vs-git-diff.html, welcher Modus sich fuer welche Eingabeform am besten liest.
