Quando Usar o Saros Cycle Eclipse Explorer 3D - Melhores Sessões e Limites

Última revisão: 2026-07-23

Abra o Saros Cycle Eclipse Explorer 3D quando precisar de uma aula real sobre a repetição de séries de eclipses, com dados publicados de saros e exeligmos - sem depender de uma efeméride em tempo real.


Uma aula sobre o padrão de repetição

A aula começa quando você avança por vários eclipses seguidos e vê a trajetória se deslocar cerca de 120 graus para oeste a cada passo, depois avança 3 passos de uma vez para ver o exeligmos trazê-la de volta perto de onde começou.


Sessões para as quais este explorador não serve

Sessões que esta página não atende são aulas sobre a geometria detalhada do cone de sombra de um único eclipse - use o Solar Eclipse 3D Explorer para isso - ou por que os eclipses se agrupam em duas estações por ano; use o explorador Lunar Nodes & Eclipse Seasons. Percorra os controles no guia passo a passo.
