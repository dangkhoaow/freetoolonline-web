Frequently asked questions

Can I open HEIC files on Windows without converting them?

Windows 10 and 11 require the Microsoft HEVC Video Extension (paid, ~$0.99 USD from the Microsoft Store) to open HEIC files natively in Photos or File Explorer. Without it, HEIC files show as blank thumbnails and fail to open. The free alternative is to convert to JPG first using the HEIC to JPG converter - no install needed, works in the browser.

Does converting HEIC to JPG reduce quality?

A one-step HEIC to JPG conversion at quality 90 or above is visually lossless for photographs at normal viewing sizes. The file size roughly doubles (HEIC averages 1.5-2.5 MB for a 12MP iPhone photo; the JPG equivalent is 3-5 MB). Converting repeatedly between lossy formats degrades quality with each pass, so convert once and keep both the original HEIC and the final JPG.

Which format is best for sending photos by email?

JPG. Every email client - Gmail, Outlook, Apple Mail, ProtonMail - renders JPG inline without any plugin. HEIC fails in most non-Apple email clients and some mobile apps. WebP renders in modern webmail but is unsupported in many desktop email clients. For email attachments that must open reliably for the recipient, JPG is the safest choice.

Is WebP replacing JPG on the web?

For new web images served by a server you control, WebP is now the recommended format. All current browsers support it (Chrome since 2014, Safari since version 14, Firefox since 2019). WebP reduces file size by 25-35% versus JPG at the same visual quality, which improves page load times. JPG remains the standard for email, print, and any context where you cannot guarantee WebP support on the receiving end.
