What the Ostafrikasaurus size numbers are not

What these size numbers are not is a measured body length or mass from Ostafrikasaurus fossils. Only the holotype tooth about 46 mm is securely published. The 8 m and 1,150 kg figures are speculative reconstruction for scale only. Soft-tissue color is still artistic. This model is not a fossil-accurate skeleton.
