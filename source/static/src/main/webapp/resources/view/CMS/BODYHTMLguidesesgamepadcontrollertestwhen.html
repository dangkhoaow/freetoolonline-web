Gamepad / Controller Test Cuando Usarlo

Last reviewed: 2026-07-21

Abre Gamepad / Controller Test cuando necesites confirmar que un mando USB o Bluetooth reporta botones y sticks - sin instalar una utilidad de escritorio.


Cuando aplica

Un mando nuevo parece muerto en un boton. Un stick deriva y quieres ver los valores del eje. Prefieres una prueba en la pestana, sin cuenta.


Que esperar

Activa el sondeo, pulsa un boton para que el navegador exponga el pad, mira indicadores y ejes. Reset limpia la sesion. No se sube nada.

Pasos exactos: guia paso a paso.
