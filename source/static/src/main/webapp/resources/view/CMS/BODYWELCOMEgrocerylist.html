Grocery List

Last reviewed: 2026-07-03

Keep a grocery list in the browser - add, check off, and delete items.

Use it for a quick checklist when you do not want to install an app or sign in.

Each item gets a checkbox to mark it done and a button to remove it. The list is saved in this browser via local storage, so it survives a page reload; clearing this site's browser data removes it.

To see how this browser tool compares to a desktop app or an upload-based service, see Grocery List vs desktop apps and online services.
