Linux no Navegador vs Maquina Virtual vs USB ao Vivo

Ha tres maneiras comuns de experimentar o Linux sem substituir seu sistema operacional: uma maquina baseada em navegador como Linux Online, um aplicativo de maquina virtual de desktop e um USB ao vivo. Elas diferem principalmente no esforco de configuracao, velocidade e onde seus arquivos ficam - esta pagina coloca os numeros lado a lado.

Como escolher

Escolha a maquina do navegador para trabalhos rapidos, seguros e sem instalacao: praticar comandos, testar um script arriscado, ensinar ou qualquer computador bloqueado onde instalar software nao e uma opcao - e para mover trabalho entre computadores, pois uma sessao exporta como um unico arquivo que voce pode importar em outro lugar para retomar exatamente. Escolha uma maquina virtual de desktop quando precisar de velocidade, software de 64 bits ou rede; escolha um USB ao vivo quando quiser testar o Linux no hardware real.
