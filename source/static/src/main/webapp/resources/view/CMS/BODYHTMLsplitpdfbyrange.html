Pull several ranges in one job

Beyond a single range, the start/end box also accepts a comma-separated list - for example 1-3,7,10-12 pulls three sections in one pass and hands back three separate PDFs (pages 1-3, page 7, and pages 10-12). Each range becomes its own file, so you can carve a long document into the exact pieces you need without running the tool once per section. Your file is uploaded over a secure connection, processed, and the outputs download from temporary storage that auto-deletes after a short retention window.
