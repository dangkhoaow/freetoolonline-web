Image EXIF Metadata Viewer Passo A Passo

Last reviewed: 2026-07-21

O Image EXIF Metadata Viewer funciona em tres passos: abrir, escolher imagem local, ler tags (strip opcional).


Passo 1 - abrir a ferramenta

Va ate Image EXIF Metadata Viewer. Nao e preciso instalar nada nem fazer login.


Passo 2 - escolher uma imagem

Use o seletor para JPEG, PNG, WebP, HEIC ou TIFF deste dispositivo. Aparece uma previa. Read Metadata fica desativado ate escolher ficheiro.


Passo 3 - ler tags (strip opcional)

Clique Read Metadata. Tabelas Camera, Date / Time, GPS e Other preenchem a pagina. Com GPS, o aviso vermelho mostra coordenadas e link OpenStreetMap. Download Without Metadata grava JPEG a 0.95 sem EXIF. Clear limpa a sessao - nada foi enviado.
