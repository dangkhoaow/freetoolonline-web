Grocery List Passo a Passo

Last reviewed: 2026-07-09

O Grocery List funciona em tres passos: abrir a ferramenta, informar sua entrada e depois baixar o resultado. Esta pagina explica o que cada passo espera.


Passo 1 - abra a ferramenta

Va para o Grocery List. Nada precisa ser instalado nem e necessario fazer login antes de comecar.


Passo 2 - informe sua entrada

Use para uma lista de verificacao rapida quando voce nao quiser instalar um aplicativo ou fazer login.


Passo 3 - obtenha o resultado

Cada item recebe uma caixa de selecao para marcar como concluido e um botao para remove-lo.

A lista e salva neste navegador via local storage, entao ela sobrevive a um recarregamento da pagina; limpar os dados do navegador deste site a remove.
