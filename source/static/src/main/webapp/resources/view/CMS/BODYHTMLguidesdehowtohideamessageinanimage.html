Wie man eine Nachricht in einem Bild versteckt (Steganografie, kostenlos + im Browser)

Steganografie versteckt eine kurze Textnachricht in den Pixeln eines gewoehnlichen Bildes, sodass das Bild unveraendert aussieht, aber heimlich Ihre Notiz traegt. Diese Anleitung zeigt, wie man eine Nachricht in einem PNG versteckt und sie spaeter wieder ausliest, optional mit einem Passwort gesperrt - alles im Browser, sodass das Bild und die Nachricht Ihr Geraet nie verlassen.

Wie man eine Nachricht in einem Bild versteckt

  1. Oeffnen Sie das Steganografie-Werkzeug und lassen Sie den Modus auf "Eine Nachricht in einem Bild verstecken".
  2. Waehlen Sie ein Traegerbild. Ein PNG funktioniert am besten; ein groesseres Bild fasst eine laengere Nachricht.
  3. Geben Sie die geheime Nachricht ein, die Sie verstecken moechten. Das Werkzeug zeigt die maximale Zeichenzahl, die dieses Bild fassen kann, bevor Sie einbetten.
  4. Wenn Sie moechten, fuegen Sie ein Passwort hinzu, damit nur wer es kennt die Nachricht wieder lesen kann.
  5. Klicken Sie auf die Verstecken-Schaltflaeche und laden Sie das Ergebnis herunter. Behalten Sie es als PNG.

Wie man eine versteckte Nachricht wieder ausliest

  1. Oeffnen Sie dasselbe Werkzeug und stellen Sie den Modus auf "Eine versteckte Nachricht anzeigen".
  2. Waehlen Sie das PNG, das mit diesem Werkzeug erstellt wurde.
  3. Wenn die Nachricht gesperrt war, geben Sie das Passwort ein. Der versteckte Text erscheint auf der Seite.

Warum das Ergebnis ein PNG bleiben muss

Die Nachricht wird im niederwertigsten Bit jedes Pixel gespeichert - eine winzige Aenderung, die das Auge nicht sieht. PNG ist verlustfrei, daher ueberleben diese Bits. Wenn Sie das Bild erneut als JPG speichern oder durch einen Kompressor schicken, werden die Bits ueberschrieben und die versteckte Nachricht wird geloescht. Die meisten sozialen und Messaging-Apps komprimieren Bilder ebenfalls neu, senden Sie das PNG also als Datei und nicht als Foto, wenn die Nachricht ueberleben soll.

Wie viel Text ein Bild fassen kann

Jedes Pixel speichert drei Bit (eines in Rot, Gruen und Blau), und jedes Zeichen verwendet sechzehn Bit, daher waechst die Kapazitaet mit der Pixelzahl. Grobe Richtlinie:

64 x 644096etwa 762
256 x 25665536etwa 12282
512 x 512262144etwa 49146

Wenn eine Nachricht fuer das gewaehlte Bild zu lang ist, nennt Ihnen das Werkzeug das Maximum und stoppt, anstatt den Text stillschweigend abzuschneiden.

Ist es privat?

Ja. Alles laeuft in Ihrem Browser auf Ihrem eigenen Geraet; das Bild und die Nachricht werden nie auf einen Server hochgeladen. Beachten Sie, dass Steganografie die Existenz einer Nachricht vor einem fluechtigen Betrachter verbirgt, aber ein entschlossener Analyst mit statistischen Pruefungen kann dennoch erkennen, dass Daten vorhanden sind. Behandeln Sie das optionale Passwort daher als den eigentlichen Schutz fuer den Inhalt.
