Kapan Pakai Mars Terrain Explorer

Last reviewed: 2026-07-15

Buka Mars Terrain Explorer saat Anda butuh literasi data elevasi nyata Olympus Mons dan Hellas Planitia - tanpa instal software GIS planet.

Pakai untuk pemanasan kelas soal kenapa "gunung tertinggi di tata surya" benar secara absolut tapi hampir tak terlihat pada skala planet, tugas membandingkan elevasi dataran tinggi dan rendah, atau lihat cepat angka MOLA asli lewat klik.

  • Cocok untuk: literasi data elevasi nyata dan rasa skala relief
  • Lewati jika: butuh gambar Mars lengkap, rute rover, atau analisis GIS mendalam
  • Privasi: di browser - tanpa unggah, tanpa akun
