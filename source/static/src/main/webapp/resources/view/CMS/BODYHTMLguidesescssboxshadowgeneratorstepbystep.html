CSS Box-Shadow Generator - Paso a Paso

Last reviewed: 2026-07-24

El CSS Box-Shadow Generator funciona en tres pasos: abrir la herramienta, arrastrar los controles, y copiar el resultado. Esta página detalla lo que hace cada paso.


Paso 1 - abre la herramienta

Ve al CSS Box-Shadow Generator. El cuadro de vista previa y una sombra inicial aparecen de inmediato - no necesitas instalar ni iniciar sesión antes.


Paso 2 - da forma a la sombra

Arrastra el desplazamiento horizontal y vertical para mover la sombra, el radio de desenfoque para suavizar su borde, y el radio de propagación para agrandarla o reducirla. Elige un color y una opacidad para la sombra, y marca Inset si quieres una sombra interna en vez de proyectada. La vista previa se actualiza con cada cambio.


Paso 3 - copia el resultado

Haz clic en Copy CSS para copiar la línea box-shadow generada al portapapeles, o selecciona el texto del cuadro de salida manualmente. Todo ocurre en la pestaña de tu navegador; no se envía nada.
