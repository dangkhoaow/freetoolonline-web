Twilight Types 3D Explorer frente a alternativas

Twilight Types 3D Explorer clasifica la zona de transición en bandas civil, náutica y astronómica y calcula el umbral de noches blancas de forma interactiva en el navegador - una calculadora de amanecer/atardecer gana para una fecha y lugar concretos; un diagrama de libro de texto gana como referencia impresa fija.
