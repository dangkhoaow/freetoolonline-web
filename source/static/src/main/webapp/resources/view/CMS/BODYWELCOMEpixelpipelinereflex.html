Pixel Pipeline Reflex

Last reviewed: 2026-07-10

Press Play and a retro pipeline HUD loads in the tab. Each round shows the next stage you must hit - Fetch, FIFO, Mix, or Light - while a scan beam counts down. Use number keys 1-4, tap the on-screen stage pad, or click the highlighted button before time runs out.

Controls at a glance:

ActionInput
Match stageKeys 1-4, tap, or click
Start runStart frame button inside game
PausePause button inside game
DifficultyChill / Arcade / Meltdown chips
SoundSound off/on toggle (optional synth blips)

Stabilize enough frames to win (10 on Chill, 12 on Arcade, 14 on Meltdown). Wrong presses and beam timeouts add glitches; exceed the glitch cap and the run ends. Correct chains raise a combo multiplier. Daily seeds rotate by date; you can load custom seeds in the seed field.

Best scores per seed and difficulty save locally in your browser only. The game is adapted from bte808's open-source Scanline Sprint project (MIT license); credits and license text ship next to the game files.
