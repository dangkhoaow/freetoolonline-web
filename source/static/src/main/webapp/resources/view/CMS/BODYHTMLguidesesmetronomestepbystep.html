Online Metronome - Paso a Paso

Last reviewed: 2026-07-21

Abre Online Metronome, fija el tempo, elige pulsos por compás y haz clic en Start click para ticks audibles.


Pasos

  1. Arrastra el control de BPM (40-240) o usa Tap tempo varias veces para igualar el pulso.
  2. Elige pulsos por compás para el acento del primer tiempo.
  3. Mira la lámpara visual - late al cargar para confirmar el tempo antes del sonido.
  4. Haz clic en Start click para ticks audibles; Stop silencia el audio y mantiene el pulso visual.

Privacidad

El tempo y el audio corren en el navegador. No se sube nada y no se usa el micrófono.
