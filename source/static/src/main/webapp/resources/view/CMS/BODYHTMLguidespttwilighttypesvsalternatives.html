Twilight Types 3D Explorer vs alternativas

O Twilight Types 3D Explorer classifica a zona de transição em faixas civil, náutica e astronômica e calcula o limite de noites brancas de forma interativa no navegador - uma calculadora de nascer/pôr do sol vence para uma data e local específicos; um diagrama de livro didático vence como referência impressa fixa.
