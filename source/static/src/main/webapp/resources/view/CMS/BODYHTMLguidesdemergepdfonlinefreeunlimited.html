Wie fuegt man PDF-Dateien online kostenlos zusammen?

Oeffnen Sie das PDF-Zusammenfuegen-Werkzeug, fuegen Sie die PDFs hinzu, die Sie kombinieren moechten, und die Seite verbindet sie Seite fuer Seite zu einer einzigen Ausgabedatei, die Sie herunterladen. Das Zusammenfuegen folgt der Reihenfolge, in der die Dateien im Panel erscheinen, also ziehen Sie die Zeilen in die gewuenschte Abfolge, bevor Sie kombinieren - zum Beispiel das Deckblatt zuerst, dann den Hauptteil, den Anhang zuletzt. Es gibt keinen Anmeldeschritt und kein Wasserzeichen auf dem Ergebnis. Die Browser-Option hat keine Obergrenze pro Zusammenfuegung ausser dem Speicher Ihres Geraets, und genau das bedeutet "unbegrenzt" in der Praxis: Sie werden nicht durch ein Tageskontingent begrenzt, sondern nur durch das, was Ihre eigene Maschine beim Erstellen der Datei aufnehmen kann.

Browser-Zusammenfuegung gegenueber dem Upload-Weg - was sollten Sie verwenden?

Die Seite bietet zwei Wege, Dateien zu kombinieren, und sie eignen sich fuer unterschiedliche Situationen. Die Browser-Zusammenfuegung laeuft vollstaendig auf Ihrem Geraet ueber eine PDF-Bibliothek, die in den Tab geladen wird; nichts verlaesst Ihren Computer, was sie zur richtigen Wahl fuer Vertraege, Krankenakten oder alles macht, das Sie lieber nicht an einen Server senden. Der Upload-Weg sendet die Dateien ueber HTTPS, erstellt die kombinierte PDF auf dem Server und loescht die hochgeladenen Quellen nach einer kurzen Aufbewahrungszeit; das ist nuetzlich, wenn Sie ein Geraet mit wenig Speicher nutzen und die kombinierte Datei gross ist. Beide erzeugen dieselbe Art von Ausgabe - eine PDF in der von Ihnen festgelegten Reihenfolge - daher geht es bei der Entscheidung um die Sensibilitaet des Dokuments und die Maschine, an der Sie arbeiten, nicht um das Ergebnis.

Was bleibt erhalten, wenn Sie PDFs kombinieren?

Beim Zusammenfuegen werden die Seiten jeder Quelldatei kopiert, statt sie neu zu rendern, sodass der Text auswaehlbar und durchsuchbar bleibt, anstatt zu einem flachen Bild zu werden. Eingebettete Schriftarten, Formularfelder und Lesezeichen werden in die kombinierte Datei uebernommen, wenn die Originale sie als gueltige Strukturen enthalten - es lohnt sich, das nach dem Herunterladen in Ihrem PDF-Reader zu pruefen, falls Ihre Quellen interaktive Formulare hatten. Da Seiten kopiert statt neu komprimiert werden, landet die kombinierte Datei ungefaehr bei der Gesamtgroesse ihrer Teile; das Kombinieren verkleinert die Summe nicht. Wenn das Ergebnis kleiner sein soll, komprimieren Sie es als separaten Schritt nach dem Zusammenfuegen, statt zu erwarten, dass das Verbinden beide Aufgaben auf einmal erledigt.

Was dieses Werkzeug nicht tut

Das Zusammenfuegen-Werkzeug kombiniert PDFs, die Sie bereits oeffnen koennen - es entsperrt keine verschluesselte Datei. Wenn eine Ihrer Eingaben passwortgeschuetzt ist, entfernen Sie das Passwort zuerst mit Remove PDF Password (Sie benoetigen dafuer das Passwort) und fuegen Sie dann die entsperrte Kopie zusammen. Es ist auch kein Format-Konverter: geben Sie ihm PDFs, keine Word- oder Bilddateien, und wandeln Sie diese zuerst in PDF um, wenn Sie sie im kombinierten Dokument brauchen. Schliesslich verbindet es ganze Dateien in der von Ihnen angeordneten Reihenfolge; es sortiert oder loescht keine einzelnen Seiten innerhalb einer Quell-PDF, also schneiden Sie unerwuenschte Seiten heraus, bevor Sie eine Datei zum Zusammenfuegen hinzufuegen.

Wenn Sie ein neues PDF von Grund auf erstellen - Ueberschriften, Listen, Tabellen und Seitenumbrueche - anstatt bestehende Dateien zu kombinieren, verwenden Sie stattdessen Compose PDF.
