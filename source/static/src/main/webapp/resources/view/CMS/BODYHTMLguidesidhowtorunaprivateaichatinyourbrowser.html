Cara menjalankan chat AI pribadi di browser Anda - gratis, tanpa unggah

Jalankan chatbot AI sepenuhnya di browser Anda - model diunduh sekali dan berjalan di perangkat Anda sendiri, jadi pesan Anda tidak pernah diunggah dan tidak perlu akun. Pilih model kecil, biarkan dimuat, lalu mengobrol. Diperlukan browser dengan WebGPU, seperti Chrome atau Edge terbaru di desktop.

Langkah

  1. Buka alat dan pilih model dari daftar - model lebih kecil dimuat lebih cepat.
  2. Klik Load model. Pertama kali, model diunduh ke browser Anda (ratusan MB) dan disimpan di cache untuk nanti.
  3. Tunggu bilah progres selesai; status menunjukkan saat model siap.
  4. Ketik pesan dan klik Send (atau tekan Ctrl+Enter). Balasan mengalir masuk, dibuat di perangkat Anda.
  5. Ganti model kapan saja; pilihan terakhir Anda diingat di browser ini.

Model dan ukuran unduhan

Model lebih kecil dimuat paling cepat; yang lebih besar menjawab lebih baik tetapi mengunduh lebih banyak. Setiap model diunduh sekali lalu disimpan di cache oleh browser Anda:

Apa yang tetap pribadi

Model berjalan di perangkat Anda sendiri melalui WebGPU. Pesan Anda tidak pernah diunggah, tidak ada akun, dan tidak ada yang dikirim ke server. Satu-satunya unduhan jaringan adalah model itu sendiri, diambil sekali dari registri model lalu disimpan di cache oleh browser Anda.

Yang tidak dilakukan alat ini

Alat ini perlu browser dengan WebGPU (Chrome atau Edge terbaru di desktop, atau Chrome di Android); browser tanpa WebGPU menampilkan pemberitahuan. Riwayat obrolan tetap di tab selama sesi dan tidak disimpan atau disinkronkan antar perangkat. Ini model obrolan lokal, jadi tidak menjelajah web atau membaca berkas Anda.
