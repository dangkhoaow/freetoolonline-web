Das Drei-Schritt-Rezept

Öffnen Sie den Ersteller, ziehen Sie den Ordner auf die Upload-Zone und klicken Sie auf Konvertieren. Die Ordnerstruktur bleibt im resultierenden Archiv intakt, eine Eigenschaft, die es zu bestätigen lohnt, wenn Sie ein Projekt übergeben: Der Empfänger entpackt die .zip und findet dieselbe verschachtelte Struktur wie auf der Festplatte, nicht einen flachen Dump jeder Datei aus jedem Unterordner. Benennen Sie das Archiv im Einstellungen-Modal, bevor Sie es bauen, damit der Empfänger beim Eintreffen etwas Offensichtliches sieht ("finanzen-q3.zip" schlägt "archive.zip"). Die Ausgabe ist eine einzelne .zip-Datei, an Ihren Browser als Download zurückgegeben.

Wann Sie ein Passwort auf dem Archiv brauchen

Wenn der Ordner über einen Kanal reist, dem Sie nicht voll vertrauen können - eine weitergeleitete E-Mail, ein geteilter Posteingang, eine Chat-Gruppe - bietet das Einstellungen-Modal ein optionales Passwortfeld mit drei Verschlüsselungsoptionen. Die Standard-ZIP-Verschlüsselung ist universell akzeptiert, aber nach aktuellen Standards schwach. AES-128 ist der moderne Standard, akzeptiert von Windows-Explorer (Rechtsklick Extrahieren) und Archive Utility von macOS. AES-256 ist die höchste Stufe mit derselben Kompatibilitätsobergrenze wie AES-128. Senden Sie das Passwort über einen separaten Kanal - Passwort und Datei zusammen zu senden macht den Zweck zunichte. Das Passwort wird zur Build-Zeit angewendet, sodass der Empfänger beim Extrahieren aufgefordert wird.

Was der Ersteller nicht tut

Der Ersteller baut neue Archive - er knackt keine Verschlüsselung auf einer .zip, die jemand anderes gebaut hat. Er gibt kein 7z oder RAR aus (beide sind dichter als ZIP, erfordern aber eine Drittinstallation auf den meisten Empfängermaschinen, sodass der Ersteller bei ZIP für plattformübergreifende Reichweite bleibt). Er entfernt keine Metadaten aus den Eingabedateien: EXIF auf einem JPG, das Autorfeld auf einem Word-Dokument und die Änderungszeitstempel reisen alle unverändert ins Archiv. Wenn diese Metadaten gereinigt werden müssen, tun Sie das vor dem Hochladen des Ordners.
