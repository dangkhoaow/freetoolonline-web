Gamepad / Controller Test Langkah Demi Langkah

Last reviewed: 2026-07-21

Gamepad / Controller Test terdiri dari tiga langkah: buka, aktifkan polling, tekan tombol dan gerakkan stick.


Langkah 1 - buka

Buka Gamepad / Controller Test. Tanpa instal dan tanpa login.


Langkah 2 - aktifkan polling

Klik Enable gamepad polling. Bingkai panel aktif dan status menampilkan Polling.


Langkah 3 - tombol dan stick

Tekan tombol apa pun pada stik. Lihat indikator bernomor dan nilai sumbu. Reset membersihkan bacaan - tidak ada yang keluar dari tab.

Bandingkan opsi: vs alternatif.
