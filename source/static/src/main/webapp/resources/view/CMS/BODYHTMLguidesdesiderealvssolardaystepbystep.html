Wie Man Sidereal vs Solar Day 3D Explorer Verwendet - Schritt Fuer Schritt

Last reviewed: 2026-07-15

Spielen Sie einen Sonnentag ab, beobachten Sie den orangen Beobachter gegen die blaue Sternlinie und die goldene Sonnenlinie, und lesen Sie dann ~23h 56m 04s vs 24h (~3m 56s) im Fakten-Panel.

  1. Oeffnen Sie Sidereal vs Solar Day 3D Explorer und warten Sie, bis Erde, Sonne und der ferne Stern gezeichnet sind.
  2. Ziehen Sie zum Drehen, halten Sie das Fakten-Panel sichtbar, und beachten Sie die mittleren Sterntag- und Sonnentag-Laengen.
  3. Nutzen Sie Play / Pause und +5% day, um einen Sonnentag Lehrzeit durchzuscrubben.
  4. Schalten Sie Star only / Sun only / Both lines um, um jede Definition zu isolieren.
  5. Setzen Sie zurueck und wiederholen Sie, damit die Luecke von ~3m 56s die verlaessliche Kernaussage bleibt.
