Cara Main Violence Town - Langkah demi Langkah

Halaman Violence Town memuat RPG 2D berbasis giliran ~1.1 MB di iframe. Anda antar burger untuk Borgir, jelajahi delapan zona overworld plus interior, bertarung dengan roda aksi radial, dan autosave lokal. Tekan Play untuk mulai.

Last reviewed: 2026-07-17

WASD atau panah bergerak (tahan untuk terus jalan); ketuk/klik tanah untuk jalur; Space membuka roda aksi; 1-9 inventori; T tunggu; Esc batal; ? bantuan; L log; E periksa; Tab Remoticon. Autosave ke ftol:violencetown:save; pengaturan di ftol:violencetown:settings.

Langkah 1 - tekan Play dan mulai atau lanjutkan

Tekan Play di halaman game untuk memuat iframe (~1.1 MB). Di splash, tekan GAME START atau Space untuk run baru, atau CONTINUE jika ada save ber-namespace. Engine-nya Violencetown oleh Caelan Gander (MIT); LICENSE dan CREDITS ada di samping file game.

Langkah 2 - bergerak dan berinteraksi

Jalan satu tile dengan WASD atau panah (tahan untuk terus) atau ketuk/klik tanah untuk mengitari bangunan. Ketuk/klik target untuk berjalan mendekat dan melakukan aksi jelas (Take / Talk / Attack). Long-press, klik kanan, atau F membuka daftar kata kerja penuh untuk lawan di depan Anda.

Langkah 3 - bertarung dengan roda aksi

Tekan Space di mana saja untuk membuka roda aksi radial yang sudah diarahkan ke musuh terdekat. Double-tap Space mengulang aksi terakhir. Di roda, Atas/Bawah pilih ring (aksi / item / arah), Kiri/Kanan putar, Space tembak, Esc mundur. Di touch, ketuk ✦ untuk membuka roda.

Langkah 4 - inventori dan Remoticon

Tekan 1 sampai 9 untuk memilih slot hotbar; Space membuka Use / Smash / Give (lempar ada di roda). Tab membuka Remoticon (ITEMS / GEAR / QUESTS / MAP); [ ] ganti tab; C / J / M loncat ke GEAR / QUESTS / MAP. Di touch, ketuk ▤ untuk Remoticon.

Langkah 5 - simpan, opsi, dan zona

Autosave menulis ke ftol:violencetown:save dengan kunci backup dan staging. Opsi (musik, SFX, mute, reduce motion, fullscreen, mode buka roda) tersimpan di ftol:violencetown:settings. Delapan zona overworld (Town, Downtown, Factory, Sewer, Carnival, Graveyard, Wilderness, Canyon) terhubung di peta dunia.
