Does Pixel Realm RPG save progress?

Yes. Your position, inventory, equipment, and map progress persist in localStorage under namespaced ftol:pixelrealmrpg keys. Settings also offer download and import of a JSON save file.

What controls does the game use?

WASD or arrows move. X attacks. F interacts with chests, doors, and fountains. I opens inventory. G opens equipment. E uses the selected item. Bracket keys cycle inventory selection.

How large is the download?

About ~3.8 MB across HTML, JavaScript modules, map CSV/JSONC data, and local sprite assets.
