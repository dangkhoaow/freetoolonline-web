When to Use the GPS Time Dilation 3D Explorer - Best Sessions and Limits

Last reviewed: 2026-07-24

Open the GPS Time Dilation 3D Explorer when you want to see why a GPS satellite's clock needs a relativistic correction and what happens if that correction is switched off - not to check a live satellite's actual current clock telemetry or run a to-scale orbital-mechanics projection.


A correction-on-vs-off lesson

A lesson starts with the correction toggle ON to see the drift held at zero no matter how far the day slider is scrubbed, then switches the toggle OFF to watch the same day count produce a growing microsecond drift and a matching kilometer-scale position-error estimate.


Sessions this explorer does not fit

Sessions this page does not fit are checking a live satellite's actual current clock telemetry, or studying the pure gravitational clock-rate effect on its own without the velocity effect mixed in - use the Gravitational Redshift 3D Explorer for that. Walk the controls in the step-by-step guide.
