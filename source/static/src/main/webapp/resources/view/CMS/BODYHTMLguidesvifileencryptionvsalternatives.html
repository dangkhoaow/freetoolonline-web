Cong Cu Ma Hoa File la mot trong ba cach pho bien de lam viec nay: mot cong cu chay tren trinh duyet nhu the nay, mot ung dung desktop, hoac mot dich vu truc tuyen dua tren tai len. Moi cach co diem manh yeu khac nhau ve noi du lieu cua ban di den va thu ban can cai dat.

File duoc xao tron bang AES-256 dung mot khoa duoc suy ra tu mat khau cua ban (PBKDF2, 250.000 vong lap) cong voi salt va IV ngau nhien moi lan, nen ma hoa cung mot file hai lan voi cung mat khau se tao ra hai file .enc khac nhau - dieu nay la binh thuong, khong phai loi.

Mat khau sai, hoac file .enc bi sua doi hoac hong, se bi tu choi voi mot thong bao loi ro rang thay vi tao ra mot file hong hoac day du lieu vo nghia.
