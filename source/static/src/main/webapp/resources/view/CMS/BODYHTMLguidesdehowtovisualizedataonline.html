Daten online visualisieren - ein Diagramm aus Ihren Zahlen erstellen

Ein Datenvisualisierer verwandelt Zahlenzeilen in ein Diagramm in Ihrem Browser. Fuegen Sie CSV, JSON oder einfache Label, Wert-Zeilen ein, waehlen Sie einen Diagrammtyp und sehen Sie es sofort gezeichnet. Balken, Linie, Kreis und mehr sind nur einen Klick entfernt. Alles laeuft auf Ihrem Geraet, ohne Upload und ohne Konto.

Schritte

  1. Fuegen Sie Ihre Daten als CSV mit einer Kopfzeile, als JSON oder als ein Paar "Label, Wert" pro Zeile ein - oder laden Sie ein eingebautes Beispiel mit einem Klick.
  2. Waehlen Sie einen Diagrammtyp: Balken, Linie, Kreis, Ring, Polarbereich, Radar, Streuung, Blase oder gestapelte Balken.
  3. Geben Sie dem Diagramm einen Titel und klicken Sie auf Visualisieren; das Diagramm wird neu gezeichnet, wenn Sie Typ, Titel oder Daten aendern.
  4. Oeffnen Sie ein letztes Diagramm aus der Liste erneut - die letzten Diagramme werden in diesem Browser gemerkt.

Eingabeformate

Drei Eingabeformate werden automatisch erkannt. CSV nutzt eine Kopfzeile mit Komma- oder Zeilenumbruch-Trennzeichen. JSON kann ein Array von Objekten sein (ein Namensschluessel plus numerische Wertschluessel) oder ein einfaches Schluessel/Wert-Objekt. Einfacher Text ist ein Paar "Label, Wert" pro Zeile. Das Beispiel unten zeichnet fuenf Werte als Balkendiagramm.

KI-Einblick auf dem Geraet

In einem Browser mit WebGPU koennen Sie einen optionalen privaten KI-Einblick ausfuehren. Das kleine Sprachmodell wird einmal heruntergeladen und laeuft auf Ihrem Geraet, liest Ihre Daten und baut das Diagramm neu auf - nichts wird hochgeladen. Ohne WebGPU arbeitet die Diagramm-Engine allein, sodass das Werkzeug Ihre Daten immer zeichnet.

Datenschutz

Alles laeuft in Ihrem Browser. Es gibt kein Upload-Widget und kein Konto; die letzten Diagramme bleiben nur auf Ihrem Geraet und Ihre Daten verlassen die Seite nie.
