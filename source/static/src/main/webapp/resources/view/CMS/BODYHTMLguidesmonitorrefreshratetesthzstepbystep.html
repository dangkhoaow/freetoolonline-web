Monitor Refresh Rate Test (Hz) Step By Step

Last reviewed: 2026-07-21

Monitor Refresh Rate Test (Hz) takes three short steps: open the page, start measuring, then read the Hz estimate.


Step 1 - open the tool

Go to Monitor Refresh Rate Test (Hz). No install and no login.


Step 2 - start measuring

Click Start measuring. Keep this tab in the foreground so requestAnimationFrame is not throttled.


Step 3 - read Hz and stop

Watch the large Hz reading settle as samples accumulate. Click Stop to freeze it, or Reset to clear. Nothing left the tab.
