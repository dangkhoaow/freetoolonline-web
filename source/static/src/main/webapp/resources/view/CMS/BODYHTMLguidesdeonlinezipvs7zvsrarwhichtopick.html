30-Sekunden-Antwort. Zip wenn der Empfaenger Windows, macOS oder ein Telefon nutzt - es ist das einzige Format, das ohne zusaetzliche Software ueberall oeffnet. 7z wenn Sie die Werkzeuge des Empfaengers kontrollieren und das kleinste Archiv wollen. Rar wenn Sie eines oeffnen muessen.

Was jedes Format ist

  • Zip. Phil Katz' Format von 1989. Nativ unter Windows seit XP, nativ unter macOS, nativ auf Android und iOS. Universelle Kompatibilitaet, bescheidene Kompression.
  • 7z. Igor Pavlovs Format von 1999. Verwendet LZMA2 standardmaessig - typischerweise 10-30% kleiner als Zip bei textlastigen Archiven. Benoetigt 7-Zip, The Unarchiver oder eine kostenpflichtige App zum Oeffnen.
  • Rar. Proprietaeres Format von 1993. Kostenlos zum Extrahieren, kostenpflichtig zum Erstellen mit dem offiziellen WinRAR. Starke Kompression aehnlich wie 7z.
