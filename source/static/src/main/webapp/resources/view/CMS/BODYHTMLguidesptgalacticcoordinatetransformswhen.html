Quando Usar o Galactic Coordinate Transforms 3D Explorer - Melhores Sessões e Limites

Última revisão: 2026-07-23

Abra o Galactic Coordinate Transforms 3D Explorer quando precisar ver como os três planos de referência do céu se relacionam entre si e como a precessão axial muda lentamente qual estrela fica junto ao polo.


Uma lição sobre referenciais de coordenadas e precessão

Uma lição começa quando você ativa e desativa cada plano para ver os ângulos de inclinação reais, depois arrasta o controle de ano de precessão para observar o polo do plano equatorial traçar seu círculo de 25.772 anos pelas eras de Thuban, Polaris e Vega.


Sessões para as quais este explorador não é indicado

Este explorador não é indicado para lições sobre grades celestes baseadas no observador (ascensão reta/declinação versus altitude/azimute) - use o Coordinate Systems Explorer para isso - nem para o percurso do Sol pelas constelações do zodíaco - use o Ecliptic & Zodiac Explorer. Percorra os controles no guia passo a passo.
