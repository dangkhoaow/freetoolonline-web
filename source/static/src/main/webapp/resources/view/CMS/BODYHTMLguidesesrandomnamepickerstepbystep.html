Selector de Nombres Aleatorio Paso a Paso

Last reviewed: 2026-07-12

Selector de Nombres Aleatorio funciona en tres pasos: abrir la herramienta, indicar tu lista y luego obtener el resultado. Esta pagina explica que esperar en cada paso.


Paso 1 - abre la herramienta

Ve al Selector de Nombres Aleatorio. No necesitas instalar nada ni iniciar sesion antes de empezar.


Paso 2 - indica tu lista

Usalo para sorteos, decidir quien empieza primero, o elegir un ganador aleatorio de una lista.


Paso 3 - obten el resultado

El resultado usa crypto.getRandomValues para garantizar imparcialidad y se ejecuta localmente - sin subir archivos y sin cuenta.

Edita la lista y vuelve a girar cuando quieras; los datos no se guardan entre visitas.
