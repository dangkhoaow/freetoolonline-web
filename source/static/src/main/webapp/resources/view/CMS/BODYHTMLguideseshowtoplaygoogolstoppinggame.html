Googol Stopping Game - ~8 KB iframe puzzle.

Last reviewed: 2026-07-20

Pulsa Play, haz clic en fichas numeradas para revelar valores enormes y pulsa I'm done cuando creas que tienes el mayor.

1 - Play

Play -> iframe (~8 KB). 50 tiles + Computer + I'm done.

2 - Reveal

Click/tap a number. History shows the value. One use per tile.

3 - Stop

Last reveal is your pick. Press I'm done when you think it is the biggest.

4 - Computer

Computer demos the n/e sample-then-threshold strategy.
