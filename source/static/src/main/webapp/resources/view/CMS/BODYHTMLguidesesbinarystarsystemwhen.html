Cuando usar Binary Star System 3D Explorer

Usa el Binary Star System 3D Explorer cuando necesites ver dos pares reales orbitando un baricentro en el navegador - y evitalo si necesitas efemerides de telescopio o un modelo triple completo.

  • Demos de clase del baricentro y cociente de masas con cifras publicadas de Sirius o Alpha Centauri
  • Repaso rapido antes de separar Sirius o leer efemerides de Alpha Centauri AB
  • Comparar un binario visual compacto (Sirius) con un par solar mas ancho y excentrico

Esta pagina no reemplaza efemerides profesionales, no incluye Proxima Centauri y no resuelve gravedad n-cuerpos. No afirma angulos en el cielo.
