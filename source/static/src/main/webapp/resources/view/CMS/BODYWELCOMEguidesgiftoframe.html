Whether a frame is going into an image editor, a sprite sheet, or a static image fallback, the first practical decision is output format: PNG for transparency, JPG for size. PNG output preserves the per-frame alpha channel - edges stay clean against any background colour. JPG output is useful when the smaller file size matters, such as frames destined for a social post or a slideshow where lossless transparency is not needed.
