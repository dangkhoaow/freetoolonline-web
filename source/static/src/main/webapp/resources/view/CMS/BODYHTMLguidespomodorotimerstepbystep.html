Pomodoro Timer Online Step By Step

Last reviewed: 2026-07-11

Pomodoro Timer Online takes three steps: set your Work/Break minutes, click Start, and let the phase keep alternating. This page walks through each step.


Step 1 - set Work and Break minutes

Open Pomodoro Timer Online. Two number fields are ready before you start: Work (1-120 minutes, default 25) and Break (1-60 minutes, default 5). Change either one if the standard 25/5 split does not fit your session.


Step 2 - click Start

Click Start to begin the Work interval. The phase label reads "Work" and the countdown ticks down from your chosen Work minutes. Pause holds the countdown in place without changing the phase; clicking Start again resumes it.


Step 3 - let it flip between Work and Break

When the Work countdown reaches zero, the phase label switches to "Break" and the countdown reloads from your Break minutes; when the Break ends it switches back to Work, and this keeps repeating for as long as you leave the tab open. Reset at any point returns to the start of the current Work interval.

Everything runs locally in this browser tab - no upload and no account. Closing or reloading the tab stops the timer; it does not run or notify in the background.
