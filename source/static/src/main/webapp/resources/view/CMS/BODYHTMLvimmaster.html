VIM MASTER

JSON-driven Vim lessons in a simulated editor. Start Learning dismisses onboarding, then type real Vim keys to clear objectives. Progress uses ftol:vimmaster:* keys. Adapted from renzorlive/vimmaster (MIT). About ~584 KB.

Keyboard required. Best on desktop or a large tablet with a physical keyboard.

Press Play to load the game.
