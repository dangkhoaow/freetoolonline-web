Como Esconder uma Mensagem em uma Imagem (Esteganografia, Gratis + No Navegador)

A esteganografia esconde uma breve mensagem de texto dentro dos pixels de uma imagem comum, entao a foto parece inalterada mas carrega secretamente a sua nota. Este guia mostra como esconder uma mensagem em um PNG e le-la de volta depois, opcionalmente protegida com senha - tudo no seu navegador, entao a imagem e a mensagem nunca saem do seu dispositivo.

Como esconder uma mensagem em uma imagem

  1. Abra a ferramenta de Esteganografia e mantenha o modo em "Esconder uma mensagem em uma imagem".
  2. Escolha uma imagem de capa. Um PNG funciona melhor; uma imagem maior comporta uma mensagem mais longa.
  3. Digite a mensagem secreta que voce quer esconder. A ferramenta mostra o maximo de caracteres que aquela imagem comporta antes de embutir.
  4. Se quiser, adicione uma senha para que apenas quem a conhece possa ler a mensagem de volta.
  5. Clique no botao de esconder e baixe o resultado. Mantenha como PNG.

Como ler uma mensagem escondida de volta

  1. Abra a mesma ferramenta e mude o modo para "Revelar uma mensagem escondida".
  2. Escolha o PNG que foi feito com esta ferramenta.
  3. Se a mensagem foi protegida, digite a senha. O texto escondido aparece na pagina.

Por que o resultado precisa continuar PNG

A mensagem fica guardada no bit menos significativo de cada pixel - uma mudanca minuscula que o olho nao percebe. O PNG e sem perdas, entao esses bits sobrevivem. Se voce salvar a imagem de novo como JPG, ou passa-la por um compressor, os bits sao reescritos e a mensagem escondida e apagada. A maioria dos apps de redes sociais e de mensagens tambem recomprime imagens, entao envie o PNG como arquivo e nao como foto se precisar que a mensagem sobreviva.

Quanto texto uma imagem comporta

Cada pixel guarda tres bits (um em vermelho, verde e azul), e cada caractere usa dezesseis bits, entao a capacidade cresce com a quantidade de pixels. Guia aproximado:

Se uma mensagem for longa demais para a imagem escolhida, a ferramenta informa o maximo e para, em vez de cortar o texto silenciosamente.

E privado?

Sim. Tudo roda no seu navegador, no seu proprio dispositivo; a imagem e a mensagem nunca sao enviadas para um servidor. Observe que a esteganografia esconde a existencia de uma mensagem de um observador casual, mas um analista determinado rodando verificacoes estatisticas ainda pode detectar que ha dados presentes, entao trate a senha opcional como a real protecao do conteudo.
