Perguntas frequentes

Por que esta ferramenta nao salva o audio como MP3?

Os navegadores nao expoe o audio sintetizado como um arquivo de audio gravavel. Para obter um MP3, um servico de API de servidor seria necessario - o que criaria um upload.

Por que escolher a ferramenta do navegador?

Nenhuma instalacao e nenhum texto e enviado a um servidor - e a escolha mais rapida para uso ocasional quando a privacidade importa.
