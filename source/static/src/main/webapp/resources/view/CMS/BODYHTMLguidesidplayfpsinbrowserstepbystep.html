Main FPS Klasik di Browser Langkah demi Langkah

Last reviewed: 2026-07-12

Buka Retro FPS Online, klik salah satu kartu kampanye, dan game berlisensi bebas Freedoom akan mengunduh sekali dan langsung mulai di tab. Halaman ini memandu klik demi klik yang tepat - memulai, bermain, menyimpan, dan memindahkan progres Anda ke komputer lain dengan file ekspor.


Langkah 1 - pilih kampanye

Halaman menampilkan dua kartu: Kampanye 1 (empat episode, 36 level) dan Kampanye 2 (32 level dalam satu sesi). Setiap kartu menyatakan ukuran unduhan satu kalinya sebelum apa pun diunduh. Klik salah satu; bilah progres menunjukkan proses unduhan, dan layar judul muncul saat selesai. Unduhan disimpan dalam cache browser ini, jadi kunjungan berikutnya akan melewati langkah ini - bahkan tanpa koneksi.


Langkah 2 - mulai bermain

Gunakan menu bawaan game untuk memulai permainan baru dan memilih tingkat kesulitan. Klik layar sekali untuk memberikan kendali keyboard dan mouse: tombol panah bergerak, Ctrl menembak, Space membuka pintu, dan menggerakkan mouse memutar arah pandang. Tombol Fullscreen di toolbar memenuhi layar; Esc keluar dari layar penuh atau membuka menu game.


Langkah 3 - simpan dan muat

Tekan Esc saat bermain, pilih Save Game, dan pilih satu slot - alur yang sama seperti versi desktop aslinya. Simpanan tersimpan di browser ini dan tetap ada setelah muat ulang; muat kembali dari menu yang sama. Panel status halaman menunjukkan berapa banyak file simpanan yang disimpan browser ini.


Langkah 4 - lanjutkan di komputer lain

Klik Export saves di toolbar; halaman akan mengunduh satu file .ftolfps yang berisi semua slot simpanan dan pengaturan. Di komputer lain, buka halaman yang sama, gunakan kartu Import untuk memuat file tersebut, lalu mulai satu kampanye dan pilih Load Game - slot Anda akan ada di sana. Jika versi engine yang dipublikasikan berubah sejak file diekspor, proses impor akan ditolak dengan pesan yang jelas dan bukan memuat secara salah. Belum yakin ini pengaturan yang tepat untuk Anda? Lihat kapan game tembak-menembak berbasis browser cocok dan bagaimana perbandingannya dengan menginstal salah satu.
