Tylosaurus Size Comparison

Last reviewed: 2026-07-18

Tylosaurus 3D Viewer pairs published Late Cretaceous Western Interior Seaway figures with a 1.8 m person on screen - length about 8-14 m (Size vs human uses 13 m), mass about 1.1 tonnes to several tonnes - so the mosasaur scale gap stays readable at a glance.

How the Tylosaurus size-vs-human toggle stays honest

The Tylosaurus model is drawn to a fixed on-screen length so it fits the canvas; the Size vs human control then places a 1.8 m person at the true length ratio used by the viewer (13 m). Tap Size vs human under the canvas to show or hide the scale figure - the comparison is proportional, not a decorative sticker. Published length still spans about 8-14 m (with some specimens near 12-13.2 m), so the toggle picks one clear scale inside that disclosed range.

What the Tylosaurus size numbers are not

What these size numbers are not is a claim of one precise length or weight: the Tylosaurus size comparison discloses the about 8-14 m / 1.1 tonnes-to-several-tonnes ranges and uses 13 m only for the Size vs human scale; it does not invent soft-tissue color accuracy, and this is not a win/lose game.
