How to extract HEX colors from an image in your browser

  1. Open Image Color Palette Extractor and choose a local image file.
  2. Select 5, 8, 12, or 16 colors from the Colors to extract list.
  3. Click Extract Palette and wait for the swatch row to appear.
  4. Click any swatch to copy its HEX value to the clipboard.
  5. Clear when you are done so the preview and swatches leave the tab.

Nothing uploads. The tool ranks bucketed RGB frequencies on a downsized canvas - it does not run a cloud AI model.
