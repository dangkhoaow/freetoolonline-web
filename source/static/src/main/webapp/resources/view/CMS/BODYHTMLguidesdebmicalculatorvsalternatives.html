Wann BMI Calculator waehlen

Waehlen Sie BMI Calculator fuer eine einmalige BMI-Pruefung, wenn Sie nichts installieren moechten und die Zahlen auf Ihrem Geraet bleiben sollen. Metrische und imperiale Einheiten sind im selben Menue verfuegbar, ohne manuelle Umrechnung.

Waehlen Sie eine Gesundheits-/Fitness-App, wenn Sie den BMI ueber die Zeit zusammen mit anderen Gesundheitswerten verfolgen moechten. Waehlen Sie einen Upload-Dienst nur, wenn Ihre Daten bereits in einem Format vorliegen, das dieser Dienst verarbeitet.

Haeufige Fragen

Warum das Browser-Werkzeug statt einer App waehlen?

Nichts zu installieren und kein Konto notwendig - Gewicht und Groesse eingeben und das Ergebnis ablesen.

Ist das Ergebnis eine medizinische Beratung?

Nein. Das Ergebnis ist eine allgemeine Schaetzung, keine medizinische oder professionelle Beratung.
