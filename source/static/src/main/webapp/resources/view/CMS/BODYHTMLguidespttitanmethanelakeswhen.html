Quando usar o Titan Methane Lakes 3D Explorer - Melhores momentos e limites

Última revisão: 2026-07-22

Abra o Titan Methane Lakes 3D Explorer quando precisar de uma lição sobre mares de metano e atmosfera espessa com dados publicados pela Cassini e pela Huygens - sem um laboratório de hidrologia.


Uma lição sobre mares de metano e atmosfera

A lição começa quando você desativa a névoa para revelar os mares polares e ativa o corte para ver a camada de gelo e o oceano, enquanto o painel mantém as faixas publicadas de atmosfera, pressão e tamanho dos mares.


Sessões que este explorador não atende

Este explorador não é indicado apenas para a mecânica de plumas de vapor de água. Use o Enceladus Geysers 3D Explorer para isso. Siga os controles no guia passo a passo.
