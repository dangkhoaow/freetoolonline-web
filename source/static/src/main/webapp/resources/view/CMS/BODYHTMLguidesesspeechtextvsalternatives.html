Voz a Texto vs Alternativas

Voz a Texto es una de las tres formas comunes de transcribir voz: una herramienta en el navegador como esta, una aplicacion de escritorio o un servicio en linea con carga. Cada una tiene ventajas distintas en privacidad e instalacion.

Preguntas frecuentes

Por que esta limitado a Chrome y Safari?

La API Web Speech Recognition no esta disponible en Firefox. Chrome, Edge (Chromium) y Safari soportan la API de forma nativa.

Puedo usarlo para transcribir archivos de audio pregrabados?

No. Esta herramienta captura audio en vivo del microfono. Para archivos pregrabados, un servicio dedicado de transcripcion seria mas adecuado.
