Frequently Asked Questions

What does Mono Stack Blocks do?

It is a line-clear stack game on a 10x20 grid. Stack tetrominoes, clear full rows for score, level up gravity every 10 lines, and use hold plus ghost preview. SRS wall kicks and 7-bag randomizer are built in.

Does it work on a phone?

The page loads on phones, but the engine has no on-screen controls. Use a keyboard on desktop or laptop for move, rotate, hard drop, and hold.

Is anything saved in my browser?

Yes. High scores and theme settings use localStorage key ftol:monostackblocks:storage via the settings stub. Reset all high scores clears bests from Settings. Nothing is uploaded.

How big is the download?

About ~58 KB of HTML, CSS, and JS after you press Play. The browser caches it for later visits on the same device.

How do I play?

Left and right arrows move. Down soft drops. Up, W, or X rotates clockwise; Z or Q rotates counter-clockwise. Space hard drops. C or Shift swaps hold once per piece. P pauses. R restarts.

Is this open source?

Yes. It is adapted from the blocks module in Cursor Arcade by flancast90 (flancast90/cursor-arcade-games) under the MIT license. This site build stubs the VS Code webview API and ships LICENSE and CREDITS.
