Online-Video-Trimmer Schritt Fuer Schritt

Last reviewed: 2026-07-11

Online-Video-Trimmer laeuft in drei Schritten: das Tool oeffnen, Start- und Endzeit angeben, dann das Ergebnis herunterladen. Diese Seite erklaert, was jeder Schritt erwartet.


Schritt 1 - das Tool oeffnen

Vor dem Start muss nichts installiert oder angemeldet werden.


Schritt 2 - die Zeiten angeben

Gib eine Start- und Endzeit in Sekunden oder im Format mm:ss ein (zum Beispiel 90 oder 1:30). Lasse die Endzeit leer, um von der Startzeit bis zum Ende der Datei zu trimmen. Das Trimmen kodiert nicht neu - es laeuft im Stream-Copy-Modus von FFmpeg, sodass Aufloesung, Bitrate und Codec des Originals genau erhalten bleiben.


Schritt 3 - das Ergebnis erhalten

Die Datei wird direkt in diesem Browser-Tab verarbeitet (FFmpeg laeuft als WebAssembly-Modul, das einmal pro Sitzung geladen wird) und wird nie auf einen Server hochgeladen.

Um den geschnittenen Clip in ein anderes Format umzuwandeln, siehe den Bereich mit verwandten Tools.
