Bangbang Artillery is a free browser cannon duel. Aim with arrow keys (or touch), set power, and fire Space (or tap) at a neural-net opponent that learns after each miss. About ~35 KB after Play, no install, no account.

Want an optimal-stopping number puzzle instead? Try Googol Stopping Game. Want canvas table tennis? Try Classic Pong.

BANGBANG ARTILLERY

You are the right-side cannon. Rotate with Left/Right, set strength with Up/Down, fire with Space. On a phone: drag right to aim, drag left for power, tap to shoot. The left cannon is a fresh neural net each round. Adapted from Victor Ribeiro's bangBang (MIT). About ~35 KB.

Keyboard preferred on desktop; touch works on phones. Session-only - no save data.
