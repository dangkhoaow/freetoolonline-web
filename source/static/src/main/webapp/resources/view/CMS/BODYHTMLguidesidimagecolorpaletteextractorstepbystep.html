Cara mengekstrak warna HEX dari gambar di browser

  1. Buka Image Color Palette Extractor dan pilih gambar lokal.
  2. Pilih 5, 8, 12, atau 16 warna di Colors to extract.
  3. Klik Extract Palette dan tunggu baris swatch muncul.
  4. Klik swatch untuk menyalin HEX ke clipboard.
  5. Clear menghapus pratinjau dan swatch. Tidak ada unggahan ke server.
