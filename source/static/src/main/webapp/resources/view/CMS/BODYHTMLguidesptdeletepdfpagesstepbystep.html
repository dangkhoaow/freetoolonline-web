Excluir Páginas do PDF Passo a Passo

Last reviewed: 2026-07-12

Excluir Páginas do PDF funciona em três passos: abra a ferramenta, forneça sua entrada e depois baixe o resultado. Esta página explica o que cada passo espera.


Passo 1 - abra a ferramenta

Abra o Excluir Páginas do PDF. Nada precisa ser instalado ou conectado antes de começar.


Passo 2 - forneça sua entrada

Não é possível excluir todas as páginas de um PDF - pelo menos uma página deve permanecer no resultado.


Passo 3 - obtenha o resultado

O arquivo é processado nesta aba do navegador e nunca é enviado a um servidor.

Para dividir, unir ou girar o PDF, veja a seção de ferramentas relacionadas.
