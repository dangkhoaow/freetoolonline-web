Cómo Usar el Explorador 3D de Chorros Relativistas y Efecto Doppler

Última revisión: 2026-07-24

Revisado por Equipo editorial de FreetoolOnline

Abre el Explorador 3D de Chorros Relativistas y Efecto Doppler, arrastra los dos controles deslizantes y observa cómo el chorro que se acerca se ilumina mientras el que se aleja se atenúa - calculado en tiempo real a partir de las fórmulas exactas del efecto Doppler.

  1. Abre /space-3d/relativistic-jets-agn.html y espera a que se rendericen el agujero negro, el disco de acreción y los dos chorros bipolares.
  2. Arrastra el control del factor de Lorentz (Γ = 1,01 a 15) para fijar la velocidad del plasma del chorro.
  3. Arrastra el control del ángulo de observación (θ = 1 a 90 grados) para fijar el ángulo entre el eje del chorro y tu línea de visión.
  4. Haz clic en "M87" para cargar los valores reales estimados por VLBA (Γ=10,9, θ=17 grados) para el chorro de M87.
  5. Haz clic en "Blazar-like" para ver el efecto Doppler extremo y unilateral que produce un ángulo de observación pequeño.
  6. Lee el panel de datos para conocer los factores Doppler calculados, la razón de flujo entre los dos chorros y la velocidad transversal aparente.
