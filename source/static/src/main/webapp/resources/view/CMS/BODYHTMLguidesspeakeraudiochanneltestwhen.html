Speaker & Audio Channel Test When To Use

Last reviewed: 2026-07-21

Open the Speaker & Audio Channel Test when you need to confirm left and right channels work - for example after plugging in headphones, switching Bluetooth speakers, or checking a stereo setup before a call.


When it fits

Use it for a quick L / R / both tone check in the browser. Click Play Left, Play Right, or Play Both; use headphones if you need a clear pan. Adjust the tone frequency, then Stop when you are done.


When to use something else

This page does not measure absolute loudness in dB or calibrate studio monitors. For microphone input checks, use the site's microphone test instead. For long sweep tones or measurement microphones, a desktop audio utility is a better fit.


What stays private

Everything runs in the browser - no microphone, no upload, and no account. Audio uses the Web Audio API StereoPannerNode so only the chosen channel plays.
