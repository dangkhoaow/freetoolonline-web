GIF ke Bingkai

Unggah GIF animasi dan alat mengekstrak setiap bingkai sebagai PNG atau JPG terpisah. PNG mempertahankan transparansi palet asli; JPG membuang transparansi tetapi menghasilkan file lebih kecil. Berguna ketika bingkai akan masuk ke editor gambar, lembar sprite, atau digunakan secara individual sebagai gambar statis.

Alat mendekode GIF di server dan mengembalikan setiap bingkai animasi sebagai file gambar tersendiri. File dihapus setelah jendela unduhan singkat. Halaman di /extract-gif-to-image-frames.html menangani unggahan dan ekstraksi.

Memilih format output yang tepat

PNG adalah pilihan tepat ketika GIF menggunakan transparansi palet. JPG adalah pilihan tepat ketika ukuran file lebih penting dari akurasi saluran alfa: setiap bingkai JPG jauh lebih kecil dan cocok untuk konten media sosial atau presentasi.

Apa yang dilakukan dengan bingkai yang diekstrak

Alur kerja paling umum adalah mengedit satu bingkai: ekstrak set, buka bingkai tertentu di editor gambar, buat perubahan, dan rakit kembali. Alur kedua adalah membangun lembar sprite. Ketiga adalah memilih satu bingkai representatif sebagai cadangan statis untuk konteks di mana GIF berulang akan memboroskan bandwidth.
