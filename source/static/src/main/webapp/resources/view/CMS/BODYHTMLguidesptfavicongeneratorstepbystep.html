Favicon Generator - Passo a Passo

Last reviewed: 2026-07-21

Abra o Favicon Generator, escolha uma imagem local e baixe favicon.ico ou PNGs individuais.


Passos

  1. Clique em Choose File e selecione PNG, JPEG, WebP, GIF ou BMP.
  2. Espere as prévias 16, 32, 48, 180, 192 e 512 aparecerem.
  3. Baixe favicon.ico para navegadores, ou 32x32 / 180x180 PNG para usos comuns.
  4. Use os links PNG por tamanho quando precisar de uma dimensão só.

Privacidade

O redimensionamento roda no navegador. Sua imagem não é enviada.
