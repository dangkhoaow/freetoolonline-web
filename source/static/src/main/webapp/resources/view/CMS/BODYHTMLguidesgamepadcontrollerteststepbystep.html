Gamepad / Controller Test Step By Step

Last reviewed: 2026-07-21

Gamepad / Controller Test takes three short steps: open the page, enable polling, then press buttons and move sticks.


Step 1 - open the tool

Go to Gamepad / Controller Test. No install and no login.


Step 2 - enable polling

Click Enable gamepad polling. The panel border turns active and the status line shows Polling.


Step 3 - press buttons and move sticks

Press any button on the controller so the browser exposes the pad. Watch numbered button indicators light and axis values move. Use Reset readout to start over. Nothing left the tab.
