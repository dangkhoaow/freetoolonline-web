Tap the button grid or type an expression - the result previews live under the input before you press "=".

How to use it:
  1. Tap digits and operators on the grid, or type directly into the display.
  2. For a trig or log function, tap its button (sin, cos, tan, sqrt, log, or ln) - it inserts the function name with an open parenthesis, so type the number and close the parenthesis.
  3. Check or uncheck Degrees to switch how sin, cos, and tan read the angle.
  4. Press "=" to commit the live-previewed result back into the input, then keep typing to chain the next operation.
