Linux en el Navegador vs Maquina Virtual vs USB en Vivo

Hay tres formas comunes de probar Linux sin reemplazar su sistema operativo: una maquina basada en navegador como Linux Online, una aplicacion de maquina virtual de escritorio y un USB en vivo. Difieren principalmente en esfuerzo de configuracion, velocidad y donde terminan sus archivos - esta pagina pone los numeros uno al lado del otro.

Como elegir

Elija la maquina del navegador para trabajos rapidos, seguros y sin instalacion: practicar comandos, probar un script arriesgado, ensenanza o cualquier ordenador bloqueado donde instalar software no sea una opcion - y para mover trabajo entre ordenadores, ya que una sesion se exporta como un solo archivo que puede importar en otro lugar para reanudar exactamente. Elija una maquina virtual de escritorio cuando necesite velocidad, software de 64 bits o red; elija un USB en vivo cuando quiera probar Linux en el hardware real.
