CSS Box-Shadow Generator - Wann Nutzen

Last reviewed: 2026-07-24

Nutze den CSS Box-Shadow Generator, wenn du einen Schlagschatten oder einen inneren (inset) Schatten auf einer Karte, einem Button oder einem Panel willst und lieber Schieberegler ziehst, statt Pixelwerte zu raten. Diese Seite zeigt, wann das passt und wann ein anderes Tool besser passt.


Wann es passt

Du gestaltest ein einzelnes Element - eine Karte, einen Button, ein Modal oder einen Bilderrahmen - und willst den Schatten in Echtzeit sehen, während du Versatz, Unschärfe, Ausbreitung, Farbe und Deckkraft anpasst, und dann eine box-shadow-Zeile in dein Stylesheet kopieren.


Wann es nicht passt

Wenn du statt eines Schattens einen mehrfarbigen Gradient-Hintergrund mit Vendor-Prefix-Fallbacks brauchst, nutze den CSS Gradient Generator - eine andere CSS-Eigenschaft für einen anderen visuellen Effekt.
