Roman Numeral Converter

Last reviewed: 2026-07-23

Convert a number between Roman numerals and standard Arabic numbers, right in the browser.

Accepts whole numbers from 1 to 3999, the range the standard subtractive Roman notation represents.

Conversion runs locally from a fixed lookup table - nothing is uploaded and no account is required.

Type a value into either the Arabic-number field or the Roman-numeral field and the other field updates live, with no separate Convert button to click.

Use the Copy result button to copy the current value for pasting into a document, spreadsheet, or chat message.
