Alat yang digunakan/zip-file.html

Mulailah dengan satu folder atau beberapa file alih-alih seluruh drive. Pada kompresi Normal, alat menyeimbangkan ukuran dan kecepatan; Maximum menekan lebih kuat tetapi memakan waktu lebih lama pada folder besar. Jika arsip harus dapat dibuka di versi Windows lama, tetap gunakan format ZIP default. Setelah mengunduh, periksa satu file di dalamnya sebelum menghapus berkas asli. Untuk perlindungan kata sandi atau arsip multi-format, gunakan pusat alat ZIP khusus alih-alih proses kompresi cepat ini.
