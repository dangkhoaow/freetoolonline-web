How the Ichthyovenator size-vs-human toggle stays honest

The Ichthyovenator model is drawn to a fixed on-screen length so it fits the canvas; the Size vs human control then places a 1.8 m person at the true length ratio used by the viewer (9.5 m). Tap Size vs human under the canvas to show or hide the scale figure - the comparison is proportional, not a decorative sticker. Published length still spans about 8.5-10.5 m, so the toggle picks one clear scale inside that disclosed range.

What the Ichthyovenator size numbers are not

What these size numbers are not is a claim of one precise length or weight: the size comparison discloses the about 8.5-10.5 m / 2-2.4 tonnes ranges and uses 9.5 m only for the Size vs human scale; it does not invent soft-tissue or sail-membrane color accuracy, and this is not a win/lose game.
