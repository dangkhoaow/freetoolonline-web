How the Seismosaurus size-vs-human toggle stays honest

The Seismosaurus model is drawn to a fixed on-screen length so it fits the canvas; the Size vs human control then places a 1.8 m person at the true revised length ratio used by the viewer (33 m). Tap Size vs human under the canvas to show or hide the scale figure - the comparison is proportional, not a decorative sticker. The page does not scale to discarded 39-52 m estimates.


What a person would see beside a Seismosaurus

On Seismosaurus Size Comparison, What a person would see is a Late Jurassic Morrison diplodocid - turn on the person in the viewer and orbit beside the hip or mid-body to feel how the long neck and tail sit relative to the scale figure. A license-clean CC-BY-SA 4.0 glTF may swap in after first paint; soft-tissue color remains an artistic choice if the model load fails. This model is not a fossil-accurate skeleton. For the broader genus see the Diplodocus 3D Viewer.


What the Seismosaurus size numbers are not

What these size numbers are not is an endorsement of discarded Gillette-era mega-estimates: the size comparison discloses revised about 33 m / 21-23 tonnes figures and uses 33 m only for the Size vs human scale; it does not invent soft-tissue color accuracy, and this is not a win/lose game.
