What the Albertaceratops size numbers are not

What these size numbers are not is a claim of one precise length or weight: the Albertaceratops size comparison discloses the about 5.8-7 m / 2-3.5 t ranges and uses 6.5 m only for the Size vs human scale; it does not invent soft-tissue color accuracy, and this is not a win/lose game.
