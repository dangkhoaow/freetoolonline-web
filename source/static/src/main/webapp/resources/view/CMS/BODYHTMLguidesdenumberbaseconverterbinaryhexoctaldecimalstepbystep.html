Number Base Converter - Schritt Für Schritt

Last reviewed: 2026-07-23

Number Base Converter (Binary/Hex/Octal/Decimal) läuft in drei Schritten: das Tool öffnen, deinen Wert eingeben und dann das umgerechnete Ergebnis kopieren. Diese Seite geht durch, was jeder Schritt erwartet.


Schritt 1 - Tool öffnen

Öffne Number Base Converter (Binary/Hex/Octal/Decimal). Vor dem Start muss nichts installiert oder angemeldet werden.


Schritt 2 - Eingabe machen

Benutzerdefinierte Basen von 2 bis 36 werden für weniger gängige Umrechnungen jenseits der vier üblichen Voreinstellungen unterstützt.


Schritt 3 - Ergebnis erhalten

Die Umrechnung nutzt exakte Ziffer-für-Ziffer-BigInt-Arithmetik statt des nativen Number-Typs, sodass Werte über 2^53 - 1 hinaus genau bleiben, statt stillschweigend an Genauigkeit zu verlieren.

Verarbeitet keine negativen Zahlen in Zweierkomplement-Darstellung für beliebige Bitbreiten - nur die vorzeichenbehaftete Ganzzahl-Semantik von JavaScript für das Minuszeichen.
