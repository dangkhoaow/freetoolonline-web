Wann eine Farbpalette aus einem Foto extrahieren

Nutzen Sie diesen Extraktor, wenn Sie ein lokales JPEG, PNG oder WebP haben und dominante HEX-Farben fuer Design-System, Moodboard oder CSS-Theme brauchen.

Datei waehlen, Anzahl Farben setzen, Extract Palette klicken. Die Abtastung bleibt im Browser - das Bild wird nie hochgeladen.

Nur manuell waehlen? Nutzen Sie den Color Picker. Fuer Text-auf-Hintergrund-Lesbarkeit den WCAG Contrast Checker.
