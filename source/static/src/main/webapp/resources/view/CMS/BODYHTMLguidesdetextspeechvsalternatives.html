Haeufige Fragen

Warum speichert dieses Werkzeug Audio nicht als MP3?

Browser stellen das synthetisierte Audio nicht als aufnehmbare Audiodatei bereit. Um eine MP3 zu erhalten, waere ein Server-API-Dienst erforderlich - was einen Upload bedeuten wuerde.

Warum das Browser-Werkzeug waehlen?

Keine Installation und kein Text wird an einen Server gesendet - es ist die schnellste Wahl fuer gelegentliche Nutzung, wenn Datenschutz wichtig ist.
