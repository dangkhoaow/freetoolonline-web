The checklist above splits into two routines you can run from the same browser. The 60-second quick path - a camera test, a microphone test, and a typed-key spot-check - is the one to run roughly 90 seconds before you join a call, because it catches the muted mic, the blocked camera, and the wrong-speaker audio that derail the first two minutes. The 5-minute full path adds the keyboard test and LCD test for stuck keys and dead pixels, and is worth running after new hardware arrives, after a major update, or before a return window closes. Every check runs on your own machine in the browser, so the result reflects the exact device that will join the meeting.
