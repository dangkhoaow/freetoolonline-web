Como extrair cores HEX de uma imagem no browser

  1. Abra Image Color Palette Extractor e escolha uma imagem local.
  2. Selecione 5, 8, 12 ou 16 cores na lista Colors to extract.
  3. Clique Extract Palette e aguarde a fila de swatches.
  4. Clique num swatch para copiar o HEX para a area de transferencia.
  5. Clear limpa preview e swatches. Nada e enviado para o servidor.
