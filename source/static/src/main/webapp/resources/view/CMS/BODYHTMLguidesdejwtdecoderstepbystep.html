Jwt Decoder Schritt Fuer Schritt

Last reviewed: 2026-07-14

JWT Decoder arbeitet in drei Schritten: Werkzeug oeffnen, Token einfuegen, dann das dekodierte Ergebnis lesen. Diese Seite erklaert, was jeder Schritt erfordert.


Schritt 1 - Werkzeug oeffnen

Gehen Sie zu JWT Decoder. Vor dem Start muss weder etwas installiert noch angemeldet werden.


Schritt 2 - Token einfuegen

Fuegen Sie den vollstaendigen Token ein - die drei durch Punkte getrennten Teile (header, payload, signature). Ein Token mit einem fehlenden Teil, oder mit einem Teil, der kein gueltiges base64url-JSON ist, wird mit einer klaren Meldung abgelehnt statt abzustuerzen.


Schritt 3 - Ergebnis abrufen

Die Ausgabe zeigt header und payload als formatiertes JSON. Wenn der payload exp oder iat enthaelt, erscheinen sie als lesbare Zeitstempel, und ein bereits abgelaufenes exp wird als EXPIRED markiert.

Das Dekodieren laeuft lokal in diesem Browser-Tab - kein Upload und kein Konto erforderlich.
