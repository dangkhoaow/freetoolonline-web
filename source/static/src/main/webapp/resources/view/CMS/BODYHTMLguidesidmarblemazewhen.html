Kapan Main Marble Maze

Marble Maze cocok saat Anda ingin puzzle marble WebGL 3D: ~1.7 MB, labirin membesar, keyboard - tanpa instal dan tanpa save.

Last reviewed: 2026-07-18

Saat punya keyboard dan WebGL

Panah atau h/j/k/l. Tahan I. Ponsel hanya sentuh bukan target.

Saat ingin labirin terus membesar

Setiap keluar membuat labirin lebih besar (+2). Tutup tab kembali ke level 1 - tanpa localStorage.

Saat unduhan sedang cukup

~1.7 MB, cache setelah load pertama. Tanpa jaringan setelahnya, tanpa akun.
