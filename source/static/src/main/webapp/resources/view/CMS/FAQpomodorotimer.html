Frequently Asked Questions

What does Pomodoro Timer Online do?

It runs a Pomodoro work/break timer that keeps alternating between a Work interval and a Break interval on its own, instead of stopping after one countdown.

Can I change the 25/5 split?

Yes. Set the Work length (1-120 minutes, default 25) and the Break length (1-60 minutes, default 5) before you start, so the cycle matches your own routine.

What do Pause and Reset do?

Pause holds the countdown in place without changing the phase. Reset returns to the start of the current Work interval using whatever Work/Break minutes are currently set.

Does it keep running if I close the tab or switch away?

No. Everything runs locally in this browser tab with no upload and no account; closing or reloading the tab stops the timer, and it does not run or notify in the background.
