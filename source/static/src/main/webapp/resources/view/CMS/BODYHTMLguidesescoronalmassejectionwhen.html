Cuando usar Coronal Mass Ejection 3D Explorer

Ultima revision: 2026-07-21

Abre el Coronal Mass Ejection 3D Explorer cuando necesites ajustar la velocidad de una CME discreta y el reloj Sol-Tierra - sin un panel MHD.


Leccion de tiempo de llegada

Empieza cerca de 250 km/s y sube hasta nearly 3000 km/s para ver el reloj pasar de varios dias a la ventana de 15-18 hours.


Cifras NOAA / NASA

Aqui encajan <250 to nearly 3000 km/s, 15-18 hours, several days (~1-3 days) y billions of tons (~1e16 g class).


Cuando este explorador no encaja

No lo uses para una tormenta historica unica o para la burbuja de viento solar ambiente. Usa el Carrington Event 3D Explorer o el Solar Wind Heliosphere 3D Explorer. Controles en la guia paso a paso.
