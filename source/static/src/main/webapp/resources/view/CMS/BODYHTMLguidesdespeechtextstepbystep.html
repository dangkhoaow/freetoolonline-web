Sprache zu Text Schritt fuer Schritt

Das Werkzeug Sprache zu Text wandelt Sprache direkt im Browser in vier Schritten in Text um: Werkzeug oeffnen, Zuhoeren starten, sprechen und das Ergebnis erhalten. Diese Anleitung erklaert, was jeder Schritt erfordert.

Last reviewed: 2026-07-04

freetoolonline.com Editorial Team

Schnellantwort. Oeffne Sprache zu Text, klicke auf Start listening, erlaube den Mikrofonzugriff, sprich und klicke auf Stop. Der Text erscheint im Browser - nichts wird an Server gesendet.

Schritt 1 - Werkzeug oeffnen

Oeffne Sprache zu Text. Vor dem Start ist keine Installation oder Anmeldung erforderlich.

Schritt 2 - Zuhoeren starten

Klicke auf Start listening. Der Browser bittet um Mikrofonberechtigung - klicke auf Zulassen. Die Transkription beginnt sofort nach der Genehmigung.

Schritt 3 - Sprechen

Sprich normal. Der Text erscheint in Echtzeit im Transkriptionsbereich. Vorlaeufige Ergebnisse werden waehrend des Sprechens angezeigt; der endgueltige Text wird aufgezeichnet, wenn du pausierst.

Schritt 4 - Ergebnis erhalten

Klicke auf Stop, wenn du fertig bist. Verwende die Schaltflaeche Download .txt, um den Text zu speichern. Wenn du die umgekehrte Richtung benoeligst, schau in den verwandten Werkzeugen nach dem TEXT zu SPRACHE-Konverter.

Haeufige Fragen

Wird meine Sprache an einen Server gesendet?

Nein. Das Werkzeug verwendet die Spracherkennungs-API des Browsers - alles geschieht auf deinem Geraet.

Funktioniert es in Firefox?

Nein. Spracherkennung erfordert einen Chromium-basierten Browser (Chrome, Edge) oder Safari. Firefox unterstuetzt diese API nicht.

Brauche ich ein Konto oder eine Installation?

Nein. Das Werkzeug laeuft direkt im Browser, ohne Installation oder Konto.
