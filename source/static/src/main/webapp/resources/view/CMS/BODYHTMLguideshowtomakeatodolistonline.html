How to Make a To-Do List Online - Free and No Sign-Up

A to-do list tool keeps a simple checklist right in your browser. Type a task and press Enter or click Add, tick items off as you finish them, and delete the ones you no longer need. The list saves locally, so it is still there after you reload the page - no app install and no sign-up.

Steps

  1. Open the tool and type your first task in the input box.
  2. Press Enter or click Add to put it on the list.
  3. Tick the checkbox next to a task when you finish it; it shows as done.
  4. Use the Delete button to remove a task you no longer need.

Where your list is saved

The list lives in your browser's local storage, so it is still there when you reload the page or come back later on the same device and browser. Nothing is uploaded and there is no account to create. Clearing this site's browser data removes the list.

When to use it

It is handy for a quick checklist when you do not want to install an app or sign in - a shopping list, a packing list, or the day's tasks. A running count shows how many items you have and how many are still left to do.
