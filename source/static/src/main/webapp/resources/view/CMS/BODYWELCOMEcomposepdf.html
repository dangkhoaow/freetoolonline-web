Compose PDF online lets you write content in a simple editor and export it as a clean PDF file.

  1. Type or paste your content into the editor.
  2. Enter a PDF name.
  3. Click Submit to generate the PDF.

Note: Use Reload to reset the editor if formatting gets messy.

Recoverable from a formatting mistake

The editor accepts inline bold, colour, and line breaks the same way a quick email composer does, but pasted-in styling sometimes collides with what you typed and the editor looks wrong. When that happens - wrong font size, leftover background colour, a stray bullet you cannot delete - click Reload instead of fighting the formatting. The Reload button resets the workspace to a clean draft in one click; you lose the current draft on purpose (that is the point) and start the page over without restarting the browser or losing this tab. This small "undo the whole thing" lever is the quick-export workflow that most PDF tools omit, and it is what makes the type-and-export loop fast enough for quick reports and meeting notes, not just for documents drafted somewhere else.

Text out of a PDF, not text into one. The opposite task - free PDF to text extraction online, pulling words out of an existing PDF into a copy-paste block or .txt download - lives at PDF to Text online. Both tools run in the browser without setup or an account. Pick by the direction your starting file is going.

Split a PDF you already have into smaller files. To break it down by single pages or by range, open Split PDF by range or Split PDF to each pages on the hub.

The two splitters differ by the output they write, and naming the difference saves a wrong pick. Split PDF by range pulls a chosen page span - pages 4 to 9, say - out as one new document, which is what you want when a contiguous section has to travel as a single file. Split PDF to each pages instead writes every page as its own separate file, which fits when each sheet is sent, signed, or stored on its own.

Merge several existing PDFs into one document. If your task is to take two or more already-built PDFs and combine them into one file, open Join PDFs from multiple files instead - that tool accepts the existing PDFs as uploads and writes the merged output.

Shrink a PDF you already have. If the file on hand is an already-built PDF that needs to get smaller, this composer is not the route - it creates a fresh document and does not shrink an existing one.
