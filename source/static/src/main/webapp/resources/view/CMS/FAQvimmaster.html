Frequently Asked Questions

What is Vim Master?

A free browser Vim lesson game. Clear short JSON lessons in a simulated editor and earn XP, combos, and badges.

How do I play?

Press Play, then Start Learning. Type real Vim keys to meet each lesson objective. Use cheat-mode practice and challenge timers when you want drills.

Does it save progress?

Yes. Progress, backup, and retention use ftol:vimmaster:progress, ftol:vimmaster:progressBackup, and ftol:vimmaster:retentionState in this browser. You can also export / import a progress code.

How big is the download?

About ~584 KB of HTML, CSS, JS, and bundled lesson JSON after you press Play.

Is this the same as Vim Motion Academy?

No. Vim Master is a JSON curriculum of command lessons with XP and badges. Vim Motion Academy is a different Vim motion RPG on this site.

Do I need to install Vim?

No. The editor is simulated in the browser. A physical keyboard works best.

Is this open source?

Yes. Adapted from renzorlive/vimmaster (MIT). No CDN runtime assets.
