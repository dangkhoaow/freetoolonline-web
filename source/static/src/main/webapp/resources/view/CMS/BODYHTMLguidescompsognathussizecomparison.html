What the Compsognathus size numbers are not

What these size numbers are not is a blended composite: the panel headline figures (1.25 m / 0.29 m hip / 3 kg) are never merged with the German juvenile range (about 0.7-0.9 m / 0.3-0.6 kg) or treated as if every specimen matched the French adult.
