Favicon Generator - Schritt für Schritt

Last reviewed: 2026-07-21

Öffne Favicon Generator, wähle ein lokales Bild und lade favicon.ico oder einzelne PNGs herunter.


Schritte

  1. Auf Choose File klicken und PNG, JPEG, WebP, GIF oder BMP wählen.
  2. Warten, bis die Vorschauen 16, 32, 48, 180, 192 und 512 erscheinen.
  3. favicon.ico für Browser laden, oder 32x32 / 180x180 PNG für gängige Plätze.
  4. Die PNG-Links pro Größe nutzen, wenn du nur eine Dimension brauchst.

Privatsphäre

Die Skalierung läuft im Browser. Dein Bild wird nicht hochgeladen.
