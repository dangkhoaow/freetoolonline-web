Create GIFs Online Easily from Any Video

Last updated: Nov 13, 2024

Last reviewed: 2026-06-26

Upload a short video to convert a trimmed segment into an animated GIF - no software, runs from any browser. The tool also supports adding text, cropping the frame, and applying effects before export.

GIF maker turns a short video clip into an animated GIF you can share in messages, forums, or a doc.

  1. Upload a short video (MP4, WebM, or MOV - under ~30 seconds works best).
  2. Pick the start point, duration, and output size on the preview slider.
  3. Download the generated GIF or copy its direct share link.

Note: Longer clips produce larger files and slower chat-app previews; trim to 3-10 seconds for best results.

Output width and height accept manual pixel values or the AutoW / AutoH checkboxes that scale proportionally on the server, so a fixed-width forum preview only needs the width typed in - the matching height drops in automatically. The frame-rate slider runs from 1 to 60 fps and is the third lever next to trim and output size; lower fps shrinks the file for chat-app previews while higher fps keeps motion smooth, so a 30 fps reaction GIF stays under typical chat-app limits when the clip is trimmed to 3 to 10 seconds first. After Make, a direct share link sits next to the Download button so a chat-app recipient can preview the GIF without first saving it; both download and direct link rotate off the server once the retention window shown on the page closes, so a working copy belongs in a permanent folder before that interval expires if the GIF has to outlive the session.


Cropping a still image, not a GIF

Cropping a single still image (PNG, JPG, or WebP) belongs on the dedicated crop image tool, not on this GIF maker. The crop image tool accepts the same source formats and outputs the format you uploaded, with a draggable selection rectangle and numeric width / height inputs. This GIF maker handles short-video to animated-GIF conversion with trim, duration, and output-size controls on the preview slider; it is the right tool when the result is an animation, and the still-image crop tool is the right tool when the result is a single picture.

Want to prep the source frames first? Compress or resize them to trim the final file size, or retouch a still in the photo editor before you animate.
