Favicon Generator - Cuándo Usar

Last reviewed: 2026-07-21

Abre Favicon Generator cuando necesites tamaños estándar de favicon desde un logo o foto sin instalar una app de diseño.


Cuándo encaja

Úsalo para el icono de la pestaña, apple-touch icon o un juego rápido de PNG cuadrados antes de publicar.


Cuándo omitirlo

Omítelo si necesitas quitar el fondo, iconos adaptativos de Android o un símbolo solo desde texto - esta página solo redimensiona un recorte cuadrado de una imagen existente.
