Code online bearbeiten - kostenlos, im Browser

Bearbeite Code direkt im Browser in einem vollwertigen, kostenlosen Editor - die Engine hinter VS Code. Waehle aus 18 Sprachen, wechsle das Theme, oeffne eine Datei zum Bearbeiten und speichere deine Arbeit in einer Datei oder lade sie herunter. Alles laeuft auf deinem Geraet; nichts wird hochgeladen und es gibt nichts zu installieren.

Schritte

  1. Oeffne das Tool und klicke auf Launch editor; der Editor laedt beim ersten Mal in deinem Browser.
  2. Waehle eine Sprache aus der Liste, damit die Syntaxhervorhebung zu deinem Code passt.
  3. Waehle ein helles, dunkles oder kontrastreiches Theme fuer deinen Bildschirm.
  4. Tippe oder fuege deinen Code ein, oder klicke auf Open a file, um eine Datei von deinem Geraet zu laden.
  5. Klicke auf Save / Download, um deine Arbeit in eine Datei zu schreiben (oder sie herunterzuladen).
  6. Optional benenne den Inhalt und klicke auf Save snippet, um ihn in diesem Browser fuer spaeter zu speichern.

Der Editor hebt 18 Sprachen hervor und waehlt beim Speichern eine passende Dateiendung. Einige gaengige:

Speichern und Snippets

In Chromium-basierten Browsern schreibt der Save-Button direkt in eine von dir gewaehlte Datei; in Firefox und Safari laedt er die Datei herunter. Benannte Snippets werden im lokalen Speicher dieses Browsers abgelegt, sind also noch da, wenn du auf demselben Geraet und Browser zurueckkommst - sie werden nicht zwischen Geraeten synchronisiert, und das Loeschen der Browserdaten dieser Seite entfernt sie.

Was dieses Tool nicht tut

Es ist ein Editor, kein Ausfuehrer. Es schreibt, hebt hervor und speichert Code, fuehrt deinen Code aber nicht aus - fuehre ihn in deiner eigenen Umgebung aus. Nichts, was du tippst, wird hochgeladen; der Editor arbeitet vollstaendig auf deinem Geraet.
