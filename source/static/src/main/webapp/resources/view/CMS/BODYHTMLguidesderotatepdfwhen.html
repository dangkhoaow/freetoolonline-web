PDF Drehen Wann Nutzen

Last reviewed: 2026-07-13

PDF Drehen lohnt sich, wenn du einen schnellen, einmaligen Durchlauf ohne Installation brauchst. Diese Seite zeigt, wann das passt und wann nicht.


Wann es passt

PDF Drehen passt, wenn jede Seite die gleiche Drehung braucht - 90, 180 oder 270 Grad - etwa bei einem Dokument, das schief oder auf dem Kopf eingescannt wurde. Der Winkel wird einheitlich auf das gesamte Dokument angewendet; einzelne Seiten koennen nicht auf unterschiedliche Winkel gedreht werden.

Die Drehung wird zur bestehenden Ausrichtung jeder Seite addiert statt sie zurueckzusetzen, daher behaelt ein Dokument mit unterschiedlich ausgerichteten Seiten diesen Unterschied, nur um den gewaehlten Winkel verschoben.


Was zu erwarten ist

Was zu erwarten ist, ist dasselbe PDF zurueck, mit jeder Seite gedreht um den gewaehlten Winkel - gleiche Seitenzahl, gleicher Inhalt, nur gedreht. Die Datei wird direkt in diesem Browser-Tab verarbeitet und nie an einen Server hochgeladen.
