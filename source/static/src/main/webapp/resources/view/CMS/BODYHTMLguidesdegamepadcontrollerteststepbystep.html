Gamepad / Controller Test Schritt Für Schritt

Last reviewed: 2026-07-21

Gamepad / Controller Test läuft in drei Schritten: öffnen, Polling aktivieren, Tasten drücken und Sticks bewegen.


Schritt 1 - öffnen

Gehen Sie zu Gamepad / Controller Test. Keine Installation und kein Login.


Schritt 2 - Polling aktivieren

Klicken Sie Enable gamepad polling. Der Rand wird aktiv und der Status zeigt Polling.


Schritt 3 - Tasten und Sticks

Drücken Sie eine Taste am Controller. Nummerierte Indikatoren und Achsenwerte bewegen sich. Reset leert die Anzeige - nichts verlässt den Tab.

Optionen vergleichen: vs Alternativen.
